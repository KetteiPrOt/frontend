Line Height

Controla la propiedad line-height de CSS.

// Dado por REM:
.leading-[3,4,5,6,7,8,9,10]

// Cualitativo
.leading-none (: 1;)
.leading-tight (: 1.25;)
.leading-snug (: 1.375;)
.leading-normal (: 1.5;)
.leading-relaxed (: 1.625;)
.leading-loose (: 2;)

Text Align

Controla la propiedad text-align de CSS.

.text-left/right/center/justify

.text-start/end (: start/end;)

Float

Controla la propiedad float de CSS.

.float-left/right/none

Clear

Controla la propiedad clear de CSS.

.clear-left/right/both/none

Text Decoration

Controla la propiedad text-decoration-line de CSS.

.underline (: underline;) .overline (: overline;) .line-through (: line-through;) .no-underline (: no-underline;)

Controla la propiedad text-decoration-colorde CSS.

.decoration-color-brightness

Controla la propiedad text-decoration-style de CSS.

.decoration-solid/double/dotted/dashed/wavy

Decoration Thickness

Controla la propiedad text-decoration-thickness de CSS.

.decoration-auto (: auto;)
.decoration-0/1/2/4/8 (: 0px/1px/2px/4px/8px;)
.decoration-from-font (: from-font;)

Underline Offset

Controla la propiedad text-underline-offset de CSS.

.underline-offset-0/1/2/4/8 (: 0px/1px/2px/4px/8px;)
.underline-offset-auto (: auto;)

Font Family

Controla la propiedad font-family de CSS.

font-sans (: ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, [...], "Noto Color Emoji";)
font-serif (: ui-serif, Georgia, Cambria, "Times New Roman", Times, serif;)
font-mono (: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, [...], monospace;)

Font Size

Controla la propiedad font-size de CSS.

// Definida por REM:
.text-xs/sm/base/lg/xl/[2-9]xl (: Xrem; line-height: 1;)

Font Weight

Controla la propiedad font-weight de CSS.

.font-thin (: 100;)
.font-extralight (: 200;)
.font-light (: 300;)
.font-normal (: 400;)
.font-medium (: 500;)
.font-semibold (: 600;)
.font-bold (: 700;)
.font-extrabold (: 800;)
.font-black (: 900;)

Font Style

Controla la propiedad font-style de CSS.

.italic (: italic;)
.not-italic (: normal;)

White Space

Controla la propiedad white-space de CSS.

.whitespace-normal/nowrap/pre/pre-line/pre-wrap/break-spaces
