Unidades de Medida

En Tailwind tenemos principalmente medidas dadas por REM, px, y medidas proporcionales al contenedor Padre.

Medidas dadas por REM y px:

class-0 property: 0px; class-px property: 1px; class-0.5 property: 0.125rem; /* 2px */ class-1 property: 0.25rem; /* 4px */ class-1.5 property: 0.375rem; /* 6px */ class-2 property: 0.5rem; /* 8px */ class-2.5 property: 0.625rem; /* 10px */ class-3 property: 0.75rem; /* 12px */ class-3.5 property: 0.875rem; /* 14px */ class-4 property: 1rem; /* 16px */ class-5 property: 1.25rem; /* 20px */ class-6 property: 1.5rem; /* 24px */ class-7 property: 1.75rem; /* 28px */ class-8 property: 2rem; /* 32px */ class-9 property: 2.25rem; /* 36px */ class-10 property: 2.5rem; /* 40px */ class-11 property: 2.75rem; /* 44px */ class-12 property: 3rem; /* 48px */ class-14 property: 3.5rem; /* 56px */ class-16 property: 4rem; /* 64px */ class-20 property: 5rem; /* 80px */ class-24 property: 6rem; /* 96px */ class-28 property: 7rem; /* 112px */ class-32 property: 8rem; /* 128px */ class-36 property: 9rem; /* 144px */ class-40 property: 10rem; /* 160px */ class-44 property: 11rem; /* 176px */ class-48 property: 12rem; /* 192px */ class-52 property: 13rem; /* 208px */ class-56 property: 14rem; /* 224px */ class-60 property: 15rem; /* 240px */ class-64 property: 16rem; /* 256px */ class-72 property: 18rem; /* 288px */ class-80 property: 20rem; /* 320px */ class-96 property: 24rem; /* 384px */

Medidas proporcionales al contenedor Padre:

class-1/2 property: 50%; class-1/3 property: 33.333333%; class-2/3 property: 66.666667%; class-1/4 property: 25%; class-2/4 property: 50%; class-3/4 property: 75%; class-1/5 property: 20%; class-2/5 property: 40%; class-3/5 property: 60%; class-4/5 property: 80%; class-1/6 property: 16.666667%; class-2/6 property: 33.333333%; class-3/6 property: 50%; class-4/6 property: 66.666667%; class-5/6 property: 83.333333%; class-1/12 property: 8.333333%; class-2/12 property: 16.666667%; class-3/12 property: 25%; class-4/12 property: 33.333333%; class-5/12 property: 41.666667%; class-6/12 property: 50%; class-7/12 property: 58.333333%; class-8/12 property: 66.666667%; class-9/12 property: 75%; class-10/12 property: 83.333333%; class-11/12 property: 91.666667%; class-full property: 100%;

Adicionalmente tambien tenemos las medidas del viewport para las propoedades height y width:

.h-vh (: 100vh;)
.w-vw (: 100vw;)

Background Repeat

Controla la propiedad background-repeat de CSS.

.bg-repeat/no-repeat/repeat-x/repeat-y

Background Size

Controla la propiedad background-size de CSS.

.bg-auto/contain/cover

Background Attachment

Controla la propiedad background-attachment de CSS.

.bg-local/scroll/fixed

Background Position

Controla la propiedad background-position de CSS.

.bg-left-top .bg-top .bg-right-top
.bg-left .bg-center .bg-right
.bg-left-bottom .bg-bottom .bg-right-bottom

Opacity

Controla la propiedad opacity de CSS.

.oopacity-0/25/50/75/100 (: 0/.25/.5/.75/1;)
.opacity-5/10/20/30/40/60/70/80/90/95

Visibility

Controla la propiedad visibility de CSS.

.visible
.invisible

Overflow

Controla la propiedad overflow de CSS.

.overflow-auto/visible/hidden/scroll

.overflow-x-value
.overflow-y-value

Word Break

Controla la propiedad overflow-wrap de CSS.

.break-normal (: normal;)
.break-words (: break-words;)

Text Overflow

Controla la propoedad text-oveflow de CSS.

.text-clip/ellipsis (: clip/ellipsis;) .truncate (: ellipsis; oveflow: hidden; white-space: nowrap;)

Box Shadow

Controla la propiedad box-shadow de CSS.

shadow-sm [: 0 1px 2px 0 rgb(0 0 0 / 0.05);]
shadow [: 0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1);]
shadow-md [: 0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1);]
shadow-lg [: 0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1);]
shadow-xl [: 0 20px 25px -5px rgb(0 0 0 / 0.1), 0 8px 10px -6px rgb(0 0 0 / 0.1);]
shadow-2xl [: 0 25px 50px -12px rgb(0 0 0 / 0.25);]
shadow-inner [: inset 0 2px 4px 0 rgb(0 0 0 / 0.05);]
shadow-none [: 0 0 #0000;]

Box Shadow Color

Controla el color de la propiedad box-shadow de CSS.

.shadow-color-brightness

Linear Gradient

Controla la propiedad background-image de CSS. En la documentacion se encuentra como Background Image.

.bg-none (: none;)

.bg-gradient-to-t/r/b/l
.bg-gradient-to-tl/tr/bl/br

Controla el color del gradiente lineal:

.from-color-brightness
.to-color-brightness

Outline Style

Controla la propiedad outline-style de CSS.

.outline (: solid;)
.outline-dashed (: dashed;)
.outline-dotted (: dotted;)
.outline-double (: double;)

.outline-none (outline: 2px solid transparent; outline-offset: 2px;)

Controla la propiedad outline-color de CSS.

.outline-color-brightness

Outline Width

Controla la propiedad outline-width de CSS.

.outline-0/1/2/4/8 (: 0px/1px/2px/4px/8px;)

Outline Offset

Controla la propiedad outline-offset de CSS.

.outline-offset-0/1/2/4/8 (: 0px/1px/2px/4px/8px;)

Border Collapse

Controla la propiedad boder-collapse de CSS.

.border-separate/collapse

List Style Type

Controla la propiedad list-style-type de CSS.

.list-none/disc/decimal (: none/disc/decimal;)

List Style Position

Controla la propiedad list-style-position de CSS.

.list-inside/outside (: inside/outside;)

List Style Image

Controla la propiedad list-style-image de CSS.

.list-image-none (: none;)

.list-image-[url(media/check.png)] {: url('media/check.png');}
