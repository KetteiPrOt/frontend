Border Width

Controla la propiedad border-width de CSS.

.border (: 1px;)
.border-0/2/4/8 (: 0px, 2px, 4px, 4px;)

Controla las propiedades border-top/right/bottom/left de CSS.

// Eje X (left, right)
.border-x (: 1px;)
.border-x-0/2/4/8 (: 0px, 2px, 4px, 4px;)

// Eje Y (top, bottom)
.border-y (: 1px;)
.border-y-0/2/4/8 (: 0px, 2px, 4px, 4px;)

// Top, Left, Bottom, Right
.border-t/l/b/r (: 1px;)
.border-t/l/b/r-0/2/4/8 (: 0px, 2px, 4px, 4px;)

Border Style

Controla la propiedad border-style de CSS.

.border-solid/dashed/dotted/double/hidden/none

Controla la propiedad border-color de CSS.

.border-color-brightness

Border Radius

Controla la propiedad border-radius de CSS.

.rounded-none (: 0px;)
.rounded-full (: 9999px;)

// Dado por REM
.rounded-sm
.rounded
.rounded-md
.rounded-lg
.rounded-xl/2xl/3xl // Redondear lados .rounded-t/r/b/l-size // Redondear aristas .rounded-tl/tr/br/rl-size

Padding

Controla la propiedad padding de CSS.

.p-0 (: 0px;)
.p[x/y/t/r/b/l]-0 (: 0px;)

// Definido por REM:
.p-0.5/1.5/2.5/3.5
.p-[0-12]
.p-12/14/16/20/24/28/32/36/40/44/48/52/56/60/64
.p-72/80/96

// Eje X (left, right)
.px-size

// Eje Y (top, bottom)
.py-size

// Top, Right, Bottom, Right
.p[t/r/b/l]-size
