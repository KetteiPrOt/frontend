Guias y Documentacion

Puedes ver las guias de instalacion de diferentes frameworks y ambientes en la pagina oficial de Tailwind.

Yo voy a seguir la guia de Laravel. Al parecer si instalas Breeze o Jetstream ya se instala tailwind.

Directivas de Tailwind

En el archivo .../resources/css/app.css vamos a definir las partes de Tailwind que Vite debe compilar en las paginas de nuestro proyecto:

@tailwind base;
@tailwind components;
@tailwind utilities;

Servidor de Vite

No olvides encender el servidor de Vite para poder visualizar Tailwind:

npm run dev

Ya podemos usar Tailwind en el proyecto simplemente incluyendo la directiva Vite en cada vista donde queramos:

@vite('resources/css/app.css')

Desde CDN

Adicionalmente tambien podemos cargar Tailwind desde una CDN (Content Distribution Network) unsando el script de la pagina oficial.

La CDN esta diseniada para propositos de desarrollo, por lo que no es la mejor manera de usar Tailwind en un ambiente de Produccion.

Como funcina Tailwind

Es una libreria que tiene cientos de Clases predefinidas a modo de "Utilidades" que mapean casi todas las propiedades de CSS.

Con Tailwind no tenemos que preocuparnos por inventar nombres de Clases para elementos tan abstractos como un contenedor Flexbox, sino que simplemente usamos la Clase .flex:

div.flex

Del mismo modo podemos agregar todas las demas propiedades CSS que nesecitemos directamente en el atributo class="..." de los elementos HTML.
