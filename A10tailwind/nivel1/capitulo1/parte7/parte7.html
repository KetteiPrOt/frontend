Resize

.resize-none/x/y (: none/vertical/horizontal;)

Min-Width/Height

Controla la propiedad min-width de CSS.

.min-w-0 (: 0px;)
.min-w-full (: full;)

.min-w-min (: min-content;)
.min-w-max (: max-content;)
.min-w-fit (: fit-content;)

Controla la propiedad min-height de CSS.

.min-h-0 (: 0px;)
.min-h-full (: full;)
.min-h-screen (: 100vh;)

.min-h-min (: min-content;)
.min-h-max (: max-content;)
.min-h-fit (: fit-content;)

Max-Width

Controla la propiedad max-width de CSS.

// Total:
max-w-none (: none;)
max-w-full (: 100%;)

// Segun la pantalla:
max-w-screen-sm (: 640px;)
max-w-screen-md (: 768px;)
max-w-screen-lg (: 1024px;)
max-w-screen-xl (: 1280px;)
max-w-screen-2xl (: 1536px;)

// Definido por REM:
max-w-0     (: 0rem;) /* 0px */
max-w-xs    (: 20rem;) /* 320px */
max-w-sm    (: 24rem;) /* 384px */
max-w-md    (: 28rem;) /* 448px */
max-w-lg    (: 32rem;) /* 512px */
max-w-xl    (: 36rem;) /* 576px */
max-w-2xl    (: 42rem;) /* 672px */
max-w-3xl    (: 48rem;) /* 768px */
max-w-4xl    (: 56rem;) /* 896px */
max-w-5xl    (: 64rem;) /* 1024px */
max-w-6xl    (: 72rem;) /* 1152px */
max-w-7xl    (: 80rem;) /* 1280px */

// En base al contenido:
max-w-min   (: min-content;)
max-w-max   (: max-content;)
max-w-fit   (: fit-content;)

Max-Height

Controla la propiedad max-height de CSS.

// Total:
max-h-none (: none;)
max-h-full (: 100%;)
max-h-screen (: 100vh;)

// Definido por REM:
.max-h-[0-12]
.max-h-12/16/20/14/28/32/36/40/44/48/56/60/64
.max-h-72/80/96

// En base al contenido:
max-h-min   (: min-content;)
max-h-max   (: max-content;)
max-h-fit   (: fit-content;)

Vertical Aling

Controla la propiedad vertical-aling de CSS.

.aling-top/middle/bottom/baseline (: top/middle/bottom/baseline;)

.aling-sub/super/text-top/text-bottom (: sub/super/text-top/text-bottom;)

Object Fit

Controla la propiedad object-fit de CSS.

.object-fill/contain/cover (: fill/contain/cover;)
.object-none (: none;)

Object Position

Controla la propiedad object-position de CSS.

.object-left-top .object-none .object-left .object-left-bottom .object-none
.object-top .object-center .object-bottom
.object-right-top .object-none .object-right .object-right-bottom .object-none

Cursor

Controla la propiedad cursor de CSS.

.cursor-auto/default/pointer/text/wait/move/help/not-allowed

(: auto/default/pointer/text/wait/move/help/not-allowed;)

User Select

Controla la propiedad user-select de CSS.

.select-auto/none/text/all (: auto/none/text/all;)
