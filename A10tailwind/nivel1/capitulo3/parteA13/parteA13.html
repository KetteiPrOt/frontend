Ejemplos

Display Grid

Establece display: grid; en el elemento.

.grid

Grid Template Colums

Controla la propiedad grid-template-colums de CSS.

.grid-cols-n [: repeat(n, minmax(0, 1fr));]
Donde n puede ser de 1 hasta 12

.grid-cols-none (: none;)

Grid Template Rows

Controla la propiedad grid-template-rows de CSS.

.grid-rows-n [: repeat(n, minmax(0, 1fr));]
Donde n puede ser de 1 hasta 6

.grid-rows-none (: none;)

Grid Gap

Controla las propiedades grid-gap, column-gap, y row-gap de CSS.

.gap-n (grid-gap: n;)
.gap-x-n (column-gap: n;)
.gap-y-n (row-gap: n;)

Donde n puede ser:

// En pixeles
0/px (: 0px/1px;)

// Dado por REM
0.5/1.5/2.5/3.5
[0-12]
12/14/16/20/24/28/32/36/40/44/48/52/56/60/64
72/80/96

Grid Column Start / End

Controla la propiedad grid-column, grid-column-start, y grid-column-end de CSS.

.col-span-n (grid-column: span n / span n;)
.col-start-n (grid-column-start: n;)
.col-end-n (grid-column-end: n;)

Donde n puede ser del 1 hasta el 12

// Valor Automatico:
.col-auto (grid-column: auto;)
.col-start-auto (grid-column-start: auto;)
.col-end-auto (grid-column-end: auto;)

// Todo el espacio:
.col-span-full grid-column: 1 / -1;

Grid Row Start / End

Controla la propiedad grid-row, grid-row-start, y grid-row-end de CSS.

.row-span-n (grid-row: span n / span n;)
.row-start-n (grid-row-start: n;)
.row-end-n (grid-row-end: n;)

Donde n puede ser del 1 hasta el 12

// Valor Automatico:
.row-auto (grid-row: auto;)
.row-start-auto (grid-row-start: auto;)
.row-end-auto (grid-row-end: auto;)

// Todo el espacio:
.row-span-full grid-row: 1 / -1;

Grid Auto Flow

Controla la propiedad grid-auto-flow de CSS.

.grid-flow-row/col (: row/column;)

Dense (value)

Establece el valor dense en la propiedad anterior de CSS.

.grid-flow-row-dense (: row dense;)

.grid-flow-col-dense (: column dense;)

Establece una propiedad equivalente a las dos anteriores juntas

.grid-flow-dense (: dense;)

Grid Auto Columns / Rows

Controla las propiedades grid-auto-columns y grid-auto-rows de CSS.

auto-cols-auto/min/max/fr (grid-auto-columns: auto/min-content/max-content/minmax(0, 1fr);)

auto-rows-auto/min/max/fr (grid-auto-rows: auto/min-content/max-content/minmax(0, 1fr);)
