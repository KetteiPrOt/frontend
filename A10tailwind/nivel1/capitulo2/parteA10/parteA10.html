Transition

Controla las propiedades transition-property, transition-timing-function, y transition-duration de CSS.

La mayoria del tiempo problamente uses .transition-all

.transition-none (transition-property: none;)

.transition-all (
    transition-property: all;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1); /* like function ease */
    transition-duration: 150ms;
)

.transition (
    transition-property: color, background-color, border-color, text-decoration-color, fill, stroke, opacity, box-shadow, transform, filter, backdrop-filter;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
    transition-duration: 150ms;
)

.transition-colors (
    transition-property: color, background-color, border-color, text-decoration-color, fill, stroke;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
    transition-duration: 150ms;
)

.transition-opacity (
    transition-property: opacity;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
    transition-duration: 150ms;
)

.transition-shadow (
    transition-property: box-shadow;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
    transition-duration: 150ms;
)

.transition-transform (
    transition-property: transform;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
    transition-duration: 150ms;
)

Transition Duration

Controla la propiedad transition-duration de CSS.

.duration-n (: 0s/75ms/100ms/150ms/200ms/300ms/500ms/700ms/1000ms;)

Donde n puede ser: 0/75/100/150/200/300/500/700/1000

Transition Timing Function

Controla la propiedad transition-timing-function de CSS.

Class CSS values
.ease-linear [: linear;]
.ease-in [: cubic-bezier(0.4, 0, 1, 1);]
.ease-out [: cubic-bezier(0, 0, 0.2, 1);]
.ease-in-out [: cubic-bezier(0.4, 0, 0.2, 1);]

Transition Delay

Controla la propiedad transition-delay de CSS.

.delay-n (: 0s/75ms/100ms/150ms/200ms/300ms/500ms/700ms/1000ms;)

Donde n puede ser: 0/75/100/150/200/300/500/700/1000

Animation

Utilidades para aplicar sencillas animaciones de CSS.

.animation-none (animation: none;)

Animacion de Spin

.animation-spin

[
    animation: spin 1s linear infinite;

    @keyframes spin {
         from {
              transform: rotate(0deg);
         }
         to {
              transform: rotate(360deg);
         }
    }
]

Animacion de Ping

.animate-ping

[
    animation: ping 1s cubic-bezier(0, 0, 0.2, 1) infinite;

    @keyframes ping {
         75%, 100% {
              transform: scale(2);
              opacity: 0;
         }
    }
]

Animacion de Pulse

.animation-pulse

[
    animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;

    @keyframes pulse {
         0%, 100% {
              opacity: 1;
         }
         50% {
              opacity: .5;
         }
    }
]

Animacion de Bounce

.animation-bounce

[
    animation: bounce 1s infinite;

    @keyframes bounce {
         0%, 100% {
              transform: translateY(-25%);
              animation-timing-function: cubic-bezier(0.8, 0, 1, 1);
         }
         50% {
              transform: translateY(0);
              animation-timing-function: cubic-bezier(0, 0, 0.2, 1);
         }
    }
]
