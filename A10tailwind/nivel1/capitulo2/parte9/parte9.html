Scale

Establece el valor scale(...) a la propiedad transform de CSS.

.scale-n [: scale(0/.5/.75/.9/.95/1/1.05/1.1/1.25/1.5);]

Donde n puede ser: 0/50/75/90/95/100/105/110/125/150

.scale-x-n [: scaleX( ... )]
.scale-y-n [: scaleY( ... )]

Rotate

Establece el valor rotate(...) a la propiedad transform de CSS.

.rotate-n [: scale(0deg/1deg/2deg/3deg/6deg/12deg/45deg/90deg/180);]

Donde n puede ser: 0/1/2/3/6/12/45/90/180

Para valores negativos de n usa:
.-rotate-n

Translate

Establece el valor translateX(...)/translateY(...) a la propiedad transform de CSS.

.translate-x-0/px [: translateX(0px/1px);]
.translate-y-0/px [: translateY(0px/1px);]

// Dado por REM
.translate-x-n
.translate-y-n

Donde n puede ser:
[1-12]
0.5/1.5/2.5/3.5
14/16/20/24/28/32/36/40/44/48/52/56/60/64
72/80/96

// Segun el Padre
.translate-x-n
.translate-y-n

Donde n puede ser:
1/2,1/3,2/3,1/4,2/4,3/4,full

// Valores Negativos
Para valores negativos de n usa: .-translate-x-n
.-translate-y-n

Skew

Establece el valor skewX(...)/skewY(...) a la propiedad transform de CSS.

.skew-x-n [: skewX(0deg/1deg/2deg/3deg/6deg/12deg);]
.skew-y-n [: skewY(...);]

Donde n puede ser: 0/1/2/3/6/12

Para valores negativos de n usa:
.-skew-x-n
.-skew-y-n

Transform Origin

Controla la propiedad tranform-origin de CSS.

.origin-top-left .origin-top .origin-top-right
.origin-left .origin-center .origin-right
.origin-bottom-left .origin-bottom .origin-bottom-right

transform-none

Usa esta utilidad para restablecer la propiedad transform a none.

.transform-none

.responsive:transform-none

transform-gpu/cpu

Si tu transicion se renderiza mejor usando el GPU que el CPU, puedes forzar la aceleracion de hardware con esta la utilidad.

.transform-gpu

Para desacer este condicional y forzar a volver a la CPU:

.transform-cpu
