Funciones para Cadenas

Acceder a un caracter

Una cadena de caracteres es simplemente una sucesion de caracteres.

Si quisieramos acceder a uno de sus caracteres podriamos acceder usando el indice del caracter como si fuera un array.

$cadena = "Elaina";
echo $cadena[1]; // l

strlen()

Retorna la cantidad de bits que contiene una cadena. Hay caracteres que ocupan mas de un bit (las tildes).

No se recomienda usarla para saber la longitud de la cadena, porque puede generar errores.

$cadena1 = "Elaina";
echo strlen($cadena1) . "<br>"; // 6

$cadena2 = "Canción";
echo strlen($cadena2); // 8

mb_strlen()

Retorna la cantidad caracteres que contiene una cadena.

$cadena1 = "Elaina";
echo mb_strlen($cadena1) . "<br>"; // 6

$cadena2 = "Canción";
echo mb_strlen($cadena2); // 7

strpos()

Devuelve el indice de un caracter en una cadena.

Recibe la cadena a analizar y el caracter que buscara. Devuelve la primera coincidencia de izquierda a derecha.

$cadena = "Elaina";
echo strpos($cadena, "a"); // 2

strrpos()

Devuelve el indice de un caracter en una cadena.

Recibe la cadena a analizar y el caracter que buscara. Devuelve la primera coincidencia de derecha a izquierda.

$cadena = "Elaina";
echo strrpos($cadena, "a"); // 5

str_contains()

Evalua si una cadena se enncuentra dentro de otra.

$cadena = "Hola Mundo";
echo str_contains($cadena, "Mundo") ? "Verdadero" : "Falso"; // Verdadero

str_starts_with()

Verifica si una cadena empieza con otra.

$cadena = "Hola Mundo";
echo str_starts_with($cadena, "Hola") ? "Verdadero" : "Falso"; // Verdadero

str_ends_with()

Verifica si una cadena termina con otra.

$cadena = "Hola Mundo";
echo str_ends_with($cadena, "Mundo") ? "Verdadero" : "Falso"; // Verdadero

strcmp()

Compara dos cadenas.

Si la primera es mas grande que la segunda devuelve 1.

$cadena1 = "123456";
$cadena2 = "1234";

echo strcmp($cadena1, $cadena2); // 1

Si la primera es mas pequenia que la segunda devuelve -1.

$cadena1 = "1234";
$cadena2 = "123456";

echo strcmp($cadena1, $cadena2); // -1

Si son estrictamente iguales (en mayusculas y minusculas) devuelve 0.

$cadena1 = "Palabra";
$cadena2 = "Palabra";

echo strcmp($cadena1, $cadena2) . "<br>"; // 0

$cadena3 = "Palabra";
$cadena4 = "palabra";

echo strcmp($cadena3, $cadena4) . "<br>"; // -1

$cadena5 = "palabra";
$cadena6 = "Palabra";

echo strcmp($cadena5, $cadena6); // 1

strcasecmp()

Tiene la misma funcionalidad que el metodo strcmp(), con la diferencia que le da igual las mayusculas y minusculas.

$cadena1 = "Palabra";
$cadena2 = "Palabra";

echo strcasecmp($cadena1, $cadena2) . "<br>"; // 0

$cadena3 = "Palabra";
$cadena4 = "palabra";

echo strcasecmp($cadena3, $cadena4) . "<br>"; // 0

$cadena5 = "palabra";
$cadena6 = "Palabra";

echo strcasecmp($cadena5, $cadena6); // 0

substr()

Retorna una porcion especificada de una cadena.

El primer parametro indica desde donde inicia la extraccion de la subcadena.

$cadena = "0123456789";

echo substr($cadena, 3) . "<br>"; // Desde el 3: 3456789

echo substr($cadena, -1) . "<br>"; // Desde el ultimo valor: 9

echo substr($cadena, -2); // Desde el penultimo valor: 89

El segundo parametro indica cuantos caracteres seran extraidos de la cadena.

$cadena = "0123456789";

echo substr($cadena, 0, 2) . "<br>"; // Dos caracteres: 01

echo substr($cadena, 3, 3) . "<br>"; // Tres caracteres: 345

echo substr($cadena, -5, 3); // Tres caracteres: 567

str_replace()

Reemplaza caracteres de una cadena por otros.

$cadena = "Hola Mundo";

$reemplazo = str_replace("Mundo", "Soy Joel", $cadena);

echo $cadena . "<br>"; // "Hola Mundo"

echo $reemplazo; // "Hola Soy Joel"

Si la palabra esta repetida varias veces, la reemplaza varias veces.

$cadena = "Hola Mundo. Hola Mundo";

$reemplazo = str_replace("Mundo", "Soy Joel", $cadena);

echo $cadena . "<br>"; // "Hola Mundo. Hola Mundo"

echo $reemplazo; // "Hola Soy Joel. Hola Soy Joel"

strtolower()

Convierte los caracteres de una cadena a minusculas.

$cadena = "Mis Waifus son Elaina, Rapthalia, y Asuna";

$minusculas = strtolower($cadena);

echo $cadena . "<br>"; // Mis Waifus son Elaina, Rapthalia, y Asuna

echo $minusculas; // mis waifus son elaina, rapthalia, y asuna

strtoupper()

Convierte los caracteres de una cadena a mayusculas.

$cadena = "Mis Waifus son Elaina, Rapthalia, y Asuna";

$mayusculas = strtoupper($cadena);

echo $cadena . "<br>"; // Mis Waifus son Elaina, Rapthalia, y Asuna

echo $mayusculas; // MIS WAIFUS SON ELAINA, RAPTHALIA, Y ASUNA

uc/lcfirst()

Convierten el primer caracter de una cadena a mayuscula (uc: upper case) o minuscula (lc: lower case).

$cadena = "hola mundo";

$mayuscula = ucfirst($cadena);

echo $mayuscula . "<br>"; // Hola mundo

$minuscula = lcfirst($mayuscula);

echo $minuscula; // hola mundo

ucwords()

Convierte todas las palabras de una cadena en mayyusculas. No hay version de esta funcion que vaya a minusculas.

$cadena = "hola mundo";

$mayusculas = ucwords($cadena);

echo $mayusculas; // Hola Mundo

str_pad()

Retorna un string rellenado segun una longitud especificada con otro string

Recibe 4 parametros: el texto de entrada, la longitud final del string, el texto de relleno, y finalmente el lugar de relleno.

$texto = "hola";

$paddingR = str_pad($texto, 8, "-", STR_PAD_RIGHT);

echo $paddingR . "<br>"; // Hola----

$paddingL = str_pad($texto, 8, "-", STR_PAD_LEFT);

echo $paddingL . "<br>"; // ----Hola

$paddingB = str_pad($texto, 12, "-", STR_PAD_BOTH);

echo $paddingB; // ----Hola----
