range()

Esta funcion se puede usar para rellenar un array en un rango de numeros.

$array = range(10, 15);

print_r($array); // Array ( [0] => 10 [1] => 11 [2] => 12 [3] => 13 [4] => 14 [5] => 15 )

array_fill()

Crea un array relleno con un valor.

Recibe 3 parametros: primero el indice numerico donde empiezan las llaves, segundo la cantidad de elementos del array, y por ultimo el valor con el que va a ser rellenado.

$array = array_fill(0, 4, "Dinosaurio");

print_r($array); Array ( [0] => Dinosaurio [1] => Dinosaurio [2] => Dinosaurio [3] => Dinosaurio )

array_key_exists()

Verifica si la clave dada o el indice dado existen dentro del arreglo.

$array = ["cero", "uno", "dos", "tres", "waifu" => "Elaina-san"];

$existe = array_key_exists("waifu", $array);

if ($existe) {
    echo "La clave \"waifu\" existe dentro del array";
} else {
    echo "La clave \"waifu\" no existe dentro del array";
} // La clave "waifu" existe dentro del array

array_key_first()

Retorna la primera clave del array.

$array = ["cero", "uno", "dos", "tres", "waifu" => "Elaina-san"];

$clave = array_key_first($array);

echo $clave; // 0

array_key_last()

Retorna la ultima clave del array.

$array = ["cero", "uno", "dos", "tres", "waifu" => "Elaina-san"];

$clave = array_key_last($array);

echo $clave; // waifu

array_keys()

Retorna un array con las claves de otro array.

$array = ["cero", "uno", "dos", "tres", "waifu" => "Elaina-san"];

$claves = array_keys($array);

print_r($claves); // Array ( [0] => 0 [1] => 1 [2] => 2 [3] => 3 [4] => waifu )

array_values()

Retorna todos los valores del array indexandolos numericamente.

$array = ["cero", "uno", "dos", "tres", "waifu" => "Elaina-san"];

$valores = array_values($array);

print_r($valores); // Array ( [0] => cero [1] => uno [2] => dos [3] => tres [4] => Elaina-san )

array_flip()

Intercambia todas las claves con sus valores asociados en un array.

$array = ["cero", "uno", "dos", "tres", "cuatro"];

$volteado = array_flip($array);

print_r($volteado); // Array ( [cero] => 0 [uno] => 1 [dos] => 2 [tres] => 3 [cuatro] => 4 )

array_pad()

Devuelve una copia de array rellenada al tamaño especificado por size con el valor value. Si size es positivo, el array se rellena hacia la derecha, si es negativo hacia la izquierda. Si el valor absoluto de size es menor o igual a la longitud de array, no se lleva a cabo el relleno. Es posible añadir como máximo 1048576 elementos de una sola vez.

array_pad(array $array, int $size, mixed $value);

list()

Se usa para extraer elementos de un arreglo como variables.

$array = [0, 1, 2];

list($a, $b, $c) = $array;

echo $a . " " . $b . " " . $c; // 0 1 2

unset()

Elimina un elemento del arreglo usando su indice.

$array = [0, 1, 2, 3, 4, 5];

unset($array[2]);

unset($array[4]);

print_r($array); // Array ( [0] => 0 [1] => 1 [3] => 3 [5] => 5 )

Tambien elimina todo el arreglo si no tiene indice.

$array = [0, 1, 2, 3, 4, 5];

unset($array);

print_r($array); // (!) Error: undefined

array_map()

Aplica un callback a los elementos de los arrays especificados.

Devuelve un array que contiene el resultado de aplicar una funcion callback a cada elemento de cada array usado como argumento de la funcion array_map().

El numero de parametros que acepta la funcion callback debe coincidir con la cantidad de arrays pasados a array_map(). Los arrays pasados de mas se ignoran, y se produce un error si no se pasan suficientes.

$array = [1, 2, 3, 4, 5];

function cubo(int $numero = null): int
{
    $cubo = $numero * $numero * $numero;

    return $cubo;
}

$cubos = array_map("cubo", $array);

print_r($cubos); // Array ( [0] => 1 [1] => 8 [2] => 27 [3] => 64 [4] => 125 )
