Que son los Arreglos?

Son conjuntos de datos (de cualquier tipo), ordenados segun un indice (inicia en 0) que indica su posicion.

Para recuperar un dato del arreglo usamos la posicion (el indice) para acceder al dato.

$waifus = ["Elaina-san", "Asuna", "Rapthalia", "Hifumi-senpai", "Ninym", "Gab-chan"];

echo $waifus[0] . "<br>"; // Elaina-san
echo $waifus[5] . "<br>"; // Gab-chan

$datos = ["Elaina-san", true, "Rapthalia", 13.56, "Ninym", 12];

echo $datos[4] . "____" . $datos[5]; // Ninym____12

Estructura interna

Un array en PHP es en realidad un mapa ordenado. Un mapa es un tipo de datos que asocia valores con claves. Este tipo se optimiza para varios usos diferentes; se puede emplear como un array, lista (vector), tabla asociativa (tabla hash - una implementación de un mapa), diccionario, colección, pila, cola, y posiblemente más. Ya que los valores de un array pueden ser otros arrays, también son posibles árboles y arrays multidimensionales.

Una explicación sobre tales estructuras de datos está fuera del alcance de este manual, aunque se proporciona al menos un ejemplo de cada uno de ellos. Para más información, consulte la extensa literatura que existe sobre este amplio tema.

Un array puede ser creado con el constructor del lenguaje array(). Éste toma cualquier número de parejas clave => valor como argumentos.

array(
clave => valor,
clave2 => valor2,
clave3 => valor3,
...
)

Además, se darán los siguientes amoldamientos de clave:

  • Strings que contiene un decimal válido integer, a menos que el número esté precedido por un signo +, será amoldado al tipo integer. Por ejemplo, la llave "8" será en realidad almacenado como 8. Por otro lado "08" no será moldeado, ya que no es un entero decimal válido.
  • Un floats también será amoldado a integer, lo que significa que la parte fraccionaria se elimina. P.ej., la clave 8.7 en realidad será almacenada como 8.
  • Un booleano son amoldados a integers también, es decir, la clave true en realidad será almacenada como 1 y la clave false como 0.
  • Un null será amoldado a un string vacío, es decir, la clave null en realidad será almacenada como "".
  • Los arrays y los objects no pueden utilizarse como claves. Si se hace, dará lugar a una advertencia: Illegal offset type.

Si varios elementos en la declaración del array usan la misma clave, sólo se utilizará la última, siendo los demás sobrescritos.

$array = array(
    1 => "a",
    "1" => "b",
    1.5 => "c",
    true => "d",
);

print_r($array); // Array ( [1] => d )

print_r()

Muestra el valor de una variable, cadena de texto u objeto/array con un formato legible.

Es recomendable usarla con arrays u objetos dado su potencial para mostrar este tipo de contenidos.

$waifus = ["Elaina-san", "Asuna", "Rapthalia", "Hifumi-senpai", "Ninym", "Gab-chan"];

print_r($waifus); // Array ( [0] => Elaina-san [1] => Asuna [2] => Rapthalia [3] => Hifumi-senpai [4] => Ninym [5] => Gab-chan )

var_dump()

Esta función muestra información estructurada sobre una o más expresiones que incluye su tipo y valor. Los arreglos y objetos se exploran para mostrar su estructura.

Se recomienda usar para saber el tipo y valor de las variables.

$numero = 23; $texto = "abcd"; float: $decimal = 12.345; $booleano = 10 > 5;

var_dump($numero); echo "<br>"; // int(23)
var_dump($texto); echo "<br>"; // string(4) "abcd"
var_dump($decimal); echo "<br>"; // float(12.345)
var_dump($booleano); echo "<br>"; // bool(true)

$array = [0, 1, 2, 3];

var_dump($array); // array(4) { [0]=> int(0) [1]=> int(1) [2]=> int(2) [3]=> int(3) }

is_array()

Sirve para saber si una variable contiene un Arreglo:

$array = [1, 2, 3, 4, 5]; $cadena = "Hola Mundo";

echo is_array($array) ? "Verdadero" . "<br>" : "Falso" . "<br>"; // Verdadero

echo is_array($cadena) ? "Verdadero" . "<br>" : "Falso" . "<br>"; // Falso

Agregar nuevos datos

Se hace indicando la posicion (ya existente o nueva) y escribiendo o reescribiendo el dato.

$waifus = ["Elaina-san", "Asuna", "Loli mapache XD"];

print_r($waifus); echo "<br><br>"; // Array ( [0] => Elaina-san [1] => Asuna [2] => Loli mapache XD )

$waifus[2] = "Rapthalia";
$waifus[3] = "Hifumi-senpai";
$waifus[4] = "Ninym";
$waifus[5] = "Gab-chan";
$waifus[6] = "Yun-chan";

print_r($waifus); // Array ( [0] => Elaina-san [1] => Asuna [2] => Rapthalia [3] => Hifumi-senpai [4] => Ninym [5] => Gab-chan [6] => Yun-chan )

Tambien es posible agregar un nuevo dato despues de la ultima posicion sin escribir el nuevo indice.

$waifus = ["Elaina-san"];

$waifus[] = "Asuna";
$waifus[] = "Rapthalia";

print_r($waifus); // Array ( [0] => Elaina-san [1] => Asuna [2] => Rapthalia )

Incluso se podria crear el array desde cero usando solo esta sintaxis.

$waifus[] = "Elaina-san";
$waifus[] = "Asuna";
$waifus[] = "Rapthalia";

print_r($waifus); // Array ( [0] => Elaina-san [1] => Asuna [2] => Rapthalia )

Indices literales

Permite asignar un texto como indice a los datos del array. Util para etiquetar informacion.

$waifus = [
    "Favorita <3" => "Elaina-san",
    "Segunda" => "Asuna",
     "Tercera" => "Rapthalia"
];

print_r($waifus); // Array ( [Favorita <3] => Elaina-san [Segunda] => Asuna [Tercera] => Rapthalia )

Cuando se empiezan a agregar datos sin indice de texto desde hay empieza a asignar los indices numericos.

$waifus = [
"Favorita <3" => "Elaina-san",
"Hifumi-senpai",
"Segunda" => "Asuna",
"Tercera" => "Rapthalia",
"Yun-chan"
];

print_r($waifus); // Array ( [Favorita <3] => Elaina-san [0] => Hifumi-senpai [Segunda] => Asuna [Tercera] => Rapthalia [1] => Yun-chan )

Arreglos multidimensionales

Se refiere a arreglos que contienen otros arreglos, es decir a Matrices de datos.

$personas = [
    "programador" => ["nombre" => "Joel", "edad" => "18"],
    "diseniador" => ["nombre" => "Fernando", "edad" => "18"],
    "tester" => ["nombre" => "Kirito", "edad" => "16"]
];

echo $personas["tester"]["nombre"] . "<br>"; // Kirito
echo $personas["programador"]["nombre"]; // Joel
