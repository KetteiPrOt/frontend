Paradigma POO

El paradigma de POO nos indica que debemos abstraer nuestro codigo y escribirlo como si se tratase de manipular objetos de la vida real.

Los Objetos se crean mediante "intanciar" una Clase. Una Clase es una plantilla que contiene los atributos y metodos comunes de un conjunto de Objetos.

Instanciar: crear en memoria un ejemplar de un conjunto de datos y c√≥digo definido por una Clase o estructura.

Clases y Propiedades

Para crear una clase usamos class. Para agregar propiedades usamos public o private.

class Persona {
    public $nombre, $edad, $rol;
private $waifu;
}

Importar Clase

Para incluir una Clase hay que importar el archivo con la funcion require_once().

require_once("clases/Persona.php");

Instanciar Clase

Para instanciar una Case usamos new.

$usuario1 = new Persona;

var_dump($usuario1); echo "<br>"; // object(Persona)#1 (3) { ["nombre"]=> NULL ["edad"]=> NULL ["rol"]=> NULL }

print_r($usuario1); // Persona Object ( [nombre] => [edad] => [rol] => )

stdClass()

Una clase vacia de PHP para crear Objetos genericos vacios:

$objeto = new stdClass();

print_r($objeto); // stdClass Object ( )

Tambien se pueden crear instanciando una clase anonima:

$objeto = new class{};

print_r($objeto); // class@anonymous Object ( )

is_object()

Esta funcion verifica si el contenido de una variable es un objeto:

$objeto = new Class{};

$array = [1, 2, 3, 4, 5];

echo is_object($objeto) ? "Verdadero" . "<br>" : "Falso" . "<br>"; // Verdadero

echo is_object($array) ? "Verdadero" . "<br>" : "Falso" . "<br>"; // Falso

Asignar y Acceder Valores

Para asignar valores a las propiedades de los objetos usamos el operador ->.

Al intentar acceder o asignar valores directamente a una propiedad private se va a producir un error.

$usuario1 = new Persona;
$usuario1->nombre = "Joel";
$usuario1->edad = 18;
$usuario1->rol = "Programador";

$usuario2 = new Persona;
$usuario2->nombre = "Elaina-san";
$usuario2->edad = 14;
$usuario2->rol = "Bruja Ceniza";

print_r($usuario1); // Persona Object ( [nombre] => Joel [edad] => 18 [rol] => Programador )
echo "<br><br>";
print_r($usuario2); // Persona Object ( [nombre] => Elaina-san [edad] => 14 [rol] => Bruja Ceniza )

Para acceder al valor de un atributo tambien usamos el operador ->.

echo "Usuario 1: " . $usuario1->nombre . ". Edad: " . $usuario1->edad . ". Rol: " . $usuario1->rol . ".";
// Usuario 1: Joel. Edad: 18. Rol: Programador.
echo "<br><br>";
echo "Usuario 1: " . $usuario2->nombre . ". Edad: " . $usuario2->edad . ". Rol: " . $usuario2->rol . ".";
// Usuario 1: Elaina-san. Edad: 14. Rol: Bruja Ceniza.

Getters y Setters

Son funciones que sirven para acceder a las propiedades privadas.

public function setWaifu(string $nombreWaifu = null)
{
    $this->waifu = $nombreWaifu;
}

public function getWaifu(): string
{
    return $this->waifu ? $this->waifu : "No tenemos waifu";
}

{"metodo"}()

Esta es una sintexis especial que sirve para ejecutar metodos de un objeto usando una cadena de texto como su nombre:

class Gato {
    public function saludar(): string
    {
         return "Soy un gato! NYAAA~";
    }
}

$michi = new Gato;

echo $michi->{"saludar"}(); // Soy un gato! NYAAA~

Constructor

Es una funcion que se ejecutara al instanciar el objeto para darle valores y a sus propiedades (asi "construirlo").

Con un constructor no es nesesario declaran antes las variables public.

public function __construct(string $nombre = "", int $edad = 0, string $rol = "")
{
    $this->nombre = $nombre;
    $this->edad = $edad;
    $this->rol = $rol;
}

$usuario1 = new Persona("Joel", 18, "Programador");

$usuario2 = new Persona("Elaina-san", 14, "Bruja Ceniza");
