Bucles e iteracion

Volver al capitulo

while

Se usa para ejecutar un bloque de codigo mientras una condicion se cumpla.

$contador = 0;

while ($contador < 10) {
    echo $contador; // 0123456789
    $contador++;
}

do while

Funciona igual que el bucle while, con la diferencia que de primero ejecuta el codigo y despues pregunta por la condicion.

$contador = 0;

do {
    echo $contador; // 0123456789
    $contador++;
} while ($contador < 10);

for

Es un bucle que ya tiene una estructura definida.

  • $i = 0; variable incial que se usara como contador.
  • $i < 10; el bucle se iterara mientras esta condicion se cumpla.
  • $i++ esta sentencia se ejecutara en cada iteracion para acercar al incumplimiento de la condicion y para el bucle.
for ($i = 0; $i < 10; $i++) {
    echo $i; // 0123456789
}

foreach

Sirve para iterar sobre cada uno de los elementos de un array, como el bucle for of de JavaScript.

$array = ["Joel", "Shimakaze", "Fubuki", "Yuudachi"];

foreach ($array as $elemento) {
    print("$elemento <br>");
}

// Joel
// Shimakaze
// Fubuki
// Yuudachi

Tambien en Arreglos [Clave => Valor] podemos acceder a la Clave en cada iteracion de la siguiente forma:

$array = [
    "otakuLider" => "Joel",
    "waifu1" => "Shimakaze",
    "waifu2" => "Fubuki",
    "waifu3" => "Yuudachi"
];

foreach ($array as $clave => $valor) {
    print("$clave: $valor <br>");
}
// "otakuLider": Joel
// "waifu1": Shimakaze
// "waifu2": Fubuki
// "waifu3": Yuudachi

Sentencia break

Su funcion es detener un bucle al ejecutarse.

$i = 0;

while ($i < 10) {

    if ($i == 6) {
         break;
    }

    echo $i . " "; // 1 2 3 4 5
    $i++;
}

Sentencia continue

Su funcion es detener la iteracion actual y pasar a la siguiente. No funciona con el bucle while.

for ($i = 0; $i < 10; $i++) {

    if ($i == 6) {
         continue;
    }

    echo $i . " "; // 1 2 3 4 5 7 8 9 10
}

exit() y die()

Estas funciones finalizan con el programa PHP, los parentisis son opcionales para pasar un mensaje de error. Ambas funciones son exactamente iguales.

echo "Hola"; // Hola

exit();

echo "Mundo!";
