if else

Esta estructura de control divide el flujo del programa de manera logica segun se cumplan o no ciertas condiciones en la ejecucion del programa.

$edad = 18;

if ($edad >= 18) {
    echo "Eres mayor de edad"; // Eres mayor de edad
} elseif ($edad < 18) {
    echo "Eres menor de edad";
} else {
    echo "No conocemos tu edad";
}

Operador ternario

Este operador tiene tres simbolos (comparacion, ?, :) y funciona como un atajo a la sentencia if else. Ademas se pueden usar para guardar en alguna variable la expresion resultante.

$edad = 18;

$resultado = $edad >= 18 ? "Eres Mayor de edad" : "Eres Menor de edad";

echo $resultado; // Eres Mayor de edad

Anidar Condiciones en operador ternario

Se pueden anidar las condiciones repitiendo el operador ternario de la siguiente forma.

$A = 10; $B = 5;

$indicacion = $A > $B ? "mayor" : ( $A < $B ? "menor" : "igual" );

echo "El valor de A es $indicacion que el valor de B"; // El valor de A es mayor que el valor de B

switch

Es una estructura de seleccion multiple. Se usa para compara el valor de una variable con muchos posibles casos y segun el que se cumpla ejecutar codigo.

$dia = 1;

switch ($dia) {
    case 1:
         echo "Lunes"; // Lunes
         break;
    case 2:
         echo"Martes";
         break;
    case 3:
         echo"Miercoles";
         break;
    case 4:
         echo"Jueves";
         break;
    case 5:
         echo"Viernes";
         break;
    case 6:
         echo"Sabado";
         break;
    case 7:
         echo"Domingo";
         break;
    default:
         echo "Numero no valido";
         break;
}

match

Esta nueva estructura sirve para seleccionar un valor a retornar de una variable.

Como es mucho mas legible, se puede usar como sustituta de la sentencia switch.

$dia = 1;

echo match($dia) {
    1 => "Lunes", // Lunes
    2 => "Martes",
    3 => "Miercoles",
    4 => "Jueves",
    5 => "Viernes",
    6 => "Sabado",
    7 => "Domingo",
    default => "El dia esta fuera de rango"
}
