Tipos de Datos

Volver al capitulo

Integer (Entero)

Numeros enteros positivos y negativos.

$entero = 23;

Float/Double (Reales)

Numeros con coma flotante (decimales) positivos y negativos.

Los numero con pocos decimales son de tipo float, los que tienen muchos son de tipo double.

$decimal = 12.345;

String (Cadena)

Cadenas de caracteres, o el texto. Pueden ir en comillas simples o comillas dobles (las dobles son mejor).

$texto = "Hola Mundo!";

Escape de Comillas

Se puede escapar comillas usando el otro tipo de comillas (', ") o tambien con la barra invertida (mas recomendado).

La barra invertida tambien se puede usar para escapar otros caracteres como el simbolo de las variables ($).

$escapeComillas = 'Mi nombre es "Joel".';
echo $escapeComillas; // Mi nombre es "Joel".

$barraInvertida = "Mi nombre es \"Joel\".";
echo $barraInvertida; // Mi nombre es "Joel".

Boolean (Logicos)

Valor logico que solo admite true o false.

En el sistema booleano true significa 1 y false significa 0.

Cualquier numero que sea evaluado como booleano (ej. en los condicionales), sea positivo o negativo, sera asumido como true. Solo el numero 0 es considerado como false.

Cualquier cadena de texto que sea evaluada como booleano sera considerada true, a menos que este vacia. En el caso de que este vacia sera considerada como false.

$verdadero = true;
echo $verdadero; // 1

$falso = false;
echo $falso; //

Null (Nulo)

Valor que indica la ausencia de valor.

$nulo = null;
echo $nulo; //

var_dump()

Sirve para conocer el tipo de dato primitivo de una variable y su valor.

$A = 23; $B = "Texto"; $C = 12.345; $D = 10 > 5;

var_dump($A); // int(23)
var_dump($B); // string(5) "Texto"
var_dump($C); // float(12.345)
var_dump($D); // bool(true)
