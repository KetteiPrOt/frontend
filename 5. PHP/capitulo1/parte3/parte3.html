Asignacion Simple

Asigna el valor de la derecha al elemento de la izquierda.

&variable = "Valor";

Asignacion por Referencia

Una variable es una referencia a un espacio de la memoria. Cuando asignamos por referencia una variable a otra lo que estamos diciendo es que ambas van a hacer referencia al mismmo espacio y si se altera una se cambia la otra.

$A = 23; $B = &$A; // $B es una referencia para $A

echo "$A - $B \n"; // 23 - 23

$B = 12;

echo "$A - $B"; // 12 - 12

Asignacion Combinada

Sirven para operar con el valor de la misma variable.

Operador Significado
$A += 10; $A = $A + 10;
$A -= 10; $A = $A - 10;
$A *= 10; $A = $A * 10;
$A /= 10; $A = $A / 10;
$A %= 10; $A = $A % 10;
$A **= 10; $A = $A ** 10;

Asignacion de incremento

Para asignar un incremento/decremento a una variable debemos indicar que primero se haga el incremento y despues la asignacion.

$A = 10;
$B = ++$A;

echo "El valor de B es: $B >br<"; // 11
echo "El valor de A es: $A"; // 11

Si lo escribimos al revez va a suceder que primero se hace la asignacion del valor, y despues el incremento/decremento.

$A = 10;
$B = $A++;

echo "El valor de B es: $B >br<"; // 10
echo "El valor de A es: $A"; // 11

Operadores de Comparacion

Funcionan igual que en JavaScript.

Nombres Sintaxis Significado
Igual A == B A es igual a B? (solo valor)
Diferente A != B A es diferente a B? (solo valor)
Identidad A === B A es igual a B en tipo y en valor?
Identidad diferente A !== B A es diferente a B en tipo y en valor?
Mayor que A > B A es mayor que B?
Menor que A < B A es menor que B?
Mayor o igual que A >= B A es mayor o igual que B?
Menor o igual que A <= B A es menor o igual que B?

Operadores Logicos

Los textos y los numeros se consideran true cuando son evaluados como booleanos. Solo un texto vacio (""), el valor nulo (null), o el numero cero se consideran false.

Nombres Sintaxis Significado
AND && Comprueba que se cumplan todas las proposiciones para que la operacion sea verdadera
OR || Comprueba que se cumplan al menos una de las proposiciones para que la operacion sea verdadera
NOT ! Niega la proposicion, convirtiendola en su contrario
