Varibles

Una variable es un espacio de memoria que sirve para almacenar un dato.

El nombre de la variable es sensible a mayusculas y minusculas.

PHP es un lenguaje de tipado debil, las variables no nesecitan ser declaradas antes de ser usadas.

$variable = 'dato';

Estilos de escritura

Hay varios estilos de ecritura:

Nombre Ejemplo
Lower Camel Case $miVariable
Upper Camel Case $MiVariable
Snake Case $mi_variable
Kebab Case $mi-variable

Se recomienda usar para programacion el estilo Lower Camel Case.

echo

Muestra el resultado de una expresion en el documento HTML.

$variable = "Hola Mundo!";

echo $variable; // Hola Mundo!

Tambien se puede usar esta funcionalidad en una etiqueta php con esta sintaxis abreviada:

<?php
    $saludo = "Hola Mundo!"; ?>

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Documento</title>
</head>
<body>

    <p> <?= $saludo // Hola Mundo! ?> </p>

</body>
</html>

Concatenacion

Para concatenar textos se usa el operador "." (punto), cuando escribimos una cadena con comillas dobles podemos utilizarla como un template string.

$nombre = 'Joel';

echo 'Hola ' . 'Mundo!'; // Hola Mundo!
echo "Mi nombre es $nombre"; // Mi nombre es Joel

Constantes (define)

Se usan para guardar datos que no varian a lo largo del programa.

Para crear una constante se usa el metodo define quien recibe dos valores: el nombre de la constante y su valor.

Para hacer uso de una constante simplemente se escribe su nombre. Se recomienda nombrarlas en mayusculas y usar "_" para nombres compuestos.

define('NUMBER_PI', 3.14);

echo NUMBER_PI; // 3.14

defined()

Este metodo se usa para saber si una constante ha sido definida usando su nombre.

Si ha sido definida retorna 1 (true), caso contrario nada (0, false).

define('NUMBER_PI', 3.14);

$respuesta = defined('NUMBER_PI');

echo $respuesta; // 1

isset()

Solo trabaja con variables. Devuelve true si el valor de una variable no es null.

$variable = "Hola";

$definida = isset($variable) ? "verdadero" : "falso";

echo $definida; // verdadero

empty()

Devuelve true si el valor de una variable es null.

$variable = "Hola";

$vacia = empty($variable) ? "verdadero" : "falso";

echo $vacia; // falso

unset()

Destruye una variable

$variable = "Hola Mundo";

unset($variable);

echo isset($variable) ? "Si Esta" : "No Esta"; // No Esta

Algunas Contantes Utiles

Las siguientes constantes ya vienen predefinidas por PHP:

PHP_VERSION

Esta constante contiene la version de PHP.

echo PHP_VERSION; // 8.2.0

PHP_OS

Contiene el sistema operativo donde se esta ejecutando PHP.

echo PHP_OS; // WINNT

PHP_EXTENSION_DIR

Contiene la ruta (el directorio) de las extensiones.

echo PHP_EXTENSION_DIR; // C:\php\ext

PHP_SAPI

Contiene la API del servidor (Server API).

echo PHP_SAPI; // apache2handler

__Line__

Contiene en que linea del archivo se esta ejecutando la linea de codigo.

echo __LINE__; // 18

__FILE__

Contiene la ruta del archivo actual.

echo __FILE__; // C:\xampp\htdocs\frontend\5. PHP\programa.php

$_SERVER["DOCUMENT_ROOT"]

Contiene la raiz del servidor.

$patch = $_SERVER["DOCUMENT_ROOT"];

echo $patch; // C:/xampp/htdocs

$_SERVER["REQUEST_URI"]

Contiene la URI del sitio.

$patch = $_SERVER["DOCUMENT_ROOT"];

echo $patch; // /5.%20PHP/capitulo1/parte1/parte1.html

$_SERVER["REQUEST_METHOD"]

Contiene el metodo usado para hacer la solicitud al servidor.

$method = $_SERVER["REQUEST_METHOD"];

echo $method; // GET
