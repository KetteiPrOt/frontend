php_error_log

La funcion ini_set() cambia el valor de una configuracion en el interprete PHP durante la ejecucion del script. El valor se restaura al finalzar la ejecucion.

En esta ocasion la vamos a usar para lograr que cuando haya un errore se escriba automaticamente en el archivo "php_error_log".

ini_set("display_errors", 1);

ini_set("log_errors", 1);

ini_set("error_log", "C:\\xampp\\htdocs\\webDev\\5. PHP\\php_error_log");

echo $variable; // Produce un error de variable indefinida

try y catch

Esta es una estructura de control que ayuda a manejar errores o excepciones:

try {

    echo $variable;

} catch (Exception $e) {

    echo $e; // Undefined variable

}
