Escapar meta caracter

Cualquier metacaracter dentro de la expresion regular se puede escapar de la misma forma que las comillas en las cadenas de texto: con la barra invertida.

$cadenas = ["Hola", "Hole", "Hol]", "Holi", "Holo", "Holu"];

$expresion = "/hol[ae\]i]/i";

foreach ($cadenas as $cadena) {
    echo preg_match($expresion, $cadena) ? "Verdadero" . "<br>" : "Falso" . "<br>";
}

Meta caracter *

Cuando usamos este metacaracter indicamos que el caracter anterior podria existir opcionalmente en la cadena y si existe, hay la posibilidad de que se este repitiendo repetir inmediatamente un numero indefinido de veces. Esto es util cuando queremos verificar una palabra como "hola", y exista la posibilidad de que se escriba como "holaaa" o "holaaaaaaaaaaaaaaaaaaa".

$cadenas = ["Hola", "Holaa", "Holaaaaaaaaaaaaaaa", "Holaooo", "Holooo", "Hol111"];

$expresion = "/holaa*/i";

foreach ($cadenas as $cadena) {
    echo preg_match($expresion, $cadena) ? "Verdadero" . "<br>" : "Falso" . "<br>";
}

Meta caracteres {}

Se usan para indicar la cantidad de veces que de debe repetir un caracter en la expresion regular.

$cadenas = ["Hola", "Hoola", "Hooola"];

$expresion = "/ho{3}la/i";

foreach ($cadenas as $cadena) {
    echo preg_match($expresion, $cadena) ? "Verdadero" . "<br>" : "Falso" . "<br>";
}
