Pseudoelementos y clases

Volver al indice

:hover

Sirve para establecer nuevos estilos al momento de poner el mouse sobre el elemento.

:root

Sirve para seleccionar la raiz del documento (en este caso html), aqui se crean variables css globales.

:root { --variable1: valor1; --variable2: valor2; --variable3: valor3;... }

:visited

Sirve para alterar el estilo de los enlaces que ya han sido visitados.

Hola Mundo! google youtube facebook

:active

Cambia el estilo de enlaces al momento de hacerles click.

Hola Mundo! google youtube facebook

::placeholder

Cambia el estilo del placeholder de los inputs.

:focus

Cambia el estilo de los elementos al hacerles focus (enfocarlos). Por ejemplo al escribir en un input.

:before

Crea un pseudoelemento antes del elemento seleccionado.

:after

Crea un pseudoelemento despues de elemento seleccionado.
