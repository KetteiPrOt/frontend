:root

:root { --variable1: valor1; --variable2: valor2; --variable3: valor3;... }

:active

Establece nuevos estilos cuando un elemento esta siendo 'activado', es decir, en el evento mousedown de JavaScript.

:nth-child(odd/even/function)

Recibe el termino general de una susecion aritmetica, y utiliza los terminos de la serie para determinar a que elementos hijos seleccionados le aplicara estilos.

Utiliza todos los elementos hijos para la cuenta, pero solo aplicara los estilos a quienes sean parte del conjunto seleccionado (en los ejemplos de abajo, los elementos p).


[odd] Palabre clave la cual refiere a la serie an = {2n+1} = {1, 3, 5, 7...} (elementos impares).

.parent-odd p:nth-child(odd) { color: red; }

Estado de Formularios

:invalid

Establece estilos para los inputs cuando son invalidos

:required

Establece estilos para los inputs que son requeridos

.parent-required input:required { color: red; ... }

:read-only

Establece estilos para los inputs que son de solo lectura

.parent-read-only input:read-only { opacity: .7; ... }

:disabled

Establece estilos para los inputs que estan desabilitados.

.parent-disabled input:disabled { color: red; ... }

:enabled

Establece estilos para los inputs que estan habilitados.

.parent-enabled input:enabled { color: green; ... }

:indeterminate

Establece estilos para inputs de tipo checkbox o radio en estado indeterminado

.parent-indeterminate input:indeterminate { border: 1px solid red; }





:checked

Establece estilos para inputs de tipo checkbox o radio en estado chekeado

.parent-indeterminate input:checked { border: 1px solid red; ... }





:default

Establece estilos para los elementos que estan seleccionados por defecto.

Por ejemplo, inputs de tipo radio y tipo checkbox con el atributo checked, o elementos option con el atributo selected
