Transiciones

Volver al indice

transition

Sirve para establecer transiciones fluidas para los cambios de estilo en los elementos.

Resumen [duration property delay timing-function]. transition: 2s background 0.5s ease;

transition-duration

Segundos [s]. Sirve para establecer la duracion de la transicion.

Duration de 5 segundos

transition-property

Propiedad/todas [property/all]. Sirve para establecer la propiedad a la que se le aplicara la transicion. Amenudo se usa el valor all para aplicar la transicion a todas.

transition-delay

Segundos [s]. Sirve para establecer un tiempo de retraso (delay) antes del cual se iniciara la transicion.

Delay de 2 segundos

transition-timing-function

Funcion tiempo[ease/linear]. Sirve para definir una funcion de tiempo en la que avanzara la transicion.

transicion linear
transicion ease
