Animaciones

Volver al indice

@keyframes

Contiene los puntos donde va a haber un cambio dentro de la animacion.

@keyframes nombreAnimacion { from { propiedad: valor1; } to { propiedad: valor2; } }

animation-name

Recive como valor el nombre que la animacion (keyframes) que va a ejecutarse en el elemento.

animation-name: nombreAnimacion;

animation-duration

Segundos [s]. Esta propiedad define el tiempo que va a durar la animacion.

animation-duration: ?s;

animation-iteration-count

Numero cualquiera/infinito [X/infinite]. Esta propiedad define la cuenta de repeticion de la animacion permitiendo que esta se ejecute mas de una sola vez. Para que la animacion se repita infinitamente se puede usar el valor 'infinite'.

name: cambiarFondo;
duration: 2s;
count: infinite;

Ejemplo de animacion cambiando ancho:

name: cambiarAncho;
duration: 2s;
count: infinite;

animation-timing-function

Permite establecer la funcion de tiempo de la animacion.

Linear [linear]. La animacion es lineal.

Suave [ease]. La animacion es mas suave al inicio y al final.

animation-delay

Permite establecer un tiempo de retraso en la animacion.

Sin retraso [0s].

Retraso [s]. Retraso de 1 segundo.

@keyframes (cambios)

Podemos crear mas de dos puntos de cambio en la animacion si usamos porcentajes en los keyframes

@keyframes nombreAnimacion { 0% { propiedad: valor1; } %50 { propiedad: valor2; } 100% { propiedad: valor3; } }

Queda un efecto tan bonito en el ancho:

0% {width: 150px;}
50% {width: 300px;}
100% {width: 150px;}

Como en el fondo:

0% {background: skyblue;}
50% {background: pink;}
100% {background: skyblue;}

Se puede hacer con muchos puntos de cambio:

0% {background: #21295C;}
25% {background: #1B3B6F;}
50% {background: #065A82;}
75% {background: #1C7293;}
100% {background: #9EB3C2;}

@keyframes (valores)

Podemos incluir mas de un valor de cambio en cada animacion

@keyframes nombreAnimacion { from {propiedad1:valor1; propiedad2:valor1;...} to {propiedad1:valor2; propiedad2:valor2;...} }
0% {background: skyblue; width: 300px;}
50% {background: pink; width: 400px;}
100% {background: skyblue; width: 300px;}

animation (resumen)

Resumen de las propiedades anteriores.

animation: name duration timing-function iteration-count delay;

animation: animacion 1s linear infinite 0s;

will-change

Usa esta propiedad para optimizar un elemento, que espera cambiar en un futuro cercano, instruyendo al navegador para preparar la animacion nesesaria antes de que esta empieze.

will-change: [auto/scroll-position/contents/transform];
