transform

Permite hacer que los elementos se transformen al cambiar sus estilos.

translate()

Valor de transform. Permite trasladar elementos en animaciones.

Pixeles [X, Y]. Elemento trasladado 200px en X y 0px en Y. Usar valor tranlateX()/translateY() para alterar un parametro especifico.

scale()

Ancho y alto [width, height]. Permite escalar en base a 1 el tama√±o de los elementos.

scale(1,1)
scale(1.5,1.5)

rotate()

Grados sexagesimales [deg]. Permite rotar los elementos en grados sexagesimales.

rotate(0deg)
rotate(45deg)
rotate(90deg)

skew()

Grados y grados sexagesimales [Xdeg, Ydeg]. Sirve para crear triangulos.

skew(0)
skewX(20deg)
skewY(20deg)

transform (resumen)

Se pueden poner varios valores a la vez.

transform: translate(X,Y) scale(width,height) rotate(deg) skew(Xdeg,Ydeg);
