flex-direction

Esta propiedad controla la direccion en la que se organizan los elementos hijos.

En fila (por defecto) [row]. Los elementos se organizan en una fila horizontal.

flex-wrap

Esta propiedad controla los "saltos de linea" que hacen los elementos hijo cuando ya no caben en el ancho del contenedor padre.

No envolver (por defecto) [nowrap]. Lo elementos hijos se acomodan en una sola linea vertical.

Esta propiedad controla el posicionamiento horizontal del contenido.

Al inicio (por defecto) [flex-start]. Pone el contenido a la derecha horizontalmente.

align-items

Esta propiedad controla el posicionamiento vertical de los elementos hijos dentro de una linea. Lo hace alterando la propiedad align-self (alinearse a si mismo) que cada uno de ellos posee. Esta propiedad no puede usarse junto a la propiedad align-content porque sus naturalezas las vuelven incompatibles.

Al inicio (por defecto) [flex-start]. Pone el contenido arriba verticalmente.

Al final [flex-end]. Pone el contenido abajo verticalmente.

align-content

Esta propiedad controla el posicionamiento vertical del contenido dentro del elemento. Lo hace alterando la alineacion de las lineas que contienen a los elementos hijos, por lo que no tendra efecto en contenedores que posean solo una linea de elementos (nowrap). Al activar algun valor de esta propiedad, align-items ya no sera utilizable en el elemento.

Inicial (por defecto) [initial]. Alinea las lineas de la forma que vienen por defecto.
