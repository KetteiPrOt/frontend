flex-grow

Define el crecimiento y el espacio que ocupa cada elemento en relacion con los demas y su contenedor padre. 1 equivale a todo el tamanio, 0.5 la mitad, 0 es nada... y asi.

Normal (por defecto) [initial]. El tamanio que viene definido por defecto y crecimiento normal.

1
2
3
4

Valor de 0 a 1 [1, 1, 1, 0.5]. El tamanio y crecimiento se da en relacion a los demas elementos usando el valor especificado. No se creara una nueva linea hasta que los elementos en la linea se reduzcan al limite (que es su tamanio original, en este caso 150px).

flex-shrink

Define el encogimiento de un elemento de 0 (no se encoge) a 1 (encogimiento normal).

Valor de 1 (por defecto) [1]. El encogimiento es normal. El elemento opaco esta afectado por la propiedad

1
2
3
4

Valor de 0.9 [0.9]. Se encoge un poquito menos que antes.

1
2
3
4

Valor de 0.5 [0.5]. Se encoge mucho menos que antes.

1
2
3
4

Valor de 0 [0]. No se encoge.

Una aplicacion practica muy util: con flex-grow: 1; se puede obtener un elemento que ocupe todo el espacio y si se combina con esto (flex-shrink: 0;) se establece un ancho minimo.

flex-basis

Define el ancho base del elemento mediante el modulo de flexbox en lugar de tomarlo desde la propiedad width.

Pixeles, %, auto (por defecto), rem, em... [px, %, auto, rem, em...]. Ejemplo con 300px. Elemento opaco es afectado por la propiedad.

Al inicio (por defecto) [flex-start]. Alinea el elemento al inicio de la linea. El elemento opaco esta afectado por la propiedad.
