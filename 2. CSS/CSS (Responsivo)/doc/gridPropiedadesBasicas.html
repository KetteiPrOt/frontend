grid-template-columns

Permite definir el numero de columnas que existen dentro del contenedor padre. Para crear una nueva columnas hay que pasarle el tamanio que esta tendra dentro del contenedor padre.

Tamanio automatico [auto auto]. Crea dos columnas, si el ancho del contenedor esta definido (por el flujo en las filas, o definicion por propiedad) ocuparan todo el espacio, caso contrario tendran el ancho que requiera el contenido que llevan dentro.

grid-template-rows

Permite definir el tamanio de las filas que existen dentro del contenedor padre. Para asignarle un nuevo tamanio a una fila hay que pasarle el tamanio que esta tendra dentro del contenedor padre empezando de arriba a abajo.

Tamanio automatico [auto auto]. Define las dos primeras filas en auto, si el alto del contenedor esta definido (por el flujo en las columnas, o definicion por propiedad) ocuparan todo el espacio, caso contrario tendran el ancho que requiera el contenido que llevan dentro.

Unidad de medida fr

Esta es una nueva unidad relativa exclusiva de CSS Grid (fr = fraccion) que nos ayuda a trabajar mejor.
Por ejemplo para definir 6 columnas que abarquen el mismo espacio cada una deberiamos pasarle el valor decimal de 6/10 a cada columna. Para evitar estos calculos molestos existe esta unidad.

En resumen fr (fraccion) es una unidad relativa la cual hace que el tamanio de un elemento se acomode en funcion de los demas, por lo que sera muy util para el disenio responsivo.

minmax()

Esta es una funcion de CSS Grid que se puede usar en las filas y las columnas para establecerles un tamanio minimo y un tamanio maximo.

Aqui se establece tres columnas de igual ancho (1fr), pero la primera tiene un minimo de 100px.
grid-template-columns: minmax(100px, 1fr) 1fr 1fr;

repeat()

Sirve para crear muchas filas o columnas con la misma medida sin nesecidad de escribirlas una por una. Funciona tanto para alterar filas como columnas.

grid-template-columns/rows: repeat(n, m); n= numero de filas/columnas m= medida que tendran
