Consulta de Medios

Este conjuro es el mas importante a la hora de trabajar con disenio responsivo.
Con esta brujeria podemos hacer un consulta del tamanio de pantalla en el que estamos (es el medio, es el ambiente, es el dispositivo), y en base a eso ejecutar otros estilos para el sitio web ademas de los principales. En consecuencia podemos manejar estilos diferentes para dispositivos diferentes y hacer que la pagina se vea bien en cualquier tamanio de pantalla.

@media screen and (max-width:?px){ ... }

Los etilos normales usualmente se trabajan para el tamanio de pantalla mayor a 1200px.
Ideal para PC de escritorio y laptops

Aqui dentro definimos estilos que se ejecutaran en un tamanio de pantalla inferior a 1200px.
Ideal para laptops pequenias

@media screen and (max-width:1200px){ ... }

Aqui dentro definimos estilos que se ejecutaran en un tamanio de pantalla inferior a 992px.
Ideal para laptops tablets

@media screen and (max-width:992px){ ... }

Aqui dentro definimos estilos que se ejecutaran en un tamanio de pantalla inferior a 768px.
Ideal para laptops telefonos

@media screen and (max-width:768px){ ... }

Estos limites que he escrito pertenecen a un framework de CSS llamado Bootstrap 5, el cual estudiaremos mas adelante.
En realidad no hay medidas recomendables ya que siempre es mejor ver poco a poco como corregir el disenio hasta que quede bien, pero esto es un buen marco de referencia.

Prueba reducir el ancho de la pantalla a menos de 768px para ver como el Media Query le cambia de fondo a la pagina
