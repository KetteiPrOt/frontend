grid-auto-rows

Define la altura de las filas pertenecientes a la grid implicita (agregadas de manera automatica). Solo funciona en Grids con el flujo automatico de elementos en las filas (grid-auto-flow: row;).

Igual que sin la propiedad (por defecto) [initial]. Las filas automaticas toman el alto que requiera su contenido.

dense (Valor)

Este valor (dense, denso) pertenece a las dos propiedades anteriores. Sirve para que CSS Grid rellene automaticamente espacios en blanco que puede haber en nuestra Grid porque no cabian los elementos hijos. Cuando se ha aplicado este conjuro, pero aun asi todavia quedan huecos, es porque ya no se pueden rellenar sin alterar las dimensiones de los elementos hijos.

auto-fit (Valor)

Es un valor de la propiedades grid-template-columns/rows. En el disenio responsivo se usa junto a la funcion minmax(), para que la Grid agrege automaticamente nuevas filas o columnas explicitas (depende del flujo automatico) en la Matriz cuando los elementos en su interior se han encogido a su ancho/alto minimo.

Aqui las columnas estan especificadas en 3 . Cuando llevamos a un tamanio mas pequenio ya no caben los tres y se desborda.

grid-template-columns: repeat(3, minmax(100px, 1fr));

Este es el problema que viene a solucionar auto-fit. Crea automaticamente nuevas columnas cuando los elementos ya no caben.

grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));

auto-fill (Valor)

Es un valor de la propiedades grid-template-columns/rows. En el disenio responsivo se usa junto a la funcion minmax(), para que la Grid agrege automaticamente nuevas filas o columnas explicitas (depende del flujo automatico) en la Matriz cuando los elementos en su interior se han encogido a su ancho/alto minimo.

A diferencia de auto-fit (auto llenar) que busca llenar la fila expandiendo los elementos hasta su maximo, auto-fill no tomara en cuenta el maximo sino solo el minimo y por eso buscara mas elementos de abajo para introducirlos en la fila.

grid-template-columns: repeat(3, minmax(100px, 1fr));

CSS Subgrid

Esta es una propiedad maravillosa que permite utiizar las lineas guia (tracks) de un elemento padre para alinear elementos hijos de los elementos hijos... esto es reutilizar la grid del elemento padre en los elementos hijos para alinear los segundos elementos hijos mas profundos.

A dia de hoy que escribo esto, 23 de noviembre de 2022, el soporte para esta propiedad por parte de los navegadores es aun rancio. Solo el 8.6% de toda la red global lo soporta.
