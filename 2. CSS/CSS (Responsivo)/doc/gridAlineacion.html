Grid (Alineacion)

Volver al indice

Hay situaciones en las que al definir el tamanio de algun elemento hijo se desacomodara en una pantalla mas pequenia. Para solucionar esto, y otras cosas mas, utilizamos propiedades de alineacion para los elementos hijos.

justify-items

Controla la alineacion de forma horizontal. Al activar esta propiedad el ancho de los elementos pasara a ser el que requiera su contenido, en lugar de ocupar todo el ancho disponible.
Esta propiedad funciona alterando el valor de la propiedad justify-self (ver mas adelante) de cada elemento hijo.

Por defecto [initial]. Igual que sin activar la propiedad.

justify-self

Controla la alineacion de forma horizontal. Al activar esta propiedad el ancho del elemento pasara a ser el que requiera su contenido, en lugar de ocupar todo el ancho disponible. Esta propiedad se usa en cada elemento hijo para controlar su alineacion horizontal individualmente.

Esta propiedad puede tomar cualquier valor de justify-items.

justify-content

Se usa para alinear horizontalmente la estructura que contiene al grupo de elementos hijos dentro del contenedor grid. Esta estructura esta definida por unas lineas de CSS Grid llamadas tracks (pistas). Esta propiedad sirve para alinear los tracks (las pistas donde se acomodan los elementos... las lineas de la matriz).
