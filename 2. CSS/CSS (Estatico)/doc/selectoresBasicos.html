Propiedad y Valor

Las reglas css tienen un selector, una propiedadad, y un valor

selector { propiedad: valor;}

Selectores basicos

Para seleccionar un elmento por su clase

.class { ... }

Para seleccionar un elemento por su identificador

#id { ... }

Para seleccionar un elemento por su etiqueta

tag { ... }

Para seleccionar varios elementos

.class1, .class2, .class3 { ... }

Para seleccionar un elemento hijo [selector descendente]

.classElemento .classHijo { ... }

Selectores Avanzados

Para seleccionar un elemento por su atributo. Solo puede haber un atributo.

elemento[atributo="valor"] { ... }

Selecciona un elemento hijo directo

.classElemento > .classHijoDirecto { ... }
