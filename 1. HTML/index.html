Etiquetas basicas

Encabezados

Se definen como h1, h2, h3, h4, h5, y h6

Parrafos

Se definen con p [paragraph]

Saltos de linea

Se definen con br [break]

Separador de seccion

Se define con hr



Marcado de texto

Negrita

Se define con strong

Cursiva

Se define con em

Texto superior

Se define con sub

Texto inferior

Se define con sup

Texto tachado

Se define con del

Texto subrayado

Se define con ins

Nota: en css se define este y mas estilos con las propiedades text-decoration-line, style ,y color

Etiquetas Avanzadas

Enlaces

Se definen con a y el atributo href

Si quieres abrirlo en otra ventana usa el atributo target: [target="_blank"]

Lista ordenada

Se definen con ol y dentro cada elemento con li

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Lista desordenada

Se definen con ul y dentro cada elemento con li

  • Elemento 1
  • Elemento 2
  • Elemento 3

Imagenes

Se definen con img y el atributo src (recurso). Tiene varios atributos utiles:

  • width/height; cambia el tamaño en px
  • alt; le da una descripcion
  • title; al sobreponer el cursor se muestra un texto

Tablas

Sirven para mostrar informacion. Se definen con table; dentro thead, tbody y tfoot. Dentro tr para filas y dentro td para columnas

Cabeza 1Cabeza 2Cabeza 3
Celda 1Celda 1Celda 1
Celda 2Celda 2Celda 2
Celda 3Celda 3Celda 3
Pie 1Pie 2Pie 3

Audio

Se define con audio, y sirve para agregar un audio. Para que se pueda reproducir hay que agregarle el atributo controls.

Video

Se define con video, y sirve para agregar un video. Para que se pueda reproducir hay que agregarle el atributo controls.

SVG

Se define con svg, y sirve para insertar una imagen de vectores escalables que no pierden resolucion al hacerle zoom

Iframe

Se define con iframe, y sirve para insertar una pagina web o algun contenido dentro del documento. Se usa principalmente para agregar:

  • Videos de youtube
  • Post de redes sociales
  • Algunos anuncios

Favicon

Se define con link y el atributo [rel="icon"], sirve para agregar un icono a la pagina web. Esta etiqueta se agrega en la cabeza (head) del documento.

>link rel="icon" href="media/icono.ico"<



Formularios

Formulario

Se define con form y el atributo [action] (accion) para indicar el archivo al que se enviaran los datos. Usa el atributo method para definir el metodo al enviar los datos: GET o POST

Entrada

Se define dentro del formulario con input. Es una entrada para que el usuario ingrese datos.

Atributos basicos:

  • value; es el valor que tiene la entrada, los datos ingresados
  • placeholder; describe que funcion tiene la entrada
  • type; define el tipo de entrada (text/submit)
  • name; define el identificador de los datos

Atributos de control:

  • maxlength; maximo de caracteres admitidos
  • minlength; minimo de caracteres admitidos
  • required; obligatorio completar
  • readonly; solo lectura no editar
  • disabled; desablita la entrada

Tipos de entrada

Diversos tipos de entradas para diversos usos

Tipos de control:

  • Tipo email; valida el formato de un correo
  • Tipo password; valida una contraseña
  • Tipo url; valida una url

Entrada de texto en area

Se define con textarea. Sirve para crear una entrada mas amplia y extensible.

Entrada de seleccion

Se define con select. Un conjunto de opciones con diferentes valores

Si se nesecitan crear grupos de diferentes valores, estos se definen con optgroup

Entrada con lista de sugerencias

Se define con una lista de sugerencias creada con datalist enlazada a una entrada usando un id

Entrada de tipo radio

Se define con el atributo type="radio", define un radio de opciones

Entrada de tipo checkbox

Se define con el atributo type="checkbox", define una confirmacion

Etiquetar elemento

Se define con label y la propiedad for, que apunta con el id a un elemento especifico

Grupo de entradas

Se defiene con fieldset y legend, sirve para agrupar un conjunto de entradas

Mas tipos de entradas

Entrada de tipo color

Se define con el atributo type="color", y sirve para recolectar un color

Entrada de tipo numero

Se define con el atributo type="number", y sirve para recolectar un numero. Los atributos de control min y max definen un intervalo limite en la recoleccion


Entrada de tipo rango

Se define con el atributo type="range", y sirve para recolectar en un rango de numeros


15

Si le asociamos una lista de datos se mostraran marcadores en cada valor especificado:

Entrada de tipo boton

Se define con el atributo type="button", y se comporta como un boton

Entradas de tipo fecha y tiempo

Se definen con el atributo type, y existen algunos valores utiles:

date recolecta una fecha value = 'aaaa-mm-dd'
time recolecta una hora value = 'hh:mm'
datetime-local recolecta una fecha y hora value = 'aaaa-mm-ddThh:mm'
month recolecta un mes y año value = 'aaaa-mm'
week recolecta una semana value = 'aaaa-ss'

Entrada de tipo archivos

Se define con type="file", y sirve para recolectar un archivo

Entrada de tipo reiniciar

Se define con type="reset", y sirve para establecer todos los campos del formulario a sus valores preseterminados

Entrada de tipo oculta

Se define con el atributo type="hidden", y crea un campo que no se muestra pero su valor sera enviado al activar la accion del formulario



Meta Datos

Charset

Establece la codificacion del sitio en utf-8 para mostrar caracteres especiales

>meta charset="UTF-8"<

Description

Describe el contenido del sitio web y debe contener entre 140-160 caracteres.

>meta name="description" content="Decipcion del sitio web y su contenido..."<

Author

Sirve para firmar el documento web. No esta de mas agregarla

>meta name="author" content="nombres apellidos"<

Viewport

Sirve para establecer la vista del sitio en otros dispositivos mas pequeños

>meta name="viewport" content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0"<


Etiquetas de Maquetado

Divisor

Se define con div, y sirve para crear una division en la estructura del documento sin significado semantico. Por ejemplo para alterar su estilo con estructuras de clases de CSS.

Hola Mundo!

Abarcador de texto

Se define con span, y sirve para seleccionar una division de texto sin significado semantico en el documento. Por ejemplo para alterar su estilo con estructuras de clases de CSS.

Hola Mundo!!!

Cabezera

Se define con header, y sirve para crear la estructura semantica de la cabezera en el documento

>header< Cabezera del documento >/header<

  • Navegacion

    Se define con nav, y sirve para crear una navegacion en el documento. Generalmente va dentro del header

    >nav< Navegacion >/nav<

Contenido Principal

Se define con main, y sirve para crear la estructura semantica del contenido principal del documento

>main< Contenido principal del documento >/main<

  • Seccion

    Se define con section, y sirve para crear una seccion en el documento

    >section< Seccion >/section<

  • Articulo

    Se define con article, y sirve para crear un articulo en el documento

    >article< Articulo >/article<

Contenido secundario

Se define con aside, y sirve para crear la estructura semantica de un contenido secundario a un lado. Por ejemplo, una lista de articulos adicinales o un indice del contenido principal que se esta leyendo.

>aside< Contenido secundario >/aside<

Pie de Pagina

Se define con footer, y sirve para crear el pie de pagina del documento

>footer< Pie de pagina >/footer<



Etiquetas especiales

Link

Se define con link, y sirve para enlazar un recurso. Por ejemplo, una hoja de estilos CSS:

>link rel="stylesheet" href="css/styles.css"<

Script

Se define con script, y sirve para enlazar un archivo con codio JavaScript o bien escribirlo directamente dentro

>script src="js/script.js"< Codigo JavaScript >/script<

No Script

Se define con noscript, y sirve para crear contenido que se mostrara si el usuario tiene deshabilitado el codigo JavaScript

>noscript< Contenido a mostrar >/noscript<
