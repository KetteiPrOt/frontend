Ejemplos Practicos

Revisa la guia junto a los ejemplos:

.form-control

Se agrega a los inputs para estilizarlos

input.form-control
textarea.form-control

Se puede cambiar el tamanios agregando una clase extra

.form-control-(sm || lg)

.form-label

Se agrega a las etiquetas label para estilizarlas

label.form-label

.form-text

Se usa para estilizar texto en el formulario (debajo de los inputs)

div.form-text

.is-invalid

Esta clase se le agrega a un input para marcarlo como invalido.

input.is-invalid

.invalid-feedback

Estiliza un texto como invalido

input.is-invalid
div.invalid-feedback

.is-valid

Esta clase se le agrega a un input para marcarlo como valido.

input.is-valid

.valid-feedback

Estiliza un texto como valido

input.is-invalid
div.valid-feedback

.form-floating

Esta clase permite crear un input con un elemento label flotante. Es obligatorio que el input tenga la propiedad placeholder

div.form-floating
    input.form-control
    label

div.form-floating
    textarea.form-control
    label

div.form-floating
    select.form-control
         option
    option
    label
