.progress

Define el contenedor principal de una barra de progreso

Relleno

Define el relleno de la barra de progreso.

  • .progress-bar Define el estilo
  • .progress-bar-stripedOpcional. Establece el estilo de lineas rayadas
  • .progress-bar-animatedOpcional. Anima las lineas rayadas
  • role="progressbar" Identifica el rol
  • .w-25 Utilidad de ancho. Se puede usar para definir el ancho
  • Etiqueta Se puede setear una etiqueta escribiendo texto dentro de este elemento

Colores

Se pueden establecer cualquiera de los colores ya conocidos usando la utilidad de background en el relleno

.bg-*color*
