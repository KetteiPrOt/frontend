Que es una ventana modal?

Es una ventana que se abre temporalmente encima del contenido principal para mostrar algun tipo de informacion.

Boton de apertura

Este boton sirve para abrir la ventana

  • .btn.btn-primary Estas clases son para estilizarlo
  • data-bs-toggle="modal" Este atributo le da la funcionalidad
  • data-bs-target="#modalOne" Este id enlaza el boton con una sola ventana

Contenedor principal

Este es el contenedor principal que contiene a toda la ventana modal

  • .modal.fade Estas clases estilizan la ventana y le dan su respectiva animacion
  • id="modalOne" Este es el id que identifica a la ventana
  • tabindex="-1" Propiedad requerida para que funcione correctamente
  • data-bs-backdrop="static" Opcional. Este atributo hace que la ventana no se cierre al hacer click afuera de ella.

Dialogo

El dialogo es el contenido que se mostrara al abrir la ventana modal

  • .modal-dialog Estiliza la caja de dialogos para incluir cualquier contenido en su interior
  • .modal-dialog-centered Opcional. Centra el modal

Contenido

Es una caja por defecto que sirve para mostrar contenido dentro del dialogo.

  • .modal-content Estiliza el contenido de la ventana modal

Encabezado

Se posiciona dentro del contenido y funciona como encabezado del mismo

div.modal-header
    h*.modal-title

Boton Cerrar

Este boton se crea al final del encabezado usando un elemento button:

  • .btn-close Estiliza el boton
  • data-bs-dismiss="modal" Le da la funcionalidad al boton

Cuerpo

Se posiciona dentro del contenido (debajo del encabezado) para mostrar el mensaje de la ventana modal

div.modal-body

Footer

Opcional. El footer va debajo del cuerpo y podria contener por ejemplo botones para cancelar o aceptar el mensaje del modal

div.modal-footer

.modal-dialog-scrollable

Se agrega al dialogo para darle un scroll cuando su contenido se desborde.

div.modal-dialog.modal-dialog-scrollable
