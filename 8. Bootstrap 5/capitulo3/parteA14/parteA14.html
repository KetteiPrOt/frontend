Ejemplos Practicos

Revisa la guia junto a los ejemplos:

.navbar

Estiliza un navbar. En su interior debe contener un contenedor para centrar el contenido

nav.navbar
    div.(container|container-fluid)

.navbar-expand-?

Esta clase determina la dimension en la que el menu se va a expandir.

.navbar-expand-(sm|md|lg|xl|xxl)

.sticky-top

Fija el menu en la parte de arriba con posicion pegajosa

nav.sticky-top

.navbar-brand

Establece un enlace de navegacion como la bandera del navbar

a.navbar-brand

Boton de expancion

Este boton permite la expansion del menu de navegacion

  • .navbar-toggler Establece la posicion del boton
  • data-bs-toggle="collapse" Establece la funcionalidad del boton
  • data-bs-target="#menuOne" Enlaze el menu con el boton

Dentro del boton va una elemento span que renderize el icono

span.navbar-toggler-icon

Contenido

Va despues de a.navbar-brand, y todo el contenido del navbar debe ir dentro de este elemento para que colapse cuando sea responsive

  • .collapse Esta clase permite colapsar el elemento
  • .navbar-collapse Esta clase establece el colapso para el navbar
  • id="menuOne" Este id identifica el menu de navegacion

Navegacion

Se crea con un elemento ul

  • .navbar-nav Esta clase estiliza el elemento como la navegacion
