Que es la Grid de Bootstrap?

Columnas

Bootstrap 5 funciona con una Grid (rejilla) de 12 columnas.

Cada columna puede ser asignada al elemento HTML que corresponda, y reasignada segun el tamanio de pantalla para crear disenios responsivos.

Breakpoints

Son dimensiones de pantalla en las que Bootstrap pasa a considerar otro dispositivo.

Breakpoint Class infix Dimensions
X-Small xs <576px
Small sm >=576px
Medium md >=768px
large lg >=992px
Extra Large xl >=1200px
Extra Extra Large xxl >=1400px

Filas y columnas

La grid de Bootstrap se construye con filas y columnas dentro de un contenedor principal.

  • Clase container: es el contendor principal que contiene toda la estructura del sitio. Puede ser "container-fluid" para eliminar los espaciados a los lados.
  • Case row: son las filas que contienen las columnas del sitio. Puede haber cuantas quieras.
  • Clase col: representa las columnas asignadas a los elementos HTML. Hay un maximo de 12.

Asignar columnas

Las columnas se asignan con la clase col seguida de el numero (n).

class="col-n"

En el caso de que no se asigne numero (n) el numero de columnas sera dividido automaticamente entre los elementos.

Tamanios de columnas

Se asignan Breakpoints a las columnas para que los elementos se reestructuren segun el tamanio de pantalla:

Class column Breakpoint
.col-n <576px
.col-n-sm >=576px
.col-n-md >=768px
.col-n-lg >=992px
.col-n-xl >=1200px
.col-n-xxl >=1400px
