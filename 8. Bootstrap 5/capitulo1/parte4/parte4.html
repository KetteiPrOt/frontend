Ejemplos Practicos

Colores

Hay colores que se pueden usar como background y tambien como color para textos:

bg-? text-?
  • primary
  • secondary
  • success
  • danger
  • warning
  • info
  • light
  • dark
  • body
  • white
  • transparent

h*

Esta clase se palica a los parrafos para darles estilos de encabezados.

Donde * es el encabezado del al 6:

h-(1 ... 6)

display-*

Esta clase se agrega a los encabezados para darles un estilo mas llamativo.

display-(1 ... 6)

lead

Esta clase se agrega a los parrafos para darles una tipografia mas llamativa.

lead

Crear una cita

Para crear una cita o referencia de algun libro podemos usar la sigiente estructura con las siguientes clases:

<figure class="text-center">
    <blockquote class="blockquote">
         Lorem ipsum dolor, sit amet consectetur adipisicing elit. Ipsa, assumenda.
    </blockquote>
    <figcaption class="blockquote-footer">
         Lorem ipsum dolor sit amet.
    </figcaption>
</figure>

font-size

Esta clase modifica el font-size del texto

fs-(1 ... 6)

font-weigth

Esta clase modifica el font-weigth del texto

fw-?
  • normal
  • light
  • bold

font-style

Esta clase modifica el font-style del texto

fst-?
  • normal
  • italic

text-decoration

Esta clase sirve para cambiar la decoracion del texto (el subrayado).

text-decoration-?
  • underline
  • line-through
  • none

Alineacion del texto

La alineacion del texto se da con:

text-?-(start, center, end)

Donde ? puede ser: sm, md, lg, xl, xxl.

Control del Case

Se puede controlar el case del texto con:

text-(uppercase, lowercase, capitalize)
