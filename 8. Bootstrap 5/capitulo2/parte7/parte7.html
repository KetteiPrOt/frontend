Contenedor del Carousel

El contenedor del carousel debe construirse con las siguientes clases y atributos:

  • .carousel .slide Estas clases indican que se trata del componente en cuestion.
  • data-bs-ride="carousel" Este atributo habilita la funcionalidad del carousel.
  • data-bs-theme="dark" Opcional. Cambia el tema del Carousel a oscuro
  • id="carouselOne" Un id personalizado que servira para identificar al carousel.

Interior del Carousel

Dentro del contenedor va el elemento interior que contendra todos los items del Carousel. Debe construirse con las siguientes clases:

  • .carousel-inner Esta clase estiliza el interior del carousel.

Elemento del Carousel

Dentro del interior van los elementos que mostrara el Carousel. Debe construirse con las siguientes clases:

  • .active Solo el primer elemento debe llevar la clase active para que se muestre.
  • .carousel-item Esta clase estiliza el item del carousel.
  • data-bs-interval="1000" Opcional. Establece el intervalo de tiempo en milisegundos para que se realize el cambio de slide.

.carousel-caption

Dentro de cada elemento se pueden agregar subtitulos usando un elemento div.carousel-caption

Botones

Los botones se crean debajo del interior del carousel para cambiar de elemento, y deben tener las sigientes clases y atributos:

  • .carousel-control-(prev || next) Esta clase estiliza el boton
  • data-bs-slide="(prev || next)" Este atributo habilita la funcionalidad del boton
  • data-bs-target="#carouselOne" Este atributo contiene el id del carousel al que pertenece el boton

Dentro de los botones debe ir un elemento <span> que mostrara el icono del boton

  • .carousel-control-prev-icon Esta clase estiliza el icono
  • aria-hidden="true" Este atributo

Indicadores

Los indicadores se crean debajo del contenedor interior del carousel en un elemento div.carousel-indicators

Este elemento debe contener un <button> por cada elemento del carousel. Cada <button> debe contener los siguientes atributos:

  • .active Solo el primer boton debe llevar la clase active para que se muestre activo
  • data-bs-target="#carouselOne" Este atributo conecta el boton con el carousel a travez de su id
  • data-bs-slide-to="0" Este atributon conecta el boton con el elemento del carousel (desde el 0, 1, 2, 3... hasta cuantos elementos haya)

.carousel-fade

Esta clase se coloca en el contenedor principal del Carousel, y cambia la animacion de transicion.

Al usar esta animacion no se admiten los captions
