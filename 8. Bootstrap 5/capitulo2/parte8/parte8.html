Ejemplos Practicos

Revisa la guia junto a los ejemplos:

.btn

Estiliza un boton

.btn-*color*

Cambia de color a los botones. Tenemos 9 colores:

  • primary
  • secondary
  • success
  • danger
  • warning
  • info
  • light
  • dark
  • transparent

.btn-outline-*color*

Crea botones con otro estilo (estilo outline). Usa los mismos colores que btn-*color*.

.btn-link

Cambia la apariencia del boton a un enlace

.disabled

Cambia la apariencia de un boton a desabilitado

.btn-lg

Hace al boton mas grande (large)

.btn-sm

Hace al boton mas pequenio (small)

.w-100

Establece el ancho del boton en 100% del ancho del contenedor padre. Como resultado el boton abarca todo el espacio disponible.

Enlaces como btn

Se pueden estilizar elementos de tipo enlace (<a>) como botones agregandoles las mismas clases.

Input como btn

Se pueden estilizar elementos de tipo input (<input>) como botones agregandoles las mismas clases.
