Alerta descartable

Se puede crear una alerta descartable con las siguientes clases y atributos:

Elemento div.alert

  • .alert-dismissible Indica a Bootstrap que se trata de una alerta descartable.
  • .fade Indica la animacion al descartar la alerta.
  • .show Esta clase muestra la alerta.

Elemento button.btn-close

  • .btn-close Estiliza el boton para cerrar.
  • data-bs-dismiss="alert" Este atributo habilita la funcionalidad del boton.
  • aria-label="cerrar" Etiqueta para los dispositivos de lectura especial.
