Navegar entre ramas

Volver a Git

git diff

Muestra los cambios de un archivo actual en comparacion a su version pasada de cualquier commit.

Se usa el identificador del commit, seguido del archivo que deseeamos comparar.

git diff identificador_commit nombre_archivo

git branch

Muestra las ramas del repositorio y en cual nos encontramos.

git branch

Eliminar rama.

git branch -d nombre_rama

git checkout

Sirve para moverse entre ramas.

Crear una nueva rama.

git checkout -b 'nombre_rama'

Cambiar de rama.

git checkout nombre_rama

Crear una nueva rama regresando a un commit pasado. Al hacerlo la rama se cambia a HEAD (rama temporal).

git checkout identificador_commit

git merge

Junta la rama actual con la rama indicada.

git merge nombre_rama
