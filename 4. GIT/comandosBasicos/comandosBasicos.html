Comandos Basicos

Volver a Git

git init

Este comando incia un repositorio de git en el directorio actual.

git init

git status

Muestra el estado actual del repositorio, es decir si hay cambios para hacer commit o todo esta guardado.

git status

git add

Este comando agrega los cambios no guardados para hacer commit. Recibe como parametro '.' para agregar todos los cambios, o el la ruta de un archivo en particular para agregar sus cambios.

git add . / nombre_archivo

git rm

Elimina un archivo el cual ha sido agregado para hacer commit.

git rm --cached nombre_archivo

git commit

Este comando se usa para hacer un commit y asi guardar los cambios del repositorio.

Recibe como parametro el nombre del commit con la siguiente sintaxis.

git commit -m "Nombre del commit"

git log

Muestra una lista de los commits realizados.

git log

Mostrar las estadisticas de cada commit: '--stat'

git log --stat

Mostrar cada commit en una linea: '--oneline'

git log --oneline

Mostrar una grafica de las ramas: '--graph'

git log --graph

Mostrar una grafica de las ramas en una linea: '--oneline --graph --all'

git log --oneline --graph --all

Archivo .gitignore

Este archivo se usa para excluir archivos del repositorio de forma que Git no los tome en cuenta para los commits.

Para excluir un archivo hay que escribir su nombre con su extension.

archivo.txt
archivo2.pdf
archivo3.html

Para excluir todos los archivos que tengan cierta extension se escribe '*' (todos los archivos) y la extension. Aqui excluye todos los archivos .txt

*.txt
