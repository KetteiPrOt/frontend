SASS

Volver a los niveles

Compilar SCSS a CSS

Se usa el comando sass, la ruta del archivo .scss, y la ruta del archivo .css

sass ruta/archivo.scss ruta/archivo.css

Vigilar cambios

Se usa la opcion -w o --watch, el nombre del archivo .scss, y el nombre del archivo .css

sass -w ruta/archivo.scss ruta/archivo.css

Anidar selectores

SASS permite anidar selectores para no tener que repetirlos.

/* SCSS */
.padre {
    .hijo {
         estilo: valor;
    }
}

/* CSS */
.padre .hijo {
    estilo: valor;
}
