Bases de Datos

Usar Base de Datos

Usar una BBDD por defecto:

USE nombre_base_datos;

Crear Base de Datos

Para crear una nueva base de datos puedo usar indistintamente cualquiera de estas dos sentencias:

CREATE DATABASE nombre_base_datos;

CREATE SCHEMA nombre_base_datos;

Eliminar Base de Datos

Para eliminar una base de datos puedo usar indistintamente cualquiera de estas dos sentencias:

DROP DATABASE IF EXISTS nombre_base_datos;

DROP SCHEMA IF EXISTS nombre_base_datos;

Tablas

Crear Tabla

Despues del tipo de dato se puede referir NOT NULL para no nulos, o DEFAULT NULL para registros que pueden ser nulos.

CREATE TABLE nombre_tabla (
    id INT NOT NULL AUTO_INCREMENT,
    columna_1 tipo_dato NOT NULL,
    columna_2 tipo_dato NOT NULL,
    columna_3 tipo_dato DEFAULT NULL,
    columna_4 tipo_dato DEFAULT NULL,
    ...
    PRIMARY KEY (id)
);

Eliminar Tabla

DROP TABLE IF EXISTS nombre_tabla;

Alterar Tabla

Renombrar tabla:

ALTER TABLE nombre_tabla RENAME AS nuevo_nombre;

Agregar columna:

ALTER TABLE nombre_tabla ADD COLUMN columna_x tipo_dato NOT NULL;

Eliminar columna:

ALTER TABLE nombre_tabla DROP IF EXISTS columna_x;

Modificar columna:

Si se cambias una columna a un tipo de dato mas peque√±o y tienes datos muy grandes, acabaras perdiendo la informacion que no entre en los limites de nuevo tipo de dato.

ALTER TABLE nombre_tabla MODIFY COLUMN columna_x tipo_dato NOT NULL;

Registros

Insertar registro

INSERT INTO nombre_tabla (
    columna_1, columna_2, columna_3...)
VALUES (dato_1, dato_2, dato_3...);

Eliminar registro

Se usa el id del registro que se deseea eliminar:

DELETE FROM nombre_tabla WHERE id = 1;

Actualizar registro

Se usa el id del registro que se deseea actualizar:

UPDATE nombre_tabla SET
    columna_1 = nuevo_dato_1,
    columna_2 = nuevo_dato_2,
    columna_3 = nuevo_dato_3
    ...
WHERE id = 1;
