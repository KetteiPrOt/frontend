Acelerar las busquedas

Los indices son columnas que internamente han sido ordenadas por MySQL para acelerar la busqueda de informacion sobre ellas.

Solo deberiamos agregar indices en las columnas donde estemos seguros que vamos a hacer busquedas, porque agregarlos consume memoria.

ADD INDEX

Agrega un indice a una columna de la tabla:

ALTER TABLE nombre_tabla ADD INDEX (nombre_columna);

Indice de tipo FULLTEXT

Al agregar este tipo especial de indice en columnas varchar y text, MySQL internamente analiza el texto de manera similar a como lo hacen los motores de busqueda para despues poder usar el lenguaje nartural al hacer busquedas en estas columnas.

ALTER TABLE nombre_tabla ADD FULLTEXT(nombre_columna);

Consulta Match Against

Esta consulta ejecutar una busqueda en lenguaje natural sobre columnas con un indice FULLTEXT:

SELECT * FROM nombre_tabla WHERE match(columna_1, columna_2...) against("expresion natural");
