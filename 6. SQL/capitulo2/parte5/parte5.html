Para datos numericos

SUM()

Suma todos los registros de una columna y los retorna en la consulta:

SELECT CAST(SUM(precio) AS FLOAT) AS suma_precios FROM productos;

MAX()

Retorna el mayor registro de tipo numerico de una columna en la consulta:

SELECT MAX(precio) AS mayor_precio FROM productos;

MIN()

Retorna el menor registro de tipo numerico de una columna en la consulta:

SELECT MIN(precio) AS menor_precio FROM productos;

AVG()

Retorna el promedio de todos los registros de tipo numerico de una columna en la consulta:

SELECT CAST(AVG(precio) AS FLOAT) AS promedio_precios FROM productos;

Para datos fecha

DATE_FORMAT()

Cambia el formato de un dato y/o registro de tipo fecha y lo retorna en la consulta.

Para que MySQL entienda que deseamos cambiar de formato debemos especificar el simbolo "%" donde: "d" es dia, "m" es mes, y "Y" es año.

SELECT DATE_FORMAT(fecha_nacimiento, "%d/%m/%Y") AS fn_nuevo_formato FROM personas;

DAY()

Retorna el dia de un dato y/o registro de tipo fecha:

SELECT DAY(fecha_nacimiento) AS dia FROM personas;

MONTH()

Retorna el mes de un dato y/o registro de tipo fecha:

SELECT MONTH(fecha_nacimiento) AS mes FROM personas;

YEAR()

Retorna el año de un dato y/o registro de tipo fecha:

SELECT YEAR(fecha_nacimiento) AS año FROM personas;

SECOND()

Retorna los segundos de un dato y/o registro de tipo fecha-tiempo o tiempo:

SELECT SECOND(estudio_diario) AS segundos FROM personas;

MINUTE()

Retorna los minutos de un dato y/o registro de tipo fecha-tiempo o tiempo:

SELECT MINUTE(estudio_diario) AS minutos FROM personas;

HOUR()

Retorna las horas de un dato y/o registro de tipo fecha-tiempo o tiempo:

SELECT HOUR(estudio_diario) AS horas FROM personas;
