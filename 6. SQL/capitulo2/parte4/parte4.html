Ejecucion

Puedes ejecutar las sentencias de esta parte y de la parte 5 en MySQL Workbench usando la base de datos "practica_funciones" (Este es el Link) y ver sus resultados.

COUNT()

Cuenta el numero de registros retornados en la consulta:

SELECT COUNT(*) AS cantidad_personas FROM personas;

CAST()

Convierte de ser posible un dato a otro tipo.

SELECT precio, CAST(precio AS INT) AS precio_redondeado FROM productos;

SQL_CALC_FOUND_ROWS

Almacena de forma oculta el numero de registros totales encontrados en la consulta. La sentencia LIMIT no tiene efecto sobre esta orden.

SELECT SQL_CALC_FOUND_ROWS * FROM personas LIMIT 0, 3;

FOUND_ROWS()

Recupera el resultado de la la orden SQL_CALC_FOUND_ROWS. Ambas tienen que ser ejecutadas en la misma consulta para que funcione.

SELECT FOUND_ROWS() AS total;

Para datos cadena de texto

CONCAT()

Concatena varios datos y/o registros retornandolos en la consulta como una sola columna:

SELECT CONCAT(nombre, " ", apellidos) AS persona FROM personas;

CONCAT_WS()

Usa el primer argumento automaticamente como separador de cada dato y/o registro concatenado:

SELECT CONCAT_WS(" ", nombre, apellidos, telefono) AS contacto FROM personas;

LENGTH()

Retorna la longitud de un dato y/o registro de tipo cadena de texto:

SELECT nombre, LENGTH(nombre) AS longitud FROM personas;

LOWER()

Cambia a minusculas todos los caracteres de un dato y/o registro de tipo cadena de texto:

SELECT LOWER(nombre) AS nombre_min FROM personas;

UPPER()

Cambia a mayusculas todos los caracteres de un dato y/o registro de tipo cadena de texto:

SELECT UPPER(nombre) AS nombre_may FROM personas;

REPLACE()

Reemplaza los caracteres de un dato y/o registro de tipo cadena de texto por otros.

Reemplaza @gmail.com por @outlook.com de los registros en la columna correos y los retorna en la consulta:

SELECT REPLACE(correo, "@gmail.com", "@outlook.com") AS correo_nuevo FROM personas;

TRIM()

Los espacios en blanco al inicio y al final de un dato y/o registro de tipo cadena de texto:

SELECT TRIM(descripcion) AS descripcion_sinespacios FROM personas;
