Guardar salida

Para guardar la salida del comando echo se puede usar el símbolo '>', seguido del nombre del archivo de destino. Esto escibirá los datos allí. 


    echo 'Hello World!' > file.txt;

Funciona igual que con el comando cat. Si usas dos '>', entonces agrega la información al archivo en lugar de escribirla.

Entrada de texto

Usando el comando cat podemos crear facilmente un script que reciba entrada por teclado. 


    cat > file.txt;

Al ejecutar el programa podemos ingresar texto por la consola.
