Volver a las partes

alias

Te permite definir alias temporales en tu sesión de la consola, pudiendo sustituir una palabra por una serie de comandos. 


alias name=value

alias admin="sudo su"

alias kettei="sudo su kettei"

unalias

Te sirve para quitar el alias que hayas configurado. 


unalias alias

htop

Te permite ver en el terminal los procesos que tienes en funcionamiento para gestionar los recursos de tu ordenador. 


htop

ps

Con este comando puedes visualizar los procesos que se ejecutan en tu sesión del terminal. Te muestra un identificador de cada proceso e información útil sobre él. 


ps

ip

Te dice tu dirección IP y muestra detalles de las interfaces de red. 


ip a
Muestra las interfaces de red y sus direcciones IP

ping

Hace ping a un servidor especificado. 


ping address

echo

Te permite mostrar un texto definido por ti en el terminal, escribiendo el comando seguido del mensaje entre comillas. 


echo text

passwd

Te permite cambiar la contraseña de tu cuenta de usuario. Al escribir solo el comando, te dirá primero que escribas tu contraseña actual y luego la nueva. 


passwd

whoami

Te muestra el nombre de usuario que tienes ahora mismo, por si no recuerdas en qué sesión estás. 


whoami

uname

Te muestra la información sobre la distribución de GNU/Linux que estás usando y su versión, además de otra información útil del sistema operativo. 


uname

uname -a
muestra toda la informacion

neofetch

Te muestra la información sobre tu sistema, la distribución que usas, la versión del kernel, del shell y de tu hardware. Todo ello con un logotipo ASCII de tu distro. 


neofetch

whatis

Este comando seguido de otro comando o el nombre de un programa te muestra su descripción, para cuando no sabes qué hace un comando o qué es un programa. 


whatis command_or_program

exit

Cierra la sesión y cierra el terminal. 


exit

shutdown

Te permite apagar el ordenador desde el terminal. Puedes usarlo junto a now, con el comando shutdown now para apagar ahora mismo, o cambiar now por una fecha para programar el apagado. 


shutdown now

shutdown date
