Volver a las partes

pwd

Te muestra el directorio en el que estás ahora mismo, y así poder saber la estructura de directorios del sistema en el que estás 


pwd
/home/kettei/Desktop

cd

El nombre significa Change Directory, y sirve para cambiar de directorio. Esto quiere decir que sirve para cambiar de carpeta. Si escribes CD a solas, irás a tu Home, tu carpeta personal. 


cd route // vas al directorio de route

cd - // vas al directorio donde estabas antes

ls

Te muestra una lista con los archivos y directorios que hay dentro del directorio actual en el que estás 


ls
file.txt    directoryA

mkdir

Crea un nuevo directorio en el sitio donde estás del sistema de archivos. 


mkdir name

rmdir

Borra y elimina un directorio vacío del sistema. 


rmdir name
