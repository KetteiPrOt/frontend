Volver a las partes

find

Sirve para buscar archivos y directorios en el sistema. 


find directory_path search_parameter

find /path/directory -name file_name.txt

find . -name file_name.txt // busca en el directorio actual

find ./parent_dir/directory -name file_name.txt

Hay varios parametros de busqueda que se pueden usar, e incluso combinar con operadores lógicos:

Parámetro Descripción
-name Filtrar por nombre de archivo
-iname Ignora mayusculas y minusculas
-type Filtrar por tipo de elemento
(f para archivos, d para directorios)
-size, -empty Filtra por tamaño de archivo
-ctime, -mtime, -atime Filtrar por marca de tiempo
-user, -group Filtrar por propietario y grupo
-perm Filtrar por derechos de archivo

Guia detallada de su uso.

cp

Sirve para copiar archivos y directorios de un lugar a otro. 


cp ./dir/file_to_copy.txt /path/file_copied.txt

cp ./empty_dir /path/empty_dir_copy

cp -r ./directory /path/directory_copy // copia recursivamente los archivos en su interior con -r

mv

Sirve para mover archivos y directorios de un lugar a otro, tendrás que usarlo escribiendo lo que quieres mover y a dónde, de forma que desaparezca de donde estaba antes. 


mv ./file.txt /path/file.txt

mv ./dir /path/dir

rm

Elimina archivos y, usando la opcion -r (recursive), también directorios junto a todo su contenido. 


rm ./file.txt

rm -r ./directory

wget

Sirve para descargar y recuperar contenidos de Internet (World Wide Web Get). Admite descargas a través de FTP, SFTP, HTTP y HTTPS. Guía detallada. 


wget https://www.domain.com/uri/file.html // descarga file.html y lo almacena en el directorio actual

wget -O file_renamed.zip https://www.domain.com/file.zip // cambia de nombre al archivo descargado con -O

wget -P /path/directory https://www.domain.com/file.zip // almacena file.zip en directory con -P

head

Muestra las 10 primeras líneas de un archivo, para poder identificarlo bien. 


head ./file.txt

head -n lines_number ./file.txt

Puedes usar este comando para tomar la salida de otro comando y ver solo las primeras lineas de la misma. 


command | head -n lines_number

tail

Muestra las 10 últimas líneas de un archivo, para poder identificarlo bien. 


tail ./file.txt

tail -n lines_number ./file.txt

Puedes usar este comando para tomar la salida de otro comando y ver solo las últimas lineas de la misma. 


command | tail -n lines_number

grep

Busca patrones dentro de un archivo, e imprimirá la línea o líneas que la contengan. Muy útil para encontrar lineas específicas en archivos de configuración. 


grep options pattern file

grep searched_word file.txt

grep searched_word ./* // search in all file of the current directory

Entre las opciones más usadas tenemos:

Opción Descripción
-i la búsqueda no distinguirá entre mayúsculas y minúsculas
-c solo mostrará la cuenta de coincidencias
-r habilita la búsqueda recursiva en el directorio actual
-n muestra el número de línea de las coincidencias
-v muestra las líneas que no coinciden con el patrón que hemos buscado

cat

Sirve para visualizar, unir (cat => concat) y crear archivos. 


cat file.txt // muestra el contenido del archivo

cat -n file.txt // muestra el número de línea

cat > file.txt // escribir el archivo file.txt y escribir su contenido

cat >> file.txt // agrega contenido al archivo file.txt

cat source.txt > file.txt // escribir el archivo file.txt con el contenido de source.txt

cat source_1.txt source_2.txt source_3.txt > new_file.txt

cat source.txt >> file.txt // agrega el contenido de source.txt al archivo file.txt

cat *.txt // muestra el contenido de todos los archivos .txt

cat *.txt | more // usa more para verlo de menos a más

wc

Cuenta el número de palabras de un archivo (Word Count). 


wc file
lines_count words_count characters_count file_name

less

Reduce la cantidad de contenido de un archivo que se ve, para verlo una página a la vez y que sea más fácil de leer. 


less file

touch

El comando touch de Linux se usa principalmente para crear archivos vacíos y cambiar marcas de tiempo de archivos o carpetas. La información de marca de tiempo de los archivos consta de tres atributos: tiempo de acceso, tiempo de modificación y tiempo de cambio. Guía detallada. 


touch options file

chmod

Permite cambiar el modo de un archivo, pudiendo añadir o quitar el modo r (lectura), w (escritura) o x (ejecución). 


chmod options permissions file_or_dir

unzip

Te permite descomprimir un archivo comprimido en formato zip. Después del comando tendrás que escribir el nombre del archivo. 


unzip file_zipped

./

Si escribes este comando seguido de un nombre de archivo, ejecutarás ese archivo con el programa compatible que tengas instalado. 


./executable

kill

Si hay algún programa que no te responde, con el comando kill seguido del nombre del programa puedes forzar su cierre. 


kill program

shred

Este comando seguido del nombre de un archivo borrará el contenido de este archivo de forma repetida para que sea casi imposible de recuperarlo. No solo lo borras, también lo destrozas para que no se recupere. 


shred file
