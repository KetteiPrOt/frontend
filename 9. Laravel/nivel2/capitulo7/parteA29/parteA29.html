Ruta dashboard

Esta ruta esta controlada por el Middleware 'auth:sanctum', quien verificara si un usuario tiene la sesion iniciada para acceder. Si no la tiene entonces lo redireccionara a la pagina de login.

Route::middleware([
    'auth:sanctum',
    config('jetstream.auth_session'),
    'verified'
])->group(function () {
    Route::get('/dashboard', function () {
         return view('dashboard');
    })->name('dashboard');
});

Plantillas Principales

Hasta Laravel 7 usabaamos el sistema de plantillas Blade para extender de los documentos ubicados en la carpeta /resourses/views/layouts/, ahora usamos los componentes de Blade para esta labor.

Centremonos en la plantilla views/layouts/app.blade.php y sus nuevos metodos.

app()->getLocale()

Establece el idioma del atributo HTML lang

Accede a la configuracion de la aplicacion (/config/app.php) con el metodo app() y recupera el valor del idioma en la llave 'locale' del arreglo con getLocale().

<html lang="{{ str_replace('_', '-', app()->getLocale()) }}">
    ...
</html>

config()

Accede a la configuracion de la aplicacion, en este caso esta accediendo al nombre establecido finalmente en el archivo .env

<title>{{ config('app.name', 'Laravel') }}</title>

Drirectiva @vite

Esta directiva invoca a Vite para que carge en el documento estilos almacenados en la carpeta resourses.

@vite(['resources/css/app.css', 'resources/js/app.js'])

Directiva @auth

Esta directiva de Blade sirve para establcer un condicional en caso de que el usuario este o no este autentificado:

@auth
    ...
@else
    ...
@endauth
