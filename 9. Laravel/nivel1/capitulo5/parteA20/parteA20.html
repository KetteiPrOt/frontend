Ruta destroy

Laravel recomienda que cuando eliminamos un registro lo hagamos mediante el metodo delete.

Route::delete('/cursos/{course}', 'destroy')->name('courses.destroy');

Metodo destroy

El metodo destroy sencillamente elimina un registro:

public function destroy(Course $course)
{
    // Eliminar registro
    $course->delete();

    return redirect()->route('courses.index');
}

Boton Eliminar

Creamos el boton en la interfaz para el usuario:

<form action="{{route('courses.destroy', $course)}}" method='post'>

    @csrf

    @method('delete')

    <button type="submit">Eliminar</button>
</form>
