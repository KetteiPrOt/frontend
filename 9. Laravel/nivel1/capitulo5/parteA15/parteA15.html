Paginacion

Las personas nunca van a leer por ejemplo 55000 registros, por esdo debemos paginarlos.

paginate()

Este metodo pagina los registros, por defecto de 15 en 15.

$courses = Course::paginate();

return view('courses.index', compact('courses'));

Si queremos aceder a alguna pagina debemos pasar por la url el dato page con el numero de la pagina que deseeamos ver

laravel.test/cursos?page=2

Paginacion de Tailwind

Puedes usar los estilos de la paginacion de tailwind simplemente incluyendo su cnd en el archivo y ya tendras lista la paginacion

https://cdn.tailwindcss.com

Usar bootstrap

Laravel por defecto usa tailwind para la paginacion, pero pueddes usar bootstrap modificando el archivo:

app/Providers/AppServiceProvider.php

En este archivo debes usar la clase:

use Illuminate\Pagination\Paginator;

Y en el metodo boot() usar el metodo estatico useBootstrap() de la clase Paginator

public function boot(): void
{
    Paginator::useBootstrap();
}

Nombrar Rutas

Por el bien de simplificar la labor de crear la navegacion en nustra aplicacion podemos asignar nombres alternativos a las rutas de navegacion con el metodo name() en el archivo de rutas:

Route::get('/cursos', 'index')->name('courses.index');
Route::get('/cursos/crear', 'create')->name('courses.create');

Ahora podemos invocar las URLs de estas rutas en las vistas blade usando el metodo route()

<a href="{{route('courses.index')}}">Principal</a> <br>
<a href="{{route('courses.create')}}">Crear registro</a>

Rutas con Parametros

Pero lo anterior se hace diferente cuando hablamos de rutas con parametros

Aqui como la ruta espera un parametro debemos pasarselo por el metodo route() como segundo parametro:

<a href="{{route('courses.show', $course->id)}}"> {{$course->name}} </a>
