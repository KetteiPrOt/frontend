view()

Este metodo solo lo vamos a usar cuando deseamos mostrar contenido estatico sin conexion a la base de datos.

Route::view('nosotros', 'about')->name('about');

dump()

Este metodo sirve para mostrar un valor en documento (verter)

{{dump(true);}}

Ruta actual

Con los metodos request() y routeIs() podemos determinar cual es la ruta que esta activa actualmente.

request()->routeIs('route_name')

El metodo routeIs() recibe el nombre de la ruta y retorna true o false si esta activa o no lo esta.

Estilizar enlaces

Con lo anterior ya podemos estilizar los enlaces usando un condicional, dependiendo de cual seccion este activa:

<a
    href="{{route('courses.index')}}"
    class="
         {{
              (request()->routeIs('courses.index') || request()->routeIs('courses.show')) ? 'active' : ''
         }}
    "
>Principal</a>

Modularizar template

Vamos a mover la logica del menu de navegacion a otro archivo en la carpeta partials:

views/layouts/partials/navigation.blade.php

Copiamos y pegamos el menu en ese archivo, y lo incluimos en el template principal con la directiva @include():

@include('layouts.partials.navigation')
