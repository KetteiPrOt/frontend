Contenido de Elementos

Las siguientes propiedades sirven para obtener y modificar el contenido de los elementos.

textContent

Contiene el texto cualquier nodo (elemento, comentario, atributo...), esto incluye los espacios y saltos de linea del codigo. No incluye etiquetas.

Hola! Soy un murcielago


let muercielago = document.getElementById('murcielago');

let texto = murcielago.textContent;

console.log(texto);
/*
    Hola! Soy un murcielago
*/

innerText

Contiene el texto de un elemento tal cual lo renderiza el navegador. No incluye etiquetas. Si hay texto oculto por algun atributo hidden, innerText tampoco lo mostrara.

Hola! Soy un cangrejo


let cangrejo = document.getElementById('cangrejo');

let texto = cangrejo.innerText;

console.log(texto); // Hola! Soy un cangrejo

outerText

Contiene el texto junto a la etiqueta de un elemento tal cual lo renderiza el navegador. No incluye etiquetas. Si hay texto oculto por algun atributo hidden, outerText tampoco lo mostrara.

Importante: Si se cambia el valor de esta propiedad acabaras reemplazando tambien la etiqueta del elemento junto al texto que contiene.

Hola! Soy una iguana en un elemento 'p'



let iguana = document.getElementById('iguana');

iguana.outerText = 'Ahora soy iguana en el div elemento'; // elemento p de iguana murio

innerHTML

Contiene el codigo html interno de un elemento.

Hola! Soy un pulpo


let pulpo = document.getElementById('pulpo');

console.log(pulpo.innerHTML);
/*
    Hola! Soy un >i< pulpo >/i<
*/

outerHTML

Contiene todo el codigo html del elemento.

Hola! Soy un celacanto


let celacanto = document.getElementById('celacanto');

console.log(celacanto.outerHTML);
/*
>div id="celacanto" class="elementoDOM"<
    Hola! Soy un >i< celacanto >/i><
>/div<
*/

Creacion de Elementos

Las siguientes propiedades sirven para crear elementos.

createElement()

Este metodo crea un elemento utilizando su etiqueta (en letras mayusculas).

let division = document.createElement('DIV');

console.log(division); // >div< >/div<

let parrafo = document.createElement('P');

console.log(parrafo); // >p< >/p<

let titulo = document.createElement('H1');

console.log(titulo); // >h1< >/h1<

createTextNode()

Este metodo crea un nodo de texto que despues puede ser introducido en algun elemento [mas adelante con el metodo appendChild()].

let nodoTexto = document.createTextNode('Este es un texto de prueba');

console.log(nodoTexto); // "Este es un texto de prueba"

createDocumentFragment()

Este metodo crea un "fragmento" del documento. Este fragmento se usa con el objetivo de ahorrar recursos. Porque cuando agregamos un nuevo nodo de texto o un nuevo elemento [mas adelante con el metodo appendChild()], el navegador se ve obligado a construir todo el arbol del DOM desde cero.

En consecuencia cuando queremos agregar muchos elementos al documento primero creamos un "fragmento" vacio, al que le vamos agregando todos los nuevos elementos. Despues le introducimos el fragmento al documento con todo lo nuevo y asi solo se tiene que reconstruir una sola vez con todos los cambios.

let fragmento = document.createDocumentFragment();

console.log(fragmento); // #document-fragment
