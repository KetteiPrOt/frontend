Propiedades de Parent (padres)

Es el elemento padre de cualquier nodo o elemento html.

parentElement

Contiene el elemento padre del cualquier elemento o nodo.

Tigre Junior



let tigreJunior = document.getElementById('tigreJunior');

console.log(tigreJunior.parentElement) // div id="tigreTocho"

parentNode

Contiene el nodo padre de cualquier nodo o elemento html. Casi siempre sera otro nodo de tipo elemento html.

Leon Junior



let leonJunior = document.getElementById('leonJunior');

console.log(leonJunior.parentElement) // div id="leonTocho"

closest()

Este metodo selecciona el elemento padre mas cercano que coincida con nuestro selector.

id="hijoFinal"


let hijoFinal = document.getElementById('hijoFinal');

console.log(hijoFinal.closest('.clase-2')); // div class="clase-2"

Propiedades de Siblings (hermanos)

Los elementos siblings (hermanos) son los que estan contenidos junto al elemento en cuestion dentro de un mismo padre.

nextSibling/previusSibling

Contiene el siguiente y el anterior nodo hermano del elemento, en cuestion. Como es natural, tambien incluye lo nodos de texto generados por la tabulacion del codigo.

Centauro Jr. 1

Centauro Jr. 2

Centauro Jr. 3



let centJunior1 = document.getElementById('centauroJunior1');

console.log(centJunior1.nextSibling); // #text
console.log(centJunior1.previousSibling); // #text

nextElementSibling/previusElementSibling

Contiene el siguiente y el anterior elemento hermano del elemento, en cuestion.

Pavo Jr. 1

Pavo Jr. 2

Pavo Jr. 3



let pavoJunior2 = document.getElementById('pavoJunior2');

console.log(pavoJunior2.nextElementSibling); // p id="pavoJunior3"
console.log(pavoJunior2.previousElementSibling); // p id="pavoJunior1"
