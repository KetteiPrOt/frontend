Presentacion del DOM

Es una interfaz de plataforma, o API de programación para documentos HTML, que proporciona un conjunto estándar de objetos para representar documentos HTML en los programas de JavaScript. Nos da un modelo estándar sobre cómo pueden combinarse dichos objetos, y una interfaz estándar para acceder a ellos y manipularlos.

  • Nodo: un nodo en el DOM es cualquier etiqueta del cuerpo, como un parrafo, el mismo body o incluso las etiquetas de una lista.
  • document: el nodo document es el nodo raiz a partir del cual derivan el resto de nodos.
  • element: nodos definidos por etiquetas html.
  • text: el texto dentro de un nodo element se considera un nuevo nodo hijo de tipo text.
  • attribute: los atributos de las etiquetas definen nodos, en JavaScript no los vemos como nodos, sino como informacion asociada al nodo de tipo element.
  • Comentarios y otros: los comentarios y otros (como las declaraciones de tipo DOCTYPE en el elemento head) generan nodos.

Seleccionar Elementos

Utilizamos los siguientes metodos para seleccionar elementos del documento (objeto document).

getElementById()

Selecciona un elemento por su id.

id="elemento1"


let elemento = document.getElementById('elemento1');

getElementsByTagName()

Selecciona un conjunto de elementos por su etiqueta. Retorna en una lista similar a un array, pero no es un array.

cite0 cite1 cite2 cite3 cite4


let citas = document.getElementsByTagName('cite');

console.log(citas); // HTMLCollection(5) [cite, cite, cite, cite, cite]

getElementsByClassName()

Selecciona un conjunto de elementos por su clase. Retorna en una lista similar a un array, pero no es un array.

class="cerdo"
class="cerdo"
class="cerdo"
class="cerdo"
class="cerdo"


let cerdos = document.getElementsByClassName('cerdo');

console.log(cerdos);
/* HTMLCollection(5) [
div.cerdo.elementoDOM,
div.cerdo.elementoDOM,
div.cerdo.elementoDOM,
div.cerdo.elementoDOM,
div.cerdo.elementoDOM
] */

querySelector()

Devuelve el primer elemento que coincida con la clase, id, o etiqueta especificada.

Para buscar por id: '#identificador'

id="elemento2"


let elemento = document.querySelector('#elemento2');

Para buscar por clase: '.clase'

class="paloma"


let paloma = document.querySelector('.paloma');

Para buscar por etiqueta: 'etiqueta'

Etiqueta 'i'


let italic = document.querySelector('i');

querySelectorAll()

Devuelve todos los elementos que coincidan con la clase, id, o etiqueta especificada.

Para buscar los elementos por sus ids: '#identificador'

id="elemento3"
id="elemento3"
id="elemento3"


let elementos = document.querySelectorAll('#elemento3');

console.log(elementos) // NodeList(3) [div#elemento3.elementoDOM, div#elemento3.elementoDOM, div#elemento3.elementoDOM]

Para buscar los elementos por su clase: '.clase'

class="elefante"
class="elefante"
class="elefante"


let elefantes = document.querySelectorAll('.elefante');

console.log(elefantes) // NodeList(3) [div.elefante.elementoDOM, div.elefante.elementoDOM, div.elefante.elementoDOM]

Modificar Atributos

Utilizamos los siguientes metodos para modificar los atributos de los elementos.

setAttribute()

Modifica el valor de un atributo. Recibe el nombre del atributo, y luego su nuevo valor.

id="fantasma"


/* --- Codigo CSS --- */
.txt-blue {
    color: blue;
}

/* --- Codigo JS --- */
let fantasma = document.getElementById('fantasma');

fantasma.setAttribute('class', 'txt-blue elementoDOM');

getAttribute()

Muestra el valor de un atributo. Recibe como parametro el nombre del atributo que se desea consultar.

name="Simung Freud" id="pescado"


let pescado = document.getElementById('pescado');

let informacion = pescado.getAttribute('name');
console.log(informacion); // Simung Freud

removeAttrubute()

Elimina un atributo. Recibe como parametro el nombre del atributo que se desea remover.

id="rinoceronte" name="Ricardo"


let rinoceronte = document.getElementById('rinoceronte'),
    nombre = rinoceronte.getAttribute('name');

console.log(nombre); // Ricardo

rinoceronte.removeAttribute('name');

nombre = rinoceronte.getAttribute('name');

console.log(nombre); // null
