El problema de las llaves

Cofla se quedo encerrado en su casa porque se le rompio la llave. Pero nesecita ir urgente a la facultad si quiere aprobar el resto de materias que le faltan, ya que no puede faltar ni una sola vez por estar al limite de asistencias. Lo que se le ocurrio es llamar a un serrajero y pedirle que le traiga la llave indicada, y el problema reside en que no hay tiempo para que el serrajero vea las llaves a elegir: el tiene que tomar una y salir... muy ooportuno para el problema actual, solo puede traer una y probar.

Cofla se da cuenta que es una llave antigua, por lo que descarta muchas posibilidades. Ahora solo falta decirle el modelo de la llave para que el serrajero la traiga y pueda solucionar su problema. Pero como no tiene internet contrata a un programador para que le haga un sistema con el objetivo de mostrarle la llave al serrajero y que este le pueda abrir la puerta.

Hay 18 modelos posibles para que Cofla elija el indicado.

Crear Soluciones

  • Mostrarle las 18 llaves posibles con sus imagenes.
  • Cofla debe poder seleccionar cual llave usar.
  • Una vez seleccionada enviar los datos al servidor para que otro programador se encarge.
