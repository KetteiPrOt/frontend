Atributos Globales

Existen un grupo de atributos que poseen todos los elementos.

  • class: Este atributo contiene las clases del elemento.
  • contenteditable: Indica si el elemento puede ser editado por el usuario (values: true, false).
  • dir: Indica la direccionalidad del texto (values: ltr, rtl).
  • hidden: Si esta propiedad esta definida (no lleva valor) oculta al elemento.
  • id: Define un identificador unico.
  • tabindex: Define el orden en que los elementos del sitio reciben focus desde el 0, 1, 2, 3... (values: 1, 2, 3, 4, 5...)
  • title: Contiene un texto con informacion relacionada al elemento presente. (value: text).

Ejemplo de con el atributo contenteditable:

id="textoEditable" name="gallo"


let gallo = document.getElementById('textoEditable');

gallo.setAttribute('contenteditable', true); // Ahora gallo se puede editar

Ejemplo con tabindex.

id="firstTab" name="galloPretensioso"


let galloPretensioso = document.getElementById('firstTab');

galloPretensioso.setAttribute('tabindex', '0'); // Pesiona tab para enfocar al Gallo Pretensioso

Ejemplo con title.

id="hipopotamo"


let hipopotamo = document.getElementById('hipopotamo');

hipopotamo.setAttribute('title', 'Hola soy un Hipopotamo'); // Ahora el hipopotamo habla al ponerle encima el mouse

Atributos de Inputs

Los siguientes son atributos importantes de los elementos tipo input. Es posible acceder a estos atributos directamente en el objeto, sin la nesecidad de utilizar el metodo setAttribute().

className

Contiene la lista de clases del input



let anguila = document.getElementById('anguila');

console.log(anguila.className); // ninguna elementoDOM

value

Contiene el valor del input.



let globo = document.getElementById('globo');

globo.value = 'Hola soy un Globo'; // El valor del input cambio!

type

Contiene el tipo de input.



let castor = document.getElementById('castor');

castor.type = 'submit'; // El input Castor cambio a tipo submit!

accept

Contiene el tipo de archivo que van a aceptar por defecto los inputs de tipo file.



let maniozo = document.getElementById('maniozo');

maniozo.accept = 'image/png'; // Por defecto esta aceptando archivos .png

placeholder

Contiene el placeholder del input.



let aguila = document.getElementById('aguila');

aguila.placeholder = 'Soy un Aguila!!'; // El placeholder cambio!

required

Es un atributo con valor boolean (true, false), el cual al ser true hace que el input se tenga que rellenar obligatoriamente para enviar el formulario.



let cerdoCreido = document.getElementById('cerdoCreido');

cerdoCreido.required = true; // Ahora es obligatorio llenar el input cerdo creido!

console.log(cerdoCreido);

form [No se puede acceder directo]

En los inputs de tipo submit contiene el id del formulario al que apunta el boton. NO SE PUEDE ACCEDER DIRECTAMENTE: hay que usar el metodo setAttribute().

Formulario 1, id="formulario1"



let perdido = document.getElementById('perdido');

perdido.setAttribute('form', 'formulario1'); // Ahora el boton si lo envia!

minlength [No se puede acceder directo]

Contiene el numero minimo de caracteres que debe tener el input para poder ser enviado. NO SE PUEDE ACCEDER DIRECTAMENTE: hay que usar el metodo setAttribute().



let jirafa = document.getElementById('jirafa');

jirafa.setAttribute('minlength', 30); // Ahora el campo requiere minimo 30 caracteres

console.log(jirafa.getAttribute('minlength'));

maxlength [No se puede acceder directo]

Contiene el numero maximo de caracteres que debe tener el input para poder ser enviado. NO SE PUEDE ACCEDER DIRECTAMENTE: hay que usar el metodo setAttribute().



let serpiente = document.getElementById('serpiente');
serpiente.setAttribute('maxlength', 5); // Ahora el campo tiene un maximo de 5 caracteres!
console.log(serpiente.getAttribute('maxlength'));

Atributo Style

Contiene los estilos en linea del elemento y atributos internos para controlar el estilo. Con este atributo se pueden modificar casi todas las propiedades de CSS... en este apartado escribire solo algunos ejemplos.

color

Este atributo sirve para cambiar el color del elemento.

id="berenjena"


let berenjena = document.getElementById('berenjena');

berenjena.style.color = 'purple'; // La berenjena se volvio morada!

background

Este atributo sirve para cambiar el color del elemento.

id="choclo"


let choclo = document.getElementById('choclo');

choclo.style.background = 'blue'; // El fondo se volvio azul!

fontSize

Controla el tamanio de fuente del elemento.

id="gigamax"


let gigamax = document.getElementById('gigamax');

gigamax.style.fontSize = '25px'; // gigamax se volvio gigante!!

padding

Controla el padding del elemento.

id="lechuza"


let lechuza = document.getElementById('lechuza');

lechuza.style.padding = '30px'; // El padding de la lechuza aumento al triple!!
