Metodos

Los siguientes metodos son para modificar los elementos hijos (childs).

appendChild()

Este metodo se usa para agregar un nodo texto o un elemento hijo a cualquier elemento. Retorna el nodo o el elemento que fue agregado como hijo.



let vacio = document.getElementById('vacio'),
    hijo = document.createElement('P');

hijo.textContent = 'Ahora el div vacio tiene un elemento hijo "p"!!';

vacio.appendChild(hijo); // Agrega el nuevo elemento hijo al div vacio

removeChild()

Este metodo se usa para remover un nodo texto o un elemento hijo a cualquier elemento. Retorna el nodo o el elemento hijo que fue removido.

El siguiente elemento "div" (con id="padreAnguila") tiene como un hijo "p" (con id="miniSerpiente").

Soy una miniSerpiente!



let padreAngila = document.getElementById('padreAngila'),
    miniSerpiente = document.getElementById('miniSerpiente');

padreAngila.removeChild(miniSerpiente); // La anguila se quedo vacia!

replaceChild()

Este metodo reemplaza un nodo hijo, por cualquier otro. Recibe primero el nuevo hijo, y despues el anterior que va a descartar. El metodo retorna el nuevo elemento hijo que se acoplo.

El sigiente elemento padre "div" (con id="padreOso") contiene un elemento hijo "p" (con id="cachorro")

Soy un cachorro!



let osito = document.createElement('P');

osito.textContent = 'Ahora soy un osito!';

let padreOso = document.getElementById('padreOso'),
    cachorro = document.getElementById('cachorro');

padreOso.replaceChild(osito, cachorro); // El cachorro se fue y quedo el osito!!

hasChildNodes()

Este metodo retorna true si el elemento tiene algun nodo hijo, caso contrario retorna false.

A continuacion tenemos tres "div" elementos, con id igual a pinquino 1, 2, y 3.

Nodo de texto

Elemento Hijo



let pinguino1 = document.getElementById('pinguino1'),
    pinguino2 = document.getElementById('pinguino2'),
    pinguino3 = document.getElementById('pinguino3');

console.log(pinguino1.hasChildNodes()); // true
console.log(pinguino2.hasChildNodes()); // true
console.log(pinguino3.hasChildNodes()); // false

Propiedades

Las siguientes propiedades son para modificar los elementos hijos (childs).

firstChild

Contiene el primer nodo hijo de un elemento. Esta propiedad toma en cuenta a los nodos de texto (espacios) generados por la tabulacion del codigo.

Elemento Hijo



let sandia = document.getElementById('sandia');

console.log(sandia.firstChild); // #text

lastChild

Contiene el ultimo nodo hijo de un elemento. Esta propiedad toma en cuenta a los nodos de texto (espacios) generados por la tabulacion del codigo.

Elemento Hijo



let pinia = document.getElementById('pinia');

console.log(pinia.lastChild); // #text

firstElementChild

Contiene el primer elemento hijo de un elemento.

Elemento Hijo



let platano = document.getElementById('platano');

console.log(platano.firstElementChild); // >p< Elemento Hijo >/p<

lastElementChild

Contiene el ultimo elemento hijo de un elemento.

Elemento Hijo

Ultimo Elemento Hijo



let aguacate = document.getElementById('aguacate');

console.log(aguacate.lastElementChild); // >p< Ultimo Elemento Hijo >/p<

childNodes

Contiene todos los nodos hijos de un elemento. Retorna una NodeList (lista de nodos) con todos los nodos hijos encontrados.

Elemento Hijo



let meyoco = document.getElementById('meyoco');

console.log(meyoco.childNodes); // NodeList(3) [text, p, text]

children

Contiene todos los elementos hijos de un elemento. Retorna una HTMLCollection (coleccion html) con todos los elementos hijos encontrados.

Elemento Hijo 1

Elemento Hijo 2

Elemento Hijo 3



let lechuga = document.getElementById('lechuga');

console.log(lechuga.children); // HTMLCollection(3) [p, p, p]
