Metodos de conteo

count()

Registra el numero de veces que se llama a count()... este argumento toma como argumento opcional una etiqueta.

let conteo1 = 'Conteo 1';
let conteo2 = 'Conteo 2';
console.count(); // default: 1
console.count(); // default: 2
console.count(); // default: 3
console.count(); // default: 4
console.count(); // default: 5
console.count(); // default: 6
console.count(); // default: 7
console.count(); // default: 8

console.log('\n');

console.count(conteo1); // Conteo 1: 1
console.count(conteo1); // Conteo 1: 2
console.count(conteo1); // Conteo 1: 3
console.count(conteo1); // Conteo 1: 4
console.count(conteo1); // Conteo 1: 5
console.count(conteo1); // Conteo 1: 6
console.count(conteo1); // Conteo 1: 7

console.log('\n');

console.count(conteo2); // Conteo 2: 1
console.count(conteo2); // Conteo 2: 2
console.count(conteo2); // Conteo 2: 3
console.count(conteo2); // Conteo 2: 4
console.count(conteo2); // Conteo 2: 5
console.count(conteo2); // Conteo 2: 6
console.count(conteo2); // Conteo 2: 7

countReset()

Resetea una cuenta que se lleva, y retorna undefined.

console.countReset(); // undefined
console.countReset(conteo1); // undefined
console.countReset(conteo2); // undefined

Metodos de Agrupacion

group()

Crea un nuevo grupo en linea en el registro de la consola. Admite un parametro, el nombre del grupo.

console.group('Grupo 1');

console.assert(false);
console.assert(false);
console.assert(false);

groupEnd()

Finaliza el grupo en linea creado en el registro de la consola. Admite un parametro, el nombre del grupo a cerrar.

console.group('Grupo 1');

console.assert(false);
console.assert(false);
console.assert(false);

console.groupEnd();

groupCollapsed()

Igual a console.group(), solo que el nuevo grupo que se crea ya aparece colapsado.

console.groupCollapsed('Grupo 1');

console.assert(false);
console.assert(false);
console.assert(false);

console.groupEnd('Grupo 1');

Metodos de temporalizacion

time()

Inicia un temporalizador. Admite un parametro, el nombre del temporalizador... si no tiene nombre usara el temporalizador 'default'.

let t1 = 'Reloj 1';
console.time(t1);

timeEnd()

Finaliza y muestra el valor actual de un temporalizador. Admite un parametro, el nombre del temporalizador... si no tiene nombre finalizara el temporalizador 'default'.

let t1 = 'Reloj 1';
console.time(t1);
console.timeEnd(t1); // Reloj 1: 0.005126953125 ms

timeLog()

Muestra el valor actual de un temporalizador. No lo finaliza, solo lo muestra.

let t1 = 'Reloj 1';

console.time(t1);
console.timeLog(t1); // Reloj 1: 0.06298828125 ms
console.timeEnd(t1); // Reloj 1: 0.32275390625 ms

Esta propiedades se ven muy utiles para saber cuanto tarda en ejecutarse el codigo.
