Consola funciones de registro

assert()

Aparece un mensaje por la consola si la afirmacion es falsa, si es verdadera no aparecera nada y retorna undefined en ambos casos.

console.assert(true); //
console.assert(false); // Assertion failed: console.assert

clear()

Limpia los valores de la consola.

console.assert(false); // ERROR
console.assert(false); // ERROR
console.clear() // Console was cleared

info()

Emite un mensaje de tipo informacion en la consola. Similar a log()

console.info('Hola Mundo!!'); // Hola Mindo!!

log()

Muestra un mensaje de parte del interprete de JS... lo hemos estado usando todo este tiempo no es asi?

console.log('Hola Mundo!!'); // Hola Mundo!!

table()

Muestra una tabla por la consola. Recibe como argumento primero un array o un objeto, y lo muestra como una tabla en la consola.

let array = ['Joel', 'Matias', 'Elias', 'Dana', 'Tobias', '???'],
    objeto = { perro: 'firulais', gato: 'Maniozo', ganzo: 'Garbagio' }

console.table(array); // Tabla de los datos y el indice
console.table(objeto) // Tabla de los valores y sus llaves

warn()

Muestra un mensaje de advertencia por consola.

console.warn('Ten cuidado con esoooo!!'); // (!) Ten cuidado con esoooo!!

dir()

Despliega una lista interactiva de las propiedades del objeto JavaScript identificado.

const objeto = {
    dia1: 'Lunes',
    dia2: 'Martes',
    dia3: 'Miercoles',
    dia4: 'Jueves'

} console.dir(objeto); // Lista de las propiedades
console.log(objeto); // {dia1: 'Lunes', dia2: 'Martes', dia3: 'Miercoles', dia4: 'Jueves'}
