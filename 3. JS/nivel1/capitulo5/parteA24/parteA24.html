Cambiar Color

Se pone '%c' frente al texto, y despues se para como segundo argumento el color: 'color: red'

console.log('Rancio') // Rancio

console.log('%cRancio', 'color:red'); // Rancio

console.log('%cMinecraft', 'color:blue'); // Minecraft

console.log('%cFumetsu no Anata', 'color:#74ff60'); // Fumetsu no Anata

Agregar mas propiedades de estilo

A partir de aqui se puese agregar en el segundo parametro mas propiedades de estilos CSS:

console.log('%cMinecraft', 'color: white; background: blue'); // Minecraft

console.log('%cMinecraft', 'color: white; background: blue; font-style: italic'); // Minecraft

console.log(
    '%cMinecraft',
    'color: white; background: blue; font-style: italic; padding: 10px'
); // Minecraft

console.log(
    '%cMinecraft',
    `color: white; background: blue; font-style: italic;
    padding: 10px; border: 3px dashed grey`
); // Minecraft

El elemento de consola se comporta como un elemento en linea (inline) de CSS.
