Conceptos Basicos de POO en JavaScript

Volver al capitulo

Clase

Una Clase es como la receta, una plantilla que creamos para que con esa plantilla podamos crear objetos. Es como una fabrica de Objetos a la que le definimos que cosas van a tener los objetos que va a crear. Si queremos crear Objetos con caracteristicas diferentes creamos otras clase diferente.

Objeto

Las clases lo que hacen es crear objetos, y los objetos son los resultados de estas clases.

Atributo

Son las caracteristicas, las particularidades que tiene el objeto.

Metodo

Son las cosas que puede hacer un objeto.

Constructor

Es una particularidad que tienen las clases, una funcion obligatoria: cuando creamos una clase tenemos que crear un construtor que nos va a definir los atributos que tiene el objeto.

Instanciacion

Es cuando creamos un nuevo objeto con una clase.

Ejemplo

class Animal {
  constructor(especie, edad, color) {
    this.especie = especie;
    this.edad = edad;
    this.color = color;
    this.info = `Soy un/a ${this.especie}, tengo ${this.edad} anios y soy de color ${this.color}`;
  }

  presentarse() {
    console.log(this.info);
  }
}

const perro = new Animal('perro', '3', 'rojo');
console.log(perro); // Animal {especie: 'Perro', edad: '3', color: 'Rojo'}

console.log(perro.edad); // 3
console.log(perro.color); // rojo

perro.presentarse();
