Principios de la POO

Volver al capitulo

Abstraccion

Se trata de abstraer detalles de un objeto hasta que quede solo no nesesario para funcionar en el programa... reducimos las cosas a sus componentes basicos.

Moduralidad

Es la capacidad de resolver un problema grande resumiendolo en pedacitos mas pequenios e independientes.

Encapsulamiento

Es hacer que todos los datos esten privados para que el usuario no pueda acceder a los datos tan facil.

Polimorfismo

Es como un Objeto se comporta distinto ante un metodo solamente por sus propiedades. Por ejemplo un objeto vaca no va a usar el metoto caminar() igual que un objeto dinosaurio.

Ejemplo

class Animal {
  constructor(especie, edad, color) {
    this.especie = especie;
    this.edad = edad;
    this.color = color;
    this.info = `Soy un/a ${this.especie}, tengo ${this.edad} anios y soy de color ${this.color}`;
  }

  presentarse() {
    console.log(this.info);
  }
}

const gato = new Animal('gato', '2', 'negrisimo');
gato.presentarse(); // info

class Canino extends Animal {
  constructor(especie, edad, color) {
    super(especie, edad, color);
  }

  ladrar() {
    console.log('Gua Gua!');
  }
}

const perro = new Canino('doberman', '3', 'morado');
perro.presentarse(); // info
perro.ladrar(); // Gua Gua!
