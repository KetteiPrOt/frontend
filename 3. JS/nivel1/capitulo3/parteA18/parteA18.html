Herencia

Se usa con la palabre extends y sirve para ahorrar codigo haciendo que unas clases hereden sus atributos y metodos de otras.

class Usuario {
  constructor(nombre, apellido) {
    this.nombre = nombre;
    this.apellido = apellido;
  }
}

class Programador extends Usuario {
  constructor(nombre, apellido, lenguaje) {
    super(nombre, apellido);
    this.lenguaje = lenguaje;
  }
}

const joel = new Programador('Joel', 'Mero', 'JavaScript');
console.log(joel); // Programador {nombre: 'Joel', apellido: 'Mero', lenguaje: 'JavaScript'}

Metodos estaticos

Los metodos estaticos son aquellos que no nesecitan que la clase sea instanciada para que puedan ser usadod... es decir se los puede invocar directamente desde la clase sin haber creado un objeto.

class Programador {
  constructor(nombre, apellido, lenguaje) {
    this.nombre = nombre;
    this.apellido = apellido;
    this.lenguaje = lenguaje;
  }

  presentarse() {
    console.log(`Hola soy ${this.nombre} ${this.apellido}, mucho gusto!`);
  }

  static hola() {
    console.log('Hola');
  }
}

Programador.presentarse() // Error

Programador.hola() // Hola

Metodos Accesores

Se usan con las palabras claves set y get, ya que sirven para acceder y modificar datos que han sido encapsulados. Ademas cuando invocamos los metodos los hacemos como si fueran propiedades.

class Programador {
  constructor(nombre, lenguaje) {
    this.nombre = nombre;
    this.lenguaje = lenguaje;
    this.experiencia = null;
  }

  presentarse() {
    console.log(`Hola soy ${this.nombre} ${this.apellido}, mucho gusto!`);
  }

  get getLevel() {
    return this.level;
  }

  set setLevel(time) {
    this.level = time;
  }
}

const joel = new Programador('Joel', 'JavaScript');

joel.setLevel = 5;

console.log(joel); // Programador {nombre: 'Joel', lenguaje: 'JavaScript', experiencia: 5}
console.log(joel.getLevel); // 5
