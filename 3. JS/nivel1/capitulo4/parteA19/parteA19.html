Metodos de cadenas

Volver al capitulo

concat()

Junta dos o mas cadenas y retorna una nueva. No altera a la cadena original quien le aplicaron este metdo.

let texto1 = 'Hola ', texto2 = 'Mundo';
let texto3 = texto1.concat(texto2);

console.log(texto3); // Hola Mundo

Los siguientes metodos se aplican a cadenas y devuelven un valor que da informacion sobre la cadena en cuestion. No alteran la cedena original sobre la que fueron aplicados.

startsWith()

Si una cadena comienza con los caracteres de otra cadena devuelve true, sino false

let texto1 = 'Hola Mundo', texto2 = 'Hola Joel', texto3 = 'Hola', texto4 = 'Ho';

console.log(texto1.startsWith(texto2)); // false
console.log(texto1.startsWith(texto3)); // true
console.log(texto1.startsWith(texto4)); // true

includes()

Si una cadena puede encontrarse dentro de otra cadena devuelve true, sino false.

let texto1 = 'Maniana hara mucho sol en la ciudad',
    texto2 = 'perro', texto3 = 'hara mucho sol', texto4 = 'ciudad';

console.log(texto1.includes(texto2)); // false
console.log(texto1.includes(texto3)); // true
console.log(texto1.includes(texto4)); // true

indexOf()

Devuelve el indice del primer caracter de la cadena, si no existe devuelve -1.

let texto1 = 'Buenos dias hoy hace un buen tiempo',
    texto2 = 'dias', texto3 = 'hace un ', texto4 = 'hace un perro', texto5 = 'racio';

console.log(texto1.indexOf(texto2)); // 7
console.log(texto1.indexOf(texto3)); // 16
console.log(texto1.indexOf(texto4)); // -1
console.log(texto1.indexOf(texto5)); // -1

console.log(texto1[7] + texto1[8] + texto1[9] + texto1[10]); // dias

lastIndexOf()

Devuelve el ultimo indice del primer caracter de la cadena, si no existe devuelve -1.

let texto1 = 'Buenos dias hoy hace un buen tiempo, hoy es un buen dias',
    texto2 = 'hoy', texto3 = 'hoy hace', texto4 = 'hace un perro', texto5 = 'rancio';

console.log(texto1.lastIndexOf(texto2)); // 37
console.log(texto1.lastIndexOf(texto3)); // 12
console.log(texto1.lastIndexOf(texto4)); // -1
console.log(texto1.lastIndexOf(texto5)); // -1

console.log(texto1[37] + texto1[38] + texto1[39]); // hoy

Los siguientes metodos rellenan la cadena a la que son aplicados en cuestion. No alteran la cedena original sobre la que fueron aplicados.

padStart()

Rellena la cadena al principio con los caracteres deseados. Recibe dos argumentos: la cantidad de caracteres totales del resultado y el relleno que va a usar al inicio para alcanzar esa cantidad de carateres en el resultado final (si es nesesario los va a repetir):

let texto1 = '3456';

let rellenar1 = texto1.padStart(6, '12'),
rellenar2 = texto1.padStart(10, '12');

console.log(rellenar1); // 123456
console.log(rellenar2); // 1212123456

padEnd()

Rellena la cadena al final con los caracteres deseados. Recibe dos argumentos: la cantidad de caracteres totales del resultado y el relleno que va a usar al final para alcanzar esa cantidad de carateres en el resultado final (si es nesesario los va a repetir):

let texto1 = '1234';

let rellenar1 = texto1.padEnd(6, '56'),
rellenar2 = texto1.padEnd(10, '56');

console.log(rellenar1); // 123456
console.log(rellenar2); // 1234565656

repeat()

Devuelve la cadena repetida el numero de veces que le pedimos:

let texto = 'Hola ',
    repetir = texto.repeat(5);

console.log(repetir); // 'Hola Hola Hola Hola Hola '

console.log(texto.repeat(0)); // ''

Los siguientes metodos tambien hacen modificaciones sobre la cadena origial, y del mismo modo sin alterarla, sino mas bien retornan una nueva cadena con la modificacion hecha.

split()

Divide la cadena segun un caracter y retorna un array con las partes de la cadena dividida segun ese caracter. No altera a la cadena original sobre la que fue aplicado.

let texto = 'Hola.como.estas';
let partes = texto.split('.');

console.log(partes); // ['Hola', 'como', 'estas']

let texto2 = 'Hola como estas joel CRACK';

console.log(texto2.split(' ')); // ['Hola', 'como', 'estas', 'joel', 'CRACK']

console.log(texto.split('como')); // ['Hola.', '.estas']

substring()

Nos retorna un pedazo de la cadena seleccionda utilizando como argumentos las posiciones de inicio y final para la nueva cadena en cuestion. [index, index)

let texto = '0123456789',
    subTexto1 = texto.substring(0, 4),
    subTexto2 = texto.substring(1, 10);

console.log(subTexto1); // 0123
console.log(subTexto2); // 123456789

toLowerCase()

Retorna una nueva cadena en la que convierte todos los caracteres en mayusculas a caracteres en minusculas. No altera la cadena original.

let texto = 'Hola Mundo',
    minusculas = texto.toLowerCase();

console.log(minusculas); // hola mundo

toUpperCase()

Retorna una nueva cadena en la que convierte todos los caracteres en minusculasa caracteres en mayusculas. No altera la cadena original.

let texto = 'Hola Mundo',
    mayusculas = texto.toUpperCase();

console.log(mayusculas); // HOLA MUNDO

toString()

Convierte cualquier otro tipo de dato en una cadena de texto (siempre que sea posible), la cual retorna. No afecta el dato original.

let numero = 1500375,
    array = [12, 'Joel', true]
    texto = numero.toString();

console.log(texto); // 1500375

console.log(array.toString()); // 12,Joel,true

trim()

Elimina los espacios en blanco al inicio y al fin de la cadena.

let texto = ' Hola Mundo ',
    sinEspacios = texto.trim();

console.log('|' + sinEspacios + '|'); // |HolaMundo|

trimEnd()

Elimina los espacios al final de la cadena.

let texto = ' Hola Mundo ',
    sinEspacios = texto.trimEnd();

console.log('|' + sinEspacios + '|'); // | HolaMundo|

trimStart()

Elimina los espacios al final de la cadena.

let texto = ' Hola Mundo ',
    sinEspacios = texto.trimStart();

console.log('|' + sinEspacios + '|'); // |HolaMundo |

valueOf()

Retorna el valor primitivo de un objeto String.

let textoObjeto = new String('Hola Mundo'),
    textoPrimitivo = textoObjeto.valueOf();

console.log(textoObjeto); // String {'Hola Mundo'}
console.log(textoPrimitivo) // Hola Mundo
