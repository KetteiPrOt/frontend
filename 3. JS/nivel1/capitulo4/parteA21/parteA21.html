Metodos

sqrt()

Recibe un numero y retorna su raiz cuadrada.

let raiz = Math.sqrt(25);

console.log(raiz); // 5

cbrt()

Recibe un numero y retorna su raiz cubica.

let raiz = Math.cbrt(27);

console.log(raiz); // 3

max()

Recibe como parametro 0 o mas numeros y retorna el numero mayor.

let maximo = Math.max(1, 3, 6, 12, 5, 12, 11);

console.log(maximo); // 12

min()

Recibe como parametro 0 o mas numeros y retorna el numero menor.

let minimo = Math.min(1, 3, 6, 12, 5, 11, 1);

console.log(minimo); // 1

random()

Devuelve un numero pseudo-aleatorio entre 0 y 1. Rango (0, 1).

for (let i = 0; i <= 5; i++) {
    console.log(Math.random()); // ? ? ? ? ?
}

round()

Devuelve el valor de un numero redondeando el numero entero mas cercano, osea que trunca los numeros decimales (o los redondea).
Redondea al numero mayor si el primer decimal es mayor que 5, y al menor si es menor que 5.

let decimal1 = 0.33345, decimal2 = 0.7777576,
decimal3 = 3.222236, decimal4 = 46.23734;

console.log(Math.round(decimal1)); // 0
console.log(Math.round(decimal2)); // 1
console.log(Math.round(decimal3)); // 3
console.log(Math.round(decimal4)); // 46

Numeros pseudoaleatorios del 0 al 10. Rango [0, 10]

let random = Math.random();

console.log(Math.round(random * 10));

fround()

JavaScript utiliza internamente números de punto flotante doble de 64 bits, que ofrecen una precisión muy alta. Sin embargo, a veces puede estar trabajando con números de punto flotante de 32 bits, por ejemplo, si está leyendo valores de un Float32Array. Esto puede crear confusión: la verificación de la igualdad entre un flotante de 64 bits y un flotante de 32 bits puede fallar aunque los números parezcan idénticos. Para resolver esto, se puede usar Math.fround() para convertir el flotante de 64 bits en un flotante de 32 bits. Internamente, JavaScript continúa tratando el número como un flotante de 64 bits, solo realiza un "redondeo a par" en el bit 23 de la mantisa y establece todos los siguientes bits de mantisa en 0. Si el número está fuera del rango de un Se devuelve float de 32 bits, Infinity o -Infinity. Debido a que fround() es un método estático de Math, siempre lo usa como Math.fround(), en lugar de como un método de un objeto Math que creó (Math no es un constructor).

console.log(Math.fround(4.5677)); // 4.567699909210205

floor()

Redondea un numero decimal siempre hacia abajo, sin impportar si su primer digito decimal es mayor a 5.

console.log(Math.floor(4.29)); // 4
console.log(Math.floor(4.59)); // 4
console.log(Math.floor(4.99)); // 4

trunc()

Elimina la parte decimal del numero y retorna su parte entera, a diferencia del redondeo no le importa si esta mas cerca del de arriba o del de abajo.

let num1 = 3.2222, num2 = 3.8888, num3 = 3.9999; console.log(Math.trunc(num1)); // 3 console.log(Math.trunc(num2)); // 3 console.log(Math.trunc(num3)); // 3
