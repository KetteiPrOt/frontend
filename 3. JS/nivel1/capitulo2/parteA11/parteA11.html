Definicion y usos

Un array es un arreglo de datos ordenados segun un indice que inicia en 0. Tambien es conocido como vector y en su interior puede contener muchos datos de distintos tipos:

let array = [dato0, dato1, dato2... , datoN];
alert(array[0]); // dato0
alert(array[1]); // dato1

Inicializacion de posiciones

Cuando creamos un arreglo automaticamente declaramos todas sus posiciones desde -infinito hasta 0 y hasta +infinito.
Aunque estas posiciones estan declaradas siguen sin inicializarse porque sus valores no fueron definidos (estan como undefined).

let array = [dato0, dato1, dato2];

Ingresarle una propiedad

Es posible ingresarle una propiedad al array de la siguiente forma:

let array = ['elemento0', 'elemento1', 'elemento2', 'elemento3'];

array.numero = 20;

console.log(array.numero); // 20

console.log(array); // ['elemento0', 'elemento1', 'elemento2', 'elemento3', numero: 20]
