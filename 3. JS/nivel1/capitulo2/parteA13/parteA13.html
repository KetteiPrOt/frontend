Bucles e Iteracion

Definicion y Usos

Los bucles son estructuras de control que sirven para realizar procesos repetitivos donde tenga que iterar varias veces el mismo codigo.
Hay varios tipos de bucles, y cada uno es mas o menos conveniente para distintas situaciones.

Bucle while (Mientras)

Este es un bucle que repitira la ejecucion de su codigo mientras se cumpla una condicion:

while (condicion) {
  // codigo
};

Bucle do while (Hacer Mientras)

Funciona de la misma manera que el bucle anterior, solo que este evaluara la condicion despues y no antes de ejecutar el codigo:

do {   // codigo } while (condicion);

Bucle for

Es un bucle mas avanzado que tiene la siguiente estructura:

for (let i = N; i >= 0; i--) {
  console.log(i); // 5 4 3 2 1 0
}

Donde:

  • i es una variable que se usara de contador.
  • N es un numero inicial cualquiera.
  • i >= 0; es una condicion cualquiera que el bucle usara para detenerse cuando esta no se cumpla.
  • i-- es una sentencia que se ejecutara al finalizar cada iteracion del bucle para ir aproximando el valor de i hasta que la condicion no se cumpla.

Con estas herramientas el bucle for (Para...) nos proporcina una facilidad increible para mantener el control sobre las iteraciones de procesos repetitivos.

Bucle for in

Este bucle esta hecho para recorrer automaticamente un arreglo utilizando una variable que en cada iteracion tomara el valor del indice del elemento del arreglo:

let arreglo = [elemento0, elemento1, elemento2, elemento3];

for (variable in arreglo) {
  console.log(variable); // 0 1 2 3
}

Bucle for of

Este bucle esta hecho para recorrer automaticamente un arreglo utilizando una variable que en cada iteracion tomara el valor del elemento del elemento del arreglo:

let arreglo = [elemento0, elemento1, elemento2, elemento3];

for (variable of arreglo) {
  console.log(variable); // elemento0, elemento1, elemento2, elemento3
}

Sentencia break

Esta sentencia (romper) se usa para indicarle a los bucles que detengan sus iteraciones:

let contador = 5;

while (contador >= 0) {
  console.log(contador); // 5 4 3 2 1 0
  contador--;
}

while (contador >= 0) {
  if (contador == 2) {
    break;
  }
  console.log(contar); // 5 4 3
  contador--;
}

La sentencia break funciona para todos los tipos de bucles:

let array = ['elemento0', 'elemento1', 'elemento2', 'elemento3', 'elemento4'];

for (indice in array) {
  console.log(indice); // 0 1 2 3 4
}

for (indice in array) {
  if (indice == 3) {
    break;
  }
  console.log(indice); // 0 1 2
}

Sentencia continue

La sentencia continue produce que el bucle detenga la iteracion actual, omita lo que le falte para terminarla, e inicie la siguiente:

let contador = 5;

while (contador >= 0) {
  console.log(contador); // 5 4 3 2 1 0
  contador--;
}

while (contador >= 0) {
  if (contador == 2) {
    continue;
  }
  console.log(contar); // 5 4 3 1 0
  contador--;
}

La sentencia continue funciona para todos los tipos de bucles:

let array = ['elemento0', 'elemento1', 'elemento2', 'elemento3', 'elemento4'];

for (indice in array) {
  console.log(indice); // 0 1 2 3 4
}

for (indice in array) {
  if (indice == 3) {
    continue;
  }
  console.log(indice); // 0 1 2 4
}

Recorrer un array dentro de otro array

Con el siguiente codigo es posible recorrer un array (mostrando por consola sus elementos) que esta guardado dentro de otro array (que de paso tambien recorrere):

let array1 = ['gato', 'perro', 'loro', 'ganzo'];

let array2 = [
  'elemento0',
  'elemento1',
  'elemento2',
  array1,
  'elemento4'
];

for (indice in array2) {

  if (indice == 3) {
    // Segundo Bucle
    for (elemento of array1) {
      console.log(elemento); // 'gato' 'perro' 'loro' 'ganzo'
    }
    break;
  }

  console.log(array2[indice]); // 'elemento0' 'elemento1' 'elemento2'
}

Etiqueta (label)

Una etiqueta es un nombre que se le da a un bucle de la siguiente manera para hacer referencia a el mas adelante en el programa:

nombreEtiqueta:
for (indice in array) {
  // codigo
}

Utilizando este nombre de la etiqueta es posible aplicar las sentencias continue y break a un bucle desde otro bucle:

nombreEtiqueta:
for (indice in array1) {

  if (indice == N) {
    for (elemento of array2) {
      if (elemento == 'elementoN') {
        break nombreEtiqueta; // Esta sentencia detiene el primer bucle desde el segundo
      }
      // codigo
    }
  }

  // codigo
}
