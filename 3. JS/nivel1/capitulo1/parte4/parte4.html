Variables y constantes

Volver al capitulo

Variable

Una variable es un espacio de memoria que utilizamos para guardar un dato, para declararla se usa el comando 'let', y para inicializarla se le asigna un valor. Despues de declararla es posible seguir invocandola a lo largo del programa, para solicitar o modificar el dato que contiene, sin la nesecidad de volver a utilizar el comando 'let'.

let variable = "dato";

Tambien se puede solo declarar la variable sin inicializarla al inicio, para asignarle su valor en cualquier otro punto del programa:

ler variable; variable = "dato";

Tipos de datos primitivos

Los datos que guardan las variables pueden ser de distintos tipos, y entre los datos simples o primitivos mas importantes tenemos:

  • Numericos: integer (int, enteros), floating point (float, punto flotante o decimales), ejemplos:
    • int = '3';
    • float = '4.56';
  • Logico: booleanos (boolean), ejemplos:
    • boolean1 = true;
    • boolean2 = false;
  • Alfanumerico: string (cadenas de texto), ejemplo:
    • string1 = 'Hola Mundo';
    • string2 = `Hola Mundo!`;

Tipos de datos especiales

Hay casos especiales de datos, es importante conocer los siguientes:

  • undefined : se da cuando solicitamos el dato de una variable declarada pero no ha sido inicializada, entonces su valor no esta definido.
  • null : es un dato que se asigna de forma intencional a una variable para indicar que su valor esta definido como nulo o vacio, por lo que la variable/constante ya esta inicializada.
  • NaN : este dato se genera cuando intentemos operar aritmeticamente un dato de tipo numerico con cualquier otro tipo de dato que no sea numerico. Significa 'No es un numero' (Not is A Number).

Constantes

Una constante es una variable a la que no se le puede cambiar su valor a lo largo de todo el programa, y se declaran con el comando 'const'. Ademas, a diferencia de las variables, es obligatorio inicializarla a la vez que se declara ya que no es posible declarar una constante y luego inicializarla.

const constante = "dato";

Declarar multiples variables/constantes

Es posible declarar o/e inicializar multiples variables usando comas:

let num1 = 3, num2 = 11, num3 = 23;
let num1, num2, num3;

Tambien es posible declarar e inicializar multiples constantes usando comas:

const num1 = 3, num2 = 11, num3 = null;

Funcion prompt()

Esta funcion sirve para solicitar un dato al usuario de la pagina y luego almacenar ese dato en una variable:

let variable = prompt('Ingrese su dato');

Funcion parseInt()

Esta es una funcion que recive un dato de tipo String y lo convierte a uno de tipo numero entero (si es posible):

let numero = parseInt('12');
alert(numero - 3); // 9

Tambien es util para truncar numeros decimales solo a su parte entera:

let numero2 = parseInt('12.5'); alert(numero); // 12

Funcion parseFloat()

Esta es una funcion que recive un dato de tipo String y lo convierte a uno de tipo numero decimal o Float (si es posible):

let numero = parseFloat('12.5');
alert(numero - 3); // 9.5

Sentencia typeof

Esta sentencia se usa para evaluar el tipo de dato de un dato:

console.log(typeof 100); // number
console.log(typeof "Hola Mundo"); // string
console.log(typeof true); // boolean
