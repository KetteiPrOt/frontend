Formas de enlazar JavaScript

Volver al capitulo

En linea

Se puede agregar JavaScript en el atributo onclick="" de un elemento html y el codigo se ejecutara a hacerle click al elemento. No se recomienda hacerlo al menos que sea para realizar pruebas rapidas.

Presioname!

Etiqueta script

Se puede agregar JavaScript dentro del documento html utilizando la etiqueta "script" y el codigo se ejecutara al inicializar el documento.

< script > "codigo JavaScript" < / script >

Archivo de formato .js

Se puede enlazar un archivo externo con codigo JavaScript (formato .js). El codigo se ejecutara al cargar el archivo en el documento html.

< script src="archivo.js" > < / script >
