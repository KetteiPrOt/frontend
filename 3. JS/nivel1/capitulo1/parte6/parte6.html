Que es la concatenacion

Concatenar significa unir dos o mas cadenas de texto.

"texto1" + " " + "texto2" ---> "texto1 texto2"

Concatenar con +

Puedo concatenar cadenas de texto con otras cadenas de texto o datos numericos usando el operador +.

nombre = "Joel"; edad = 18; presentacion = "Hola soy " + nombre + ". Y tengo " + edad + " anios!";
alert(presentacion); // Hola soy Joel. Y tengo 18 anios!

Forzar String

No es posible concatenar dos numeros usando solo el operador +, ya que acabaria sumandolos. Para lograrlo es nesesario forzar el cambio de los datos numericos a Strings:

forzarString = "" + 1 + 2 + 3 + 4 + 5 + 6;
alert(forzarString); // 123456

Funcion concat()

Es una funcion que sirve para concatenar algun dato a una cadena de texto. Se aplica obligatoriamente sobre un dato de tipo String.

texto1 = "Hola"; texto2 = " Mundo"; concatenacion = texto1.concat(texto2);
alert(concatenacion); // Hola Mundo

numero = 5; texto = "Hola"; concatenacion = texto.concat(numero);
alert(concatenacion); // Hola5

Con backtiks y ${}

Se puede concatenar datos de tipo numerico o String utilizando backtiks (``) de la siguiente forma:

nombre = "Joel", edad = 18, apellido = "Mero";
alert(`Hola ${nombre} ${apellido}! Tienes ${edad} anios`); // Hola Joel Mero! Tienes 18 anios
