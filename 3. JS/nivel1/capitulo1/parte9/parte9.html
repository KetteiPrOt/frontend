Camel Case

Volver al capitulo

Definicion

Camel Case significa Case de camello, quiere decir que a las variables las vamos a nombrar utilizando las mayusculas y minusculas como si fuera un camello.

Usos

Se usa para evitar errores al momento de nombrar las variables, como poner simbolos no permitidos, ademas de que al incluir nombres de variables mas largos hago que el programa sea mas facil de entender.

Ejemplo

Nombrar variables con nombreclatura de camello:

NU ----> nombreUsuario
EU ----> edadUsuario
num1 ----> numeroUno
