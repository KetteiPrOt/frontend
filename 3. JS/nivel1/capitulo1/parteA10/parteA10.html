Condicionales

Volver al capitulo

If

Un condicional es una estructura de control que ejecutara un bloque de codigo si la condicion que tiene para evaluar es verdadera, caso contrario no ejecutara el codigo.

if (condicion) { ... }

Else If

Es otro condicional que se puede agregar directamente despues del primero para probar otra pregunta, y se pueden agregar cuantos quiera. Significa 'Si no, Si...'

if (condicion) { ... } else if (condicion2) {...} else if (condicion3) {...}...

Else

Es el bloque de codigo que se ejecutara si no se cumple ningun condicional en la estructura. Significa 'Si no...'

if (condicion) { ... } ... else if (condicionN) {...} else {...}
