Condicionales

Volver al capitulo

If

Un condicional es una estructura de control que ejecutara un bloque de codigo si la condicion que tiene para evaluar es verdadera, caso contrario no ejecutara el codigo.

if (condicion) { ... }

Else If

Es otro condicional que se puede agregar directamente despues del primero para probar otra pregunta, y se pueden agregar cuantos quiera. Significa 'Si no, Si...'

if (condicion) { ... } else if (condicion2) {...} else if (condicion3) {...}...

Else

Es el bloque de codigo que se ejecutara si no se cumple ningun condicional en la estructura. Significa 'Si no...'

if (condicion) { ... } ... else if (condicionN) {...} else {...}

Sentencia switch

Esta sentencia de control de flujo utiliza una variable la cual compara con varios 'casos', ejecutara un codigo u otro segun el valor de la variable en cuestion.

Para que pare de compara la variable y seguir ejecutando codigo, hay que darle la orden break.

Para que ejecute un codigo por defecto se usa el caso 'default'.

let variable = 'rojo';

switch (variable) {
    case 'amarillo':
         console.log('El color es amarillo!');
         break;
    case 'azul':
         console.log('El color es azul!');
         break;
    case 'rojo':
         console.log('El color es rojo!');
         break;
    case 'purpura':
         console.log('El color es purpura!');
         break;
    default:
         console.log('El color es desconocido');
}

// El color es rojo
