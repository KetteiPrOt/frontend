Saltos de linea

Si uso backticks para concatenar puedo crear saltos de linea directamente en el codigo:

alert(`Hola ${nombre} ${apellido}!
Tienes ${edad} anios`);
// Hola Joel Mero!
// Tienes 18 anios

Pero si no se puede dar el caso (por ejemplo, a causa de la tabulacion del codigo) puedo escribir directamente el caracter "\n" para crear un salto de linea:

alert(`Hola ${nombre} ${apellido}!` + `\n` + `Tienes ${edad} anios`);
// Hola Joel Mero!
// Tienes 18 anios

Escape de comillas simples

Se refiere a utilizar comillas simples dentro de Strings definidos por comillas dobles para no afectar al dato:

string = "Texto Texto 'Texto' Texto Texto"; // Texto Texto 'Texto' Texto Texto

Escape de comillas dobles

Se refiere a utilizar comillas dobles dentro de Strings definidos por comillas simples para no afectar al dato:

string = 'Texto Texto "Texto" Texto Texto'; // Texto Texto "Texto" Texto Texto

Escape de barra invertida

Se escribe una barra invertida para escapar caracteres especiales como comillas y saltos de linea.

let string = "Mi nombre es \"Joel\". \nMi apellido es \"Mero\".";
