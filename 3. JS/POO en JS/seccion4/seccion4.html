La Asociacion

Esto es relacionar Objetos independientes, Por ejemplo si tenemos los Objetos independientes jhon y maria los puedo relacionar como padre e hija de la sigiente forma:

class Persona {
  constructor(nombre, apellido) {
    this.nombre = nombre;
    this.apellido = apellido;
  }
}

const jhon = new Persona('Jhon', 'Ray'),
      maria = new Persona('Maria', 'Perez');

// Asociacion
maria.padre = jhon;
console.log(jhon); // Persona {nombre: 'Jhon', apellido: 'Ray'}
console.log(maria); // Persona {nombre: 'Maria', apellido: 'Perez', padre: Persona}

Y hacer esta Asociacion no quiere decir que si quito la relacion de que Jhon es el padre de Maria los objetos ya no funcionaran, porque los objetos son independientes. Esta es la relacion en JavaScript.

Agregacion

En la agregacion tenemos una Asociacion jerarquica entre objetos. El objeto Mayor (mas alto en la jerarquia) contiene otros objetos Menores y es llamado Aggegate, mientras que los Menores quienes son contenidos son llamados Component. Pero aun asi, como es una Asociacion, cada objeto puede tener una existencia independiente.

Por ejemplo, si creo un nuevo Objeto Literal llamado 'companiia' puedo hacer una Agregacion de 'empleados' (Jhon, y Maria) de la siguiente manera:

const companiia = {
  nombre: 'FreeD games Pro',
  empleados: []
}

.class Persona {
  constructor(nombre, apellido) {
    this.nombre = nombre;
    this.apellido = apellido;
  }
}

const jhon = new Persona('Jhon', 'Ray'),
      maria = new Persona('Maria', 'Perez');

// Agregacion
companiia.empleados.push(jhon);
companiia.empleados.push(jhon);

console.log(companiia); // {nombre: 'FreeD games Pro', empleados: Array(2)}

Composicion

Es similar a la Agregacion, muchos objetos pueden pertenecer a un objeto que va a tener un rol mayor pero ninguno de estos va a ser independiente del objeto mayor.

Por ejemplo, en el siguiente codigo tenemos un objeto que representa la direccion de una persona... si quito este objeto que representa la direccion del objeto que representa la persona, por si solo el objeto de la dirrecion no tiene sentido:

const persona = {
  nombre: 'Joel',
  apellido: 'Mero',
  // Composicion
  direccion: { calle: '23 Octubre', ciudad: 'Jaramijo', pais: 'Ecuador' }
}

Entonces estamos hablando de que funciona como parte de la Composicion del objeto persona.
