Screen y scroll

Estas propiedades contienen informacions sobre la pantalla y el scroll del documento.

screen

Devuelve una referencia al Objeto Screen (pantalla) asociado al Objeto Window (ventana).

En otros terminos, el objeto window contiene en su interior un objeto 'pantalla' que guarda muchos detalles sobre la pantalla en la que se aloja la ventana.

let pantalla = window.screen;

console.log(pantalla); // Screen { ... width: 1366, height: 768 ... }

Entre las propiedades mas interesantes que posee el objeto Screen estan:

  • availHeight: contiene el alto disponible.
  • availWidth: contiene el ancho disponible.
  • width: contiene el ancho actual de la ventana.
  • height: contiene el alto actual de la ventana.
  • pixelDepht: contiene la profundidad de pixeles de la pantalla.

screenLeft

Contiene la distancia horizontal izquierda (pixeles) entre el borde izquierdo de la ventana del navegador y el borde izquierdo de la pantalla.

let screenLeft = window.screenLeft;

console.log(screenLeft); // Distancia izquierda horizontal en pixels

screenTop

Contiene la distancia vertical superior (pixeles) entre el borde superior de la ventana del navegador y el borde superior de la pantalla.

let screenTop = window.screenTop;

console.log(screenTop); // Distancia superior vertical en pixels

scrollX

Contiene el numero de pixels que el documento se desplaza actualmente horizontalmente.

let scrollX = window.scrollX;

console.log(scrollX); // Desplazamiento horizontal actual (pixels)

scrollY

Contiene el numero de pixels que el documento se desplaza actualmente verticalmente.

let scrollY = window.scrollY;

console.log(scrollY); // Desplazamiento vertical actual (pixels)

scroll()

Desplaza la ventana hacia una coordenada particular X y Y (en pixels) del documento.

Con Pixels: recibe como primer argumento la posicion X (en pixels), y como segundo argumento la posicion Y (en pixels).

window.scroll(0, 100); // 0px en X, 100px en Y

Con Options: Recibe un Objeto con tres atributos. Con esta forma se puede hacer un desplazamiento mas suave.

  • top: la posicion X (en pixels).
  • left: la posicion y (en pixels).
  • behavior: un string cuyo valor puede ser smooth, instant, o auto (Por defecto el valor es auto).
const options = {
    top: 100,
    left: 0,
    behavior: 'smooth' // instant (al instante), smooth (mas lentamente)
}

window.scroll(options);

scroolBy()

Desplaza la ventana de forma relativa segun una distancia X y Y (en pixels) del documento.

Con Pixels: recibe como primer argumento el desplazamiento X (en pixels), y como segundo argumento el desplazamiento Y (en pixels).

window.scrollBy(0, 100); // Desplaza el documento 100px hacia abajo

Con Options: Recibe un Objeto con tres atributos. Con esta forma se puede hacer un desplazamiento mas suave.

  • top: el desplazamiento X (en pixels).
  • left: el desplazamiento y (en pixels).
  • behavior: un string cuyo valor puede ser smooth, instant, o auto (Por defecto el valor es auto).
const options = {
    top: 500,
    left: 0,
    behavior: 'smooth' // instant (al instante), smooth (mas lentamente)
}

window.scrollBy(options);
