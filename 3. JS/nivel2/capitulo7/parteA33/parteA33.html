Manipular la Ventana

Los siguientes metodos sirven para manipular la ventana del navegador. Algunos de ellos estan limitados por razones de seguridad y solo se pueden utilizar de formas especificas (como resizeTo y resizeBy).

resizeTo()

Cambia el tamanio de una ventana controlada por el script a una cantidad especificada.

Para que le sea posible al script cambiar el tamanaio de la ventana hace falta pasarle el atributo resizable al metodo open().

const nuevaVentana = window.open('https://youtube.com', 'nombreVentana', 'resizable');

nuevaVentana.resizeTo(800, 800); // La nueva ventana se redimenciono a 800x800 pixeles

resizeBy()

Cambia el tamanio de una ventana controlada por el script en una cantidad especificada. Para encoger la ventana se le pasan valores negativos.

Para que le sea posible al script cambiar el tamanaio de la ventana hace falta pasarle el atributo resizable al metodo open().

const nuevaVentana = window.open('https://youtube.com', 'nombreVentana', 'resizable');

nuevaVentana.resizeTo(500, 500);

nuevaVentana.resizeBy(100, 100); // La ventana final es de 600x600 pixeles

moveTo()

Mueve la ventana a una ubicacion absoluta. Recibe como primer parametro el eje x (en pixels) mueve esa disntancia hacia la izquierda, como segundo parametro el eje y (en pixels) mueve esa distancia hacia abajo.

let nuevaVentana = window.open('https://google.com', 'nombreVentana', 'resizable');

nuevaVentana.resizeTo(500, 500); // Redimensiona la ventana a 500x500 pixels

nuevaVentana.moveTo(200, 100); // La mueve 200px a la izquierda y 100px abajo

moveBy()

Mueve la ventana de forma relativa segun las medidas especificadas. Recibe como primer parametro el eje x (en pixels) mueve esa disntancia hacia la izquierda, como segundo parametro el eje y (en pixels) mueve esa distancia hacia abajo.

let nuevaVentana = window.open('https://google.com', 'nombreVentana', 'resizable');

nuevaVentana.resizeTo(500, 500); // Redimensiona la ventana a 500x500 pixels

nuevaVentana.moveTo(200, 100); // La mueve 200px a la izquierda y 100px abajo

nuevaVentana.moveBy(100, 50); // El movimiento final es de 300 px a la izquierda y 150px abajo

Objetos BarProp

Son objetos pertenecientes al prototipo BarProp (apoyo de barra), se usaban para comprobar si ciertos de la interfaz del navegador son actualmente es visible o no.

Por motivos de privacidad e interoperabilidad, el valor de las propiedad visible ahora es falso si esta ventana es una ventana emergente y verdadero en caso contrario.

locationbar.visible

Cotenia si la ventana esta visible en la barra de ubicaciones.

Ahora en una ventana emergente no es visible y retorna false:

let nueva = window.open('', 'nombre', 'resizable');

nueva.resizeTo(500, 500);

console.log(nueva.locationbar.visible); // false

Pero una ventana normal que esta abierta y visible junto a las demas si retornara true:

let nueva = window.open();

console.log(nueva.locationbar.visible); // true

Este comportamiento se repite con todos los objetos BarProps.

  • locationbar.visible
  • menubar.visible
  • personalbar.visible
  • scrollbars.visible
  • statusbar.visible
  • toolbar.visible
