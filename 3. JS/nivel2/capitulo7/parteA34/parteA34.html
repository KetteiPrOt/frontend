Que es el objeto Location?

Es un objeto contenido dentro de objeto window (que no esta contenido hay xd?), el cual contiene informacion importante referente al archivo donde estamos actualmente ubicados.

console.log(window.location); // Location { ... }

href

Contiene el hipertext reference (en vinculo, enlace, link) del archivo en el que estamos ubicados.

let href = window.location.href;

console.log(href); // 'file:///C:/Users/Admin/Desktop/ ... /parteA34.html'

hostname

Contiene el nombre del host donde estamos ubicados. Como este es un archivo (actualmente no esta en ningun servidor), no retorna nada. Pero en una pagina web alojada en un servidor real o local si retorna el nombre de host.

let hostname = window.location.hostname;

console.log(hostname); // ''

pathname

Retorna el nombre de la ruta utilizada para acceder al archivo.

let pathname = window.location.pathname;

console.log(pathname); // '/C:/Users/Admin/Desktop/ ... /parteA34.html'

protocol

Retorna el nombre del protocolo utilizado para acceder al archivo. Como este es un archivo local el protocolo utilizado es file... en una web podria ser http o https.

let protocol = window.location.protocol;

console.log(protocol); // file:

origin

Contiene el origen del documento. En este caso como es un archivo retorna 'file://'.

let origin = window.location.origin;

console.log(origin); // 'file://'

port

Contiene el puerto desde donde se tiene el archivo. Como es un archivo local no retorna nada porque no esta en ningun puerto.

let port = window.location.port;

console.log(port); // ''

assign()

Sirve para cargar un nuevo documento en la ubicacion actual.

window.location.assign('https://google.com'); // Carga la pagina de google en la ubicacion actual
