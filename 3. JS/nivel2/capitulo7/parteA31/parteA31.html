Que es el Objeto window?

Es un objeto, pero no cualquiera: es el objeto mas grande que hay porque es quien esta mas alto en la jerarquia... de modo que hasta el DOM depende de window.

Contiene los metodos y propiedades para manipular la ventana del navegador por lo que tiene muchos usos:

  • Conocer la resolucion de la pantalla.
  • Conocer cuanto scroll se hizo en X o en Y.
  • Navegar a otro sitio web sin la nesecidad de usar href.
  • Cerrar la ventana sin usar el mouse ni el teclado.
  • Conocer datos importantes del sitio (host name, protocolo utilizado, historial, objeto document... etc).
  • Mover, redimencionar o manipular la ventana.

Hay muchos usos mas que estos (y hasta mas utiles incluso), el objeto window es verdaderamente impresionante.

Pendiente por explicar: El objeto window hereda las propiedades de Event Target (?). Investigar despues.

Metodos y Atributos Basicos

Estos metodos y propiedades son los basicos del objeto window, tambien son bastante utiles.

open()

Abre una ventana nueva. Carga un recurso en el contexto del navegador que ya existe o uno nuevo.

window.open(); // Abre una nueva ventana

window.open('https://google.com') // Abre google

Retorna la ventana que fue abierta como un nuevo objeto window.

let ventana = window.open('https://youtube.com');

console.log(ventana); // Window { ... }

close()

Cierra la ventana ventana actual, por razones de seguridad solo se pueden cerrar ventanas que fueron abiertas con el metodo open().

window.close(); // Los scripts solo pueden cerrar ventanas que fueron abiertas por ellos

let ventana = window.open('https://google.com'); // La ventana se abrio

ventana.close(); // Y la ventana se cerro

closed

Contiene si la ventana referenciada esta cerrada o no. Solo lectura.

let ventana = window.open('https://youtube.com');

console.log(ventana.closed); // false

ventana.close();

console.log(ventana.closed); // true

name

Basicamente solo es una propiedad que se usa para darle un nombre a la ventana, o definir el nombre del 'contexto de exploracion...

Obtiene / establece el nombre del contexto de exploracion de la ventana.

let ventana = window.open('https://youtube.com');

console.log(ventana.name); // ''

console.log(ventana) // Window { ... name: '' ... }

ventana.name = 'Espinosaurio';

console.log(ventana.name); // 'Espinosaurio'

console.log(ventana); // Window { ... name: 'Espinosaurio' ... }

ventana.close();

stop()

Es apretar el boton X izquierdo para parar que carge el sitio web.

Detiene la carga de recursos en el contexto de exploracion actual.

let ventana = window.open('https://youtube.com');

ventana.stop(); // Detuvo inmediatamente la carga del sitio web

alert()

Muestra una alerta en el contexto de exploracion actual.

window.alert('Hola Mundo!'); // Hola Mundo!

Tambien funciona en otros contextos de exploracion referenciados, porque ellos tambien son objetos window y poseen ese metodo.

let ventana = window.open('https://youtube.com');

ventana.alert('Hola Mundo!!'); // Hola Mundo!!

print()

Abre el cuadro de dialogo Imprimir para imprimir el documento actual.

window.print(); // Abre el cuadro para enviar el sitio web a la impresora XD

prompt()

Abre un cuadro de dialogo que solicita al usuario un dato. El dato es retornado como String.

let dato = window.prompt('Ingrese un dato'); console.log(dato); // dato

confirm()

Abre un cuadro de dialogo con un mensaje y pregunta al usuario si aceptar o cancelar. Dependiendo de la respuesta del usuario retorna true o false.

let eleccion = window.confirm('Desea ejecutar la accion?');

console.log(eleccion); // true / false
