La nueva PC de Cofla

Despues de aprobar el semestre el buen Cofla se fue a comprar una nueva computadora para hacer las cosas de manera mas practica, y es nesesario que tenga una resolucion optima para poder programar...

Crear Soluciones

  • Mostrarle la resolucion de pantalla debe ser al menos Full HD.
  • Dar la opcion de comprar e implementar una confirmacion antes de realizar la compra.

Fallos de la Pantalla Completa

Cofla va a su casa entra a un sitio web, y presiona F11... ahora Cofla esta navegando en pantalla completa. Pero esto tiene un problema, ya que asi no puede visualizar ni la URL ni el protocolo ni la barra de marcadores... ni absolutamente nada que nos muestra la interfaz de un navegador...

Creae Soluciones

  • Crear un sistema que muestre los suficientes datos para poder conocer el sitio web (url, protocolo, hostname, y el puerto).
