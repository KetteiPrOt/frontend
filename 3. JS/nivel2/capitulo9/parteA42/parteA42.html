Eventos de Interfaz

Algunos eventos referentes a la interfaz del navegador.

Evento error

Se dispara cuando ocurre un error durante la carga de un archivo. Se llama con 'error'.



let pinguinodef = document.getElementById('pinguinodef');

pinguinodef.addEventListener('error', () => {
    alert('Ha ocurrido un error en la carga de la imagen!');
    console.log('El evento error se ha disparado'); // 'El evento error se ha disparado'
});

pinguinodef.setAttribute('src', '/hudevgvgd');

Evento load

Se dispara cuando un elemento o archivo de carga en la pagina. Se llama con 'load'.



let fartangux = document.getElementById('fartangux');

fartangux.addEventListener('load', () => {
    alert('Se ha cargado la imagen multimedia!!');
    console.log('Disparo el Evento load'); // Disparo el Evento load
});

fartangux.setAttribute('src', '../../../../media/asuna.png');

Evento beforeunload

Se dispara cuando la ventana, el documento, y sus recursos estan a punto de ser ser desalojados. El documento aun es visible, y el evento es aun cancelable e este punto. Se llama con 'beforeunload'.

let ventana = window.open('https://youtube.com');

ventana.addEventListener('beforeunload', () => {
    window.alert('La ventana se cerrara!');
});

ventana.close();

resize

Se dispara cuando se cambia la vista del objeto o documento. Se llama con 'resize'.

window.addEventListener('resize', () => {
    console.log('Se ha redimenionado el objeto window!');
});

scroll

Se dispara cuando se hace scroll en la pagina. Seguro que el Event Object de este evento tiene datos muy interesantes... Se llama con 'scroll'.

window.addEventListener('scroll', (eventObject) => {
    console.log('Se activo el evento scroll!!');
    console.log(eventObject);
});

Temporalizadores

Son metodos que permiten medir el tiempo y hacer cualquier otra cosa en funcion de esa informacion.

setTimeout()

Es un metodo de window que recibe como primer parametro una funcion, la cual ejecutara dentro de cierto tiempo el cual recibe como segundo parametro (en milisegundos).

window.setTimeout(() => { alert('Hola Mundo!') }, 2000); // Ejecuta la funcion en 2 segundos

setInterval()

Es un metodo de window que recibe como primer parametro una funcion, la cual ejecutara repetidamente en un intervalo de tiempo el cual recibe como segundo parametro (en milisegundos).

window.setInterval(() => { console.log('Se ejecuto la funcion!') }, 500); // Ejecuta la funcion cada medio segundo

clearTimeout()

Elimina un temporalizador.

let temporalizador = window.setTimeout(() => { alert('Hola Mundo!') }, 1000); // Ejecuta una funcion en 1 segundo

window.clearTimeout(temporalizador); // La funcion nunca se ejecuta porque el temporalizador fue eliminado

clearInterval()

Elimina un intervalo.

let intervalo = window.setInterval(() => { alert('Hola Mundo!') }, 500); // Define un intervalo window.clearInterval(intervalo); // La funcion nunca se ejecuta porque el intervalo fue eliminado
