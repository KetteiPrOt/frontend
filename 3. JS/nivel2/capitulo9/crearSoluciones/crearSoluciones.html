Notas del Penultimo Semenestre

Cofla esta en ultimo anio y nesecita rescuperar sus notas para saber si paso al ultimo semestre, de ser asi cofla estaria a un ultimo paso de finalizar su carrera como ingenirero. Pero hay un problema: las notas estan almacenadas en un servidor. Para poder mostrar las notas de cofla habra de hacer una solicitud a este servidor remoto.

Crear Soluciones

  • Crear un sistema que le muestre las notas a Cofla pidiendoselas al servidor remoto.
  • Las notas deben mostrarse de forma ordenada.
