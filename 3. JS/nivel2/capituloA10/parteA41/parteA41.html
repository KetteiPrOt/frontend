Que es una peticion HTTP?

Es un protocolo de la septima capa (Aplicacion) del modelo de redes OSI, el cual se usa para hacer peticiones de datos a un servidor remoto. Este protocolo no guarda ninguna informacion solo sirve de "mensajero".

Para usar este protocolo en JavaScript se utilizan APIs como AJAX, Flecth, y librerias como AXIOS.

Datos estructurados JSON

Significa Notacion de Objetos en JavaScript (JavaScript Object Notation). Es simplemente una forma estructurada de escribir datos... la forma en que escribimos Objetos en JavaScript.

Los Objetos JSON no tienen propiedades como los objetos, en su lugar solo son estructuras de datos. Se escriben con comillas dobles para que no haya problemas al enviar el Objeto JSON al servidor.

const objeto = {
    propiedad1: 'dato1',
    propiedad2: 'dato2',
    propiedad3: 'dato3',
    propiedad4: 'dato4'
}

const json = {
    "identificador1": "dato1",
    "identificador2": "dato2",
    "identificador3": "dato3",
    "identificador4": "dato4"
}

console.log(typeof objeto); // Object

console.log(typeof json); // Object

Serializacion y Deserializacion

Un Objeto JSON esta serializado en JavaScript cuando ha sido escrito como una cadena de texto. Esto se hace para facilitar su envio a travez de la red.

// Objeto JSON deserializado
const json1 = {
    "identificador1": "valor1",
    "identificador2": "valor2",
    "identificador3": "valor3",
}

// Objeto JSON serializado
const json2 = '{"identificador1": "valor1", "identificador2": "valor2", "identificador3": "valor3"}';

stringify()

Serializa un Objeto JSON para enviarlo a traves de la red.

const json1 = {"identificador1": "valor1", "identificador2": "valor2"};

const json2 = JSON.stringify(json1);

console.log(json2); // '{"identificador1": "valor1", "identificador2": "valor2"}'

parse()

Deserializa un Objeto JSON recibido a travez de la red.

const json1 = '{"identificador1" : "valor1", "identificador2" : "valor2"}';

const json2 = JSON.parse(json1);

console.log(json2); // {"identificador1": "valor1", "identificador2": "valor2"}
