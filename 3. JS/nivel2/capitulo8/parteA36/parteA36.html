Eventos del Mouse

Mouse

Eventos sobre el mouse.

Click

Ocurre con un click sobre un elemento. Se llama con 'click'.



let pajazo = document.getElementById('pajazo');

pajazo.addEventListener('click', () => { alert('Has activado el evento click!') });

Doble Click

Ocurre cuando sehace doble click sobre un elemento. Se llama con 'dblclick'.



let gallozo = document.getElementById('gallozo');

gallozo.addEventListener('dblclick', () => {
    alert('Hiciste Doble Click!');
});

mouseover

Mouse encima. Ocurre cuando el puntero se mueve sobre un elemento, o sobre uno de sus hijos. Se llama con 'mouseover'.

Pon la flecha sobre mi!


let codrilaxo = document.getElementById('codrilaxo');

codrilaxo.addEventListener('mouseover', () => {
    alert('Activaste el Evento!');
});

mouseout

Mouse fuera. Ocurre cuando se mueve el puntero fuera de un elemento o de sus elementos secundarios. Se llama con 'mouseout'.

Pon el puntero sobre mi, y sacalo!


let dongup = document.getElementById('dongup');

dongup.addEventListener('mouseout', () => {
    alert('Activaste el Evento mouseout!!');
});

contextmenu

Menu contextual. Ocurre cuando se hace click derecho en algun elemento para abrir el menu contextual. Se llama con 'contextmenu'.

Hasme click derecho!


let trotocho = document.getElementById('torotocho');

torotocho.addEventListener('contextmenu', () => {
    alert('Ejecutaste el Evento contextmenu!!');
});

mousemove

Ocurre cuando el puntero se mueve mientras esta sobre un elemento.

Este evento tiene muchas propiedades. Entre ellas las coordenadas x, z donde esta el cursor dentro el elemento... ellas se podrian usar en un juego o para dibujar con la API Canvas.

Muevete encima para activar el evento!


let udon = document.getElementById('udon');

udon.addEventListener('mousemove', () => {
    console.log('Se Activo el Evento!!'); // Se Activo el Evento!!
});

mousedown

Cae el mouse. Ocurre cuando presionamos hacia abajo el click del mouse sobre un elemento, aunque luego no lo subamos. Se llama con 'mousedown'.

Deja caer el click!


let roedxr = document.getElementById('roedxr');

roedxr.addEventListener('mousedown', () => {
    alert('Activaste el evento mousedown!!');
});

mouseup

Sube el mouse. Ocurre cuando subimos el click del mouse sobre un elemento, aunque no lo hayamos dejado caer sobre el mismo elemento. Se llama con 'mouseup'.

Sube el click aqui!


let deinsleif = document.getElementById('deinslef');

deinsleif.addEventListener('mouseup', () => {
    alert('Activaste el Evento mouseup!!');
});

Teclado

Eventos sobre el teclado.

keydown

Ocurre cuando una tecla se presiona. Se llama con 'keydown'.

Este evento tiene muchas propiedades, entre ellas code retorna un string con el codigo de la tecla fisica que fue presionada (ignora la disposicion de teclado del usuario, siempre trabaja con teclado QWERTY, si quieres que funcione con otras puedes usar Keyboard.getLayoutMap()).



let rexcapto = document.getElementById('rexcapto');

rexcapto.addEventListener('keydown', (eventObject) => {
console.log('Activaste el Evento keydown!!');

let codigo = eventObject.code;

// Tecla que fue presionada console.log(codigo);
});

keyup

Ocurre cuando una tecla se suelta. Se llama con 'keyup'. En orden, ocurre despues de keydown.

Este evento tiene muchas propiedades, entre ellas code retorna un string con el codigo de la tecla fisica que fue presionada (ignora la disposicion de teclado del usuario, siempre trabaja con teclado QWERTY, si quieres que funcione con otras puedes usar Keyboard.getLayoutMap()).
