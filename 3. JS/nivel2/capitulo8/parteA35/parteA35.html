Pestaña Elements (DevTools)

Volver al capitulo

Filtros y buscar por palabras claves

La opcion search (ctrl + shift + f) permite buscar por palabras clave en los elementos del archivo html.

Modificar crear y eliminar etiquetas

Se pueden modificar eliminar y crear elementos html localmente en la pestania elements.

Modificar propiedades de CSS

Se pueden cambiar localmente los estilos y ver que estilos se aplican a los elementos utilizando la pestaña styles, que se ubica dentro de elements.

Event Listeners

Se ubica dentro de elements, esta pestania permite ver los escuchadores de eventos asociados a un elemento html seleccionado.

Properties

Se ubica dentro de elements, esta pestania permite ver todas la propiedades que tiene asociadas un elemento html seleccionado.
