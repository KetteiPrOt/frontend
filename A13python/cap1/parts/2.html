Secuencias

Las secuencias de Python son estructuras de datos capacez de almacenar una colección de elementos.

Las principales con Listas, Tuplas, y Rangos.

El largo de una secuencia se puede obtener con la función len(). 


sequence = [1, 2, 3, 4, 5]
print(len(sequence)) # 5

Listas

Las listas representan una colección ordenada de elementos que puede ser modificada. 


lista = ['1er elemento', '2do elemento', '3er elemento']

print(lista) # ['1er elemento', '2do elemento', '3er elemento']
print(lista[0]) # 1er elemento

Tuplas

Las listas representan una colección ordenada de elementos que no puede ser modificada. 


tupla = ('1er elemento', '2do elemento', '3er elemento')

print(tupla) # ('1er elemento', '2do elemento', '3er elemento')
print(tupla[0]) # 1er elemento

Rangos

Son una secuencia de números enteros. Toma tres argumentos: valor inicial (incluido), valor final (no incluido), y paso. 


range = range(1)

print(range[0]); # 1

Si solo se recibe el primer argumento, entonces se crea un rango [0, 1, 2, 3, ... N] sea N el primer argumento.

Diccionarios

Es una estrutura de datos de tipo mapa.

Los diccionarios (dict) de Python permiten almacenar un conjunto de datos como pares clave-valor, donde cada valor es accesible a través de una clave única. 


dictionary = {
    'key1': 'value1',
    'key2': 'value2',
    'keyN': 'valueN'
}

print(dictionary['key1']) # value1

Son útiles para almacenar y acceder a datos de manera eficiente.

Conjunto

Es una estructura de datos que almacena una colección desordenada de elementos no repetidos.

Puedes crearla a partir de otras estructuras con la función set(), o declararlas usando corchetes. 


list = [1, 2, 3, 4, 5, 6, 1, 2, 3, 4]

set_0 = set(list)
print(set_0) # {1, 2, 3, 4, 5, 6}

set_1 = {'a', 'b', 'c', 'd'}
