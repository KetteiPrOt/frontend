Definir función

Las funciones en Python se definen con la pabra clave def. 


def f(x):
    return x + 3

domain = (0, 1, 2, 3, 4, 5)

for x in domain:
    print(f(x))
    # 3, 4, 5, 6, 7, 8

Si una función g es definida dentro de otra función f, entonces g solo existirá dentro del ámbito de f. No existirá en el ámbito global.

Recursividad

Se produce cuando una función se invoca a si misma hasta que se cumpla una condición de parada.

El ejemplo más básico de recursividad es cuando se escribe una función para calcular el factorial de un número. 


number = 5

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
    
print(factorial(number))
