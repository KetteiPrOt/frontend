Condicional

Evalúa una condicion para decidir que bloque de código ejecutar. 


a = 3; c = 2

if a == c:
    print(f'{a} es igual a {c}')
elif a > c:
    print(f'{a} es mayor a {c}')
else:
    print(f'{a} es menor a {c}')

# 3 es mayor a 2

Bucle for

Se usa principalmente para iterar sobre una secuencia. 


list = ['element 1', 'element 2', 'element 3', 'element 4']

for element in list:
    print(element)
"""
element 1
element 2
element 3
element 4
"""

for number in range(5):
    print(number)
"""
0
1
2
3
4
"""

Tenemos la palabra clave break para detener el bucle, y continue para saltarnos una iteración. 


for number in range(10):
    if number == 6:
        break
    print(number)
"""
0
1
2
3
4
5
"""

for number in range(10):
    if number == 6:
        continue
    print(number)
"""
0
1
2
3
4
5
7
8
9
"""

Bucle while

Se usa para iterar un bloque de código mietras se cumpla una condición. 


count = 0
while count < 5:
    print(count)
    count += 1

"""
0
1
2
3
4
"""

try except

Se usa para intentar ejecutar un bloque código y controlar una excepción que podría surgir dentro de él. 


a = input('Ingrese un número A ')
b = input('Ingrese un número B ')

c = int(a) / int(b)
print(f'La división A / B  es igual a {c}')

try:
    c = int(a) / int(b) # could throw ValueError or ZeroDivisionError
    print(f'La división A / B  es igual a {c}')
except ValueError:
    print('Un número ingresado no es válido')
except ZeroDivisionError:
    print('No puedes dividir por cero')
except Exception:
    # general error handling
    print('Ha ocurrido un error')
