Volver a las partes

__init__

Es el constructor que construye el objeto al instanciar la clase. 


class Person():
    def __init__(self, name):
        self.name = name

__str__

Define como se muestra el objeto como cadena de texto. 


class Person():
    def __init__(self, name):
        self.name = name

    def __str__(self):
        return f'Hola, soy {self.name}'
    
joel = Person('Joel')
print(joel) # Hola, soy Joel

__repr__

Retorna una representación del objeto usando la función repr(). El objeto puede ser reconstruido pasandole la representación a la función eval(). 


class Person():
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __repr__(self):
        return f'Person("{self.name}", {self.age})'
    
    def __str__(self):
        return f'Hola, soy {self.name} y tengo {self.age} años'
    
object = Person('Joel', 19)
representation = repr(object)
rebuilded = eval(representation)
print(rebuilded) # Hola, soy Joel y tengo 19 años

Sobrecarga de operadores

Si cada cosa es instancia de una clase (incluso los números), entonces tiene que haber una forma de definir como se comportan los objetos cuando se utilizan los operadores (+, -, *, /, >, <, ==) para jugar con ellos.

Para realizar esta definición se utilizan los métodos especiales.

__add__

Sobrecarga del operador suma (+). 


class Money():
    def __init__(self, amount):
        self.__amount = amount

    @property
    def amount(self): return self.__amount

    def __add__(self, other):
        summation = self.amount + other.amount
        return Money(summation)

moneyBag = Money(50)
moneyBox = Money(20)

money = moneyBag + moneyBox

print(money.amount) # 70

__sub__

Sobrecarga del operador resta (-). 


class Money():
    def __init__(self, amount):
        self.__amount = amount

    @property
    def amount(self): return self.__amount

    def __sub__(self, other):
        summation = self.amount - other.amount
        return Money(summation)

moneyBag = Money(50)
moneyBox = Money(20)

money = moneyBag - moneyBox

print(money.amount) # 30
