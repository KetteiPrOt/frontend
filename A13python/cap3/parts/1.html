Crear un módulo

Un módulo en Python es un archivo con la extensión '.py', que contiene código.

Importar módulo

Cuando el módulo esta en el mismo directorio que el programa se importa con import, seguido de su nombre. 


# module.py
print('Module')

# program.py
import module
print('Main Program')

# program.py output on execution:
Module
Main Program

Usar objetos

Cuando en un módulo estan definidos objetos (variables, funciones, clases, entre otros): 


# module.py
variable = 'Variable value'

def function():
    return 'Function output'

class Object():
    def method():
        return 'Method executed'

Podemos acceder a estos objetos usando la notación del punto: 


import module

print(module.variable) # Variable value
print(module.function()) # Function output
object = module.Object()
print(object.method()) # Method executed

Importar objetos

En lugar de importar el módulo completo, aveces podemos preferir importar del módulo directamente solo los objetos que vamos a usar. Esto se consigue usando import junto a from. 


from module import variable, function, Object

print(variable) # Variable value
print(function()) # Function output
object = Object()
print(object.method()) # Method executed

Cuando son muchos objetos puedes usar * para importarlos de forma rápida. 


from module import *

print(variable) # Variable value
print(function()) # Function output
object = Object()
print(object.method()) # Method executed

Renombrar importación

Podemos renombrar un módulo al importarlo usando as. 


import module as new_name

Navegar entre directorios

Cuando los módulos están en diferentes directorios se usa la notación del punto para acceder a ellos. 


'''
program.py
directory
    module.py
'''

# program.py
import directory.module as module

print(module.variable) # Variable value
print(module.function()) # Function output
object = module.Object()
print(object.method()) # Method executed

Para retroceder un nivel de profundidad en los directorios se escriben dos puntos. No se puede importar un módulo del nivel raíz hacia un módulo anidado. 


'''
program.py
directory_a
    directory_b
        module_b.py
    module_a.py
'''

# module_b.py
import ..directory.module_a
